Formularios HTML

Son herramientas que permiten la interaccion (intercambio de datos) entre cliente y servidor. Para implementarlos dentro de una pagina web debemos utilizar la etiqueta <form> y solo se considerara para el envio, los datos que se encuentren dentro del mismo, ademas se deben especificar algunas caracteristicas para que este se realice correctamente.

¿Como funcionan?

A la etiqueta mencionada anteriormente, tenemos que especificarle 2 atributos, por un lado action="..." que indica al formulario que hacer al momento de realizar el envio y method="..." que indica quien tiene la prioridad al momento de realizar el mismo, dentro de los metodos existentes podemos utilizar:

  • GET: metodo poco seguro, que prioriza la respuesta del servidor, los datos permanecen visibles en la url y se utiliza principalmente para buscadores
  • POST: metodo seguro que cifra los datos antes y despues del envio, debido a esto es mas lento, pero ofrece cierta garantia para proteger los datos del usuario. Se utiliza para formularios de registro, acceso y consulta web

¿Como estan Compuestos?

Dentro de un formulario utilizaremos varios elementos, pero entre ellos destacan las Etiquetas (LABEL) que funcionan como rotulos para indicar al usuario que debe seleccionar o completar; otro elemento necesario son los CAMPOS (INPUT / TEXTAREA / SELECT) que permiten al usuario la carga o seleccion de datos, estos poseen atributos que en ciertas ocasiones deben estar escritos de manera explicita.

  • name="..." corresponde al nombre del grupo de campos al que pertenece el control, necesario para el envio.
  • value="..." indica el dato a ser enviado, no es necesario si el usuario puede escribir el valor.
  • type="..." corresponde al formato del dato a ingresar/seleccionar, es necesario para los campos de tipo INPUT.
