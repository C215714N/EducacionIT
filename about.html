Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente y servidor. Para implementarlos dentro de la pagina debemos utilizar la etiqueta <form> y dentro de la misma colocar los elementos que se procesaran cuando se realice el envio de los datos.

¿Cómo funcionan?

A la etiqueta mencionada anteriormente debemos colocarle al menos 2 atributos, action="..." que indica que sucede al momento de realizar el envio y method="..." que especifica como deben procesarse los datos que seran enviados, para este ultimo los valores mas comunes son:

  • GET: Metodo que prioriza la respuesta del servidor, ya que envia los datos en texto plano y los mismos permanecen visibles en la barra de direcciones. Se utiliza principalmente en formulario de Busqueda
  • POST: Metodo que prioriza la seguridad de los datos, debido a que cifra los mismos antes de realizar el envio y por lo tanto es mas lento. Se utiliza en formularios de registro, acceso, contacto, etc...

¿Cómo estan compuestos?

Dentro de un formulario podemos colocar cualquiera de los elementos existentes, pero principalmente debemos utilizar las etiquetas (LABEL) que funcionan a modo de rotulos, indicando al usuario lo que debe completar o seleccionar; y loscampos (INPUT / TEXTAREA / SELECT) que permiten al usuario interactuar con la pagina y poseen los siguientes atributos.

  • name="..." corresponde al nombre que el que se van a identificar los valores enviados a traves del formulario. Cuando varios elementos comparten nombre corresponden al mismo control.
  • type="..." especifica el tipo de dato que esta permitido ingresar o seleccionar, es obligatorio en caso de utilizar la etiquetas <input>
  • value="..." atributo que indica el valor del campo de formulario, es implicito en caso que el usuario escriba y explicito/obligatorio si el usuario selecciona opciones.

Tablas HTML

Son elementos que permiten organizar la informacion de manera tabular y deben utilizarse cuando necesitamos estructurar datos de una manera determinada. Para incorporarlas dentro de la pagina debemos utilizar la etiqueta <table> y dentro de la misma colocar los elementos que forman que parte de su estructura respetando el modelo de cajas.

¿Poseen Estructura Semantica?

Dentro de la etiqueta principal, mencionada anteriormente, debemos colocar las filas (table row) y luego las celdas (table division) utilizando las etiquetas <tr> y <td> respectivamente. Ademas podemos indicar a que parte de la estructura corresponde cada bloque de contenido utilizando los siguientes elementos semanticos

  • THEAD: Cabecera que contiene la fila de encabezados, que se crean utilizando la etiqueta <th> (Table Heading).
  • TBODY: Cuerpo de tabla que contiene la informacion principal, estructurada en varias filas.
  • TFOOT: Pie de tabla que contiene datos complemeantios como totales, promedios, maximos, minimos, etc...

¿Celdas Combinadas?

En algunos casos, puede que necesitemos que una celda ocupe mas de una fila o una columna, para ello debemos colocar dentro de la misma los atributos rowspan="..." y colspan="..." respectivamente, con el valor total de celdas a ocupar. Cabe destacar que todo contenido posterior, a la celda expandida, se desplazara hacia la derecha.

Contenido Multimedia

Se conoce con este nombre a todo elemento audiovisual que podemos utilizar en nuestro sitio. Actualmente los navegadores incluyen una API de reproduccion que permite incorporar estos elementos sin necesidad de instalar plugins o complementos de terceros. Podemos corroborar la compatibilidad de nuestro navegador con dichos elementos desde el sitio Can I Use.

¿Contenido Embebido?

Corresponde a una categoria de la web que indica que dicho elemento fue compartido con autorizacion del autor original, normalmente para incorporar estos elementos se utiliza la etiqueta <iframe> que permite la carga asincrona de contenido, pudiendo mostrar el contenido se otro sitio, desde nuestra pagina web.

¿Multimedia Sin Plugins?

Como se menciona al principio, los navegadores actuales brindan soporte nativo a este tipo de elementos. Para incorporarlos simplemente debemos utilizar las etiquetas <audio> y <video> que son compatibles con los formatos .mp3, .wav, .ogg y .mp4, .webm, .ogv respectivamente; tambien podemos utilizar atributos como controls autoplay muted y loop segun corresponda

Acceso FTP

Se conoce con este nombre al Protocolo de Transferencia de Archivos que permite a la computadora intercambiar datos con otro host. Es necesario para poder publicar nuestro sitio web en un servidor, asi como tambien, los cambios que vayamos realizando al mismo. Cabe mencionar que el proveedor de hosting debe brindarnos los siguientes datos para la conexion.

  • HOST corresponde a la direccion del servidor, donde subiremos nuestros archivos
  • USER nombre con el que se identifica a quien solicita realizar la transferencia
  • PASSWORD dato que valida la identidad del usuario, obligatorio para la conexion
  • PORT numero que indica el canal de comunicaciones utilizado, por defecto es 21
