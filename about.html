Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente y servidor, para implementarlos dentro de la pagina debemos colocar la etiqueta <form> y dentro de la misma los elementos que seran procesados para el envio. Cabe destacar que, por defecto, hasta que el usuario no presione un boton que ejecute dicha accion, ningun dato sera enviado al servidor; tampoco se consideraran elementos que no se encuentran dentro de la etiqueta.

¿Como funcionan?

A la etiqueta mencionada anteriormente debemos especificarle por lo menos 2 atributos, estos son action="..." que indica que debe suceder cuando el usuario realiza el envio del formulario; y method="..." el cual indica como debe realizarse dicho envio, entre los valores que podemos utilizar se encuentran:

  • GET metodo que prioriza la respuesta del servidor, muestra los datos enviados en la URL y los envia en texto plano. Este metodo es ideal para buscadores.
  • POST metodo cuya prioridad es garantizar la seguridad de los datos enviados por el usuario, debido a que cifra los mismos antes de realizar el envio. Este metodo se utiliza en mayoria de formularios como contacto, registro, inicio de sesion, etc....
