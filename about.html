Formularios HTML

Son elementos que permiten el intercambio de datos entre cliente y servidor, para utilizarlos en nuestro sitio debemos colocar la etiqueta <form> y dentro de la misma los elementos que seran procesados cuando se realice el envio del mismo.

¿Como Funcionan?

A la etiqueta mencionada anteriormente debemos colocarle por los menos 2 atributos, action="..." que indica lo que debe suceder cuando se realiza el envio del formulario; y method="..." que indica como se deben procesar los datos, es decir, quien tiene prioridad durante el envio de los mismos.

  • GET metodo que prioriza la respuesta del servidor, los datos permanecen visibles en la barra de direcciones y el envio se hace en texto plano. Se utiliza principalmente en Buscadores.
  • POST metodo que prioriza la seguridad de los datos, ya que cifra los mismos y estos no se visualizan en la barra de direcciones. Se utiliza en formularios de registro, acceso, contacto, etc...

¿Como estan Compuestos?

Dentro de los formularios podemos utilizar cualquier etiqueta conocida, pero exitens 2 tipos de etiquetas necesarias para el envio del mismo, las Etiquetas (LABEL) que indican al usuario lo que debe ingresar o seleccionar y tambien facilitan la accesibilidad; y los Campos (INPUT / TEXTAREA / SELECT) que corresponden a los controles del formulario, aquellos con lo que el usuario interactua y que poseen algunos de los siguientes atributos.

  • name="..." corresponde al nombre con el que se identifica el campo, es necesario para la extraccion de los datos. En caso de opciones multiples el nombre es compartido.
  • type="..." indica el tipo de valor que se puede ingresar o seleccionar, limita el campo para su completado. Es obligatorio cuando utilizamos la etiqueta <input>
  • value="..." hace referencia al dato ingresado, por defecto es implicito, pero si el usuario selecciona una opcion debe figurar dentro del campo de formulario.
