Contenido Multimedia

Podemos encontrar este tipo de elementos en cualquier pagina web, cada uno se representa con su propia etiqueta y entre ellos encontramos imagenes, video y sonidos. En la actualidad la mayoria de los navegadores es compatible con multiples formatos que se detallan mas adelante.

Imágenes en HTML

Para utilizar una imagen dentro de una pagina web debemos colocar la etiqueta <img> e indicar donde se encuentra con el atributo src="...", ademas de indicar de que trata la misma con el atributo alt="..." que se visualizara cuando la imagen no este disponible.

Reglas de Nomenclatura

Cuando incorporamos este tipo de contenido en nuestras paginas, debemos tener en cuenta unos cuantos consejos con respecto al nombre de los archivos. Esto no solamente aplica para las imagenes, sino para cualquier elemento que podemos vincular con nuestra web. Dichas reglas son las siguientes:

  • Evitar el uso de caracteres especiales como por ejemplo el espacio, que se traduce en el conjunto %20. Una solucion a esto es utilizar guiones (-_) para separar palabras.
  • Utilizar nombres simples y descriptivos para facilitar la implementacion de los archivos. Evitar nombres genericos o atomatizados como los que utilizan las camaras digitales.
  • Implementar solamnete minusculas ya que los servidores son CaseSensitive, es decir, que distinguen entre mayusculas y minusculas. De esta manera se reduce el margen de error.

Formatos de Imágen

Si bien existen multiples formatos que podemos utilizar, debemos tener en cuenta la compatibilidad con los navegadores web. Las imagenes se pueden catalogar como Mapa de Bits o rasters que corresponde a la mayoria de los formatos y utilizan una grilla basada en pixeles para definir su contenido o Graficos vectoriales que corresponde a un tipo de imagen compuesta de lineas y curvas, calculadas matematicamente.

  • JPG (Joint Photographic Experts Group)
  • PNG (Portable Network Graphics)
  • GIF (Graphic Interchange Format)
  • WEBP (Web picture)
  • SVG (Scalable Vector Graphics)
