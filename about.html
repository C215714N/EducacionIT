Formularios HTML

Son herramientas que permiten la interaccion o intercambio de datos entre cliente y servidor, para implementarlos dentro de una pagina web debemos colocar la etiqueta <form> y dentro de la misma los elementos que seran contemplados para el envio. Todo elemento que se encuentre por fuera de esta etiqueta no sera considerado parte del formulario.

¿Como funcionan?

A la etiqueta mencionada anteriormente debemos especificarle por los menos 2 atributos, estos son action="..." que indica lo que sucede cuando se envia el formulario y method="..." que indica quien tiene la prioridad en la comunicacion, entre los valores mas utilizados se encuentran:

  • GET: Metodo cuya prioridad radica en la respuesta del servidor, los datos figuran en la barra de direcciones y son enviados en texto plano. Se utilizan principalmente para Buscadores
  • POST: Metodo cuya prioridad radica en la seguridad de los datos provistos por el usuario, ya que la informacion se cifra durante el envio. Se utiliza para formularios de registro, acceso, contacto.

¿Como estan compuestos?

Dentro de un formulario podemos colocar practicamente cualquier etiqueta conocida, pero 2 elementos son propios del mismo. Por un lado estan las Etiquetas (LABEL) que indican a modo de rotulo que debe seleccionar o completar el usuario, ademas de facilitar la accesibilidad; y por el otro, estan los Campos (INPUT / TEXTAREA / SELECT) que corresponden a las areas donde el usuario puede ingresar o seleccionar datos, entre sus atributos mas comunes se encuentran:

  • name="..." indica el grupo de pertenencia del control de formulario, ademas que los datos se obtienen mediante este identificador.
  • type="..." corresponde al formato del dato a ingresar dentro del campo, se utiliza principalmente en los input.
  • value="..." Dato a ser procesado, esta implicito en caso que el usuario escriba y solamente debe especificarse cuando se selecciona una opcion.
