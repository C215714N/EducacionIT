Contenido Multimedia

Hace referencia a todo tipo de elemento audiovisual que podemos incorporar dentro de nuestra pagina web, mediante el uso de etiquetas.

Antes de incorporar este tipo de archivos debemos tener en cuenta los formatos soportados por el navegador, ademas de algunas reglas para su correcta implementacion.

Formatos

  • JPG (Joint Photographics Experts Group): admite compresion, admite hasta 16.777.216 colores
  • PNG (Portable Networks Graphics): admite transparencia, pero no utiliza compresion
  • WEBP (Web Picture): Formato desarrollado para la compaticion de contenidos
  • SVG (Scalable Vector Graphics): imagenes calculadas matematicamente mediante lineas y curvas
  • Evitar el uso de caracteres especiales ya que estos no se procesan correctamente y si por ejemplo nuestra imagen se llama "foto 01.jpg" el navegador interpretara "foto%2001.jpg", por este motivo se recomienda utilizar guiones (_-)
  • Utilizar nombres descriptivos como por ejemplo "background.jpg", para facilitar la implementacion de los archivos y evitar nombres simples como "foto_01.jpg" o automatizados como "2023-04-13-2055100.jpg"
  • Implementar nombres solamente en minusculas debido a que los servidores son caseSensitive y no es lo mismo "foto.jpg", "Foto.jpg", "FOTO.jpg". De esta manera reducimos el margen de error.
