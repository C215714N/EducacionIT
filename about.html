Formularios HTML

Son herramientas que permiten la interaccion (intercambio de datos) entre el cliente y el servidor, para utilizarlos dentro de la pagina debemos colocar la etiqueta <form> y en su interior deben encontrarse los elementos que formen parte del mismo. Al momento de realizar el envio, no se tomara en cuenta cualquier elemento que se encuentre por fuera.

¿Como funcionan?

A la etiqueta mencionada anteriormente debemos colocarle por lo menos 2 atributos, action="..." par indicar que sucede al enviar el formulario y method="..." para establecer quien tiene la prioridad al momento de realizar el envio, los valores mas comunes para este atributo son:

  • GET: metodo cuya prioridad radica en la respuesta del servidor, los datos permanecen visibles en la barra de direcciones y se utiliza principalmente para buscadores.
  • POST: metodo cuya prioridad se encuentra en la seguridad de los datos del usuario, ya que cifra los datos antes y despues del envio, se utiliza para formularios de registro, acceso, consulta web, etc...

Contenido Multimedia

Corresponde a todo elemento grafico y audiovisual que podemos incorporar dentro de una pagina web. Antiguamente eran poco frecuentes debido a las limitaciones de ancho de banda, pero actualmente se utilizan para reemplazar textos y realizar demostraciones, ademas que se pueden reproducir de forma nativa.

¿Contenido Embebido?

Es una categoria de la web que indica que dicho material fue utilizado con autorizacion del publicador o autor original. Normalmente se incorporan estos elemento mediante la etiqueta <iframe> que permite la carga asincrona de contenido.

¿Multimedia sin plugins?

Como se hace mencion en el apartado anterior hoy en dia este tipo de contenido se puede reproducir gracias a la API de reproduccion que poseen los navegadores actuales. Para acceder a la misma solamente debemos utilizar las etiquetas <audio> y <video> que son compatibles con los formatos .mp3, .ogg, .wav y .mp4, .ogv, .webm respectivamente.
