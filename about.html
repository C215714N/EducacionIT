Fluid/Liquid Web Design

Diseño que utiliza maximos y minimos, cuyos elementos, al igual que los liquidos, se adaptan a sus contenedores. El problema principal de este diseño radica en los espacios generados en pantallas grandes y problemas para conetener elementos en pantallas pequeñas.

autor: Cristian Racedo

Adaptive Web Design

Diseño que implica, generar contenido para cada tamaño de pantalla (general), permitiendo compatibilidad con multiples dispositivos. El problema de este diseño radica en la duplicidad de contenido debido a que se realiza un diseño particular para cada tamaño de pantalla.

autor: Cristian Racedo

Responsive Web Design

Diseño que utiliza los breakpoints para definir el aspecto de nuestro sitio segun el tamaño de la ventana del navegador. Uno de los inconvenientes de este tipo de diseño radica en la mala planificacion del sitio o la falta de esta. Debemos tener en cuenta que una de las premisas de este diseño es el mobile first.

autor: Cristian Racedo
