Introduccion a Bootstrap

Es un framework de css que nos ofrece una plantilla de hoja de estilos y scripting, para agilizar el desarrollo de una pagina utilizando diferentes clases de css que se encuentran definidas previamente. Esta herramienta fue desarrollada por el equipo de twitter y es la misma que se implementa en muchos sitios debido a que esta pensado para el desarrollo responsive
