Responsive Web Design

Fluid Design

Modelo que utiliza porcentajes y ems para definir el tamaño de los elementos de la pagina y la tipografia de la misma. Se utiliza para mantener la proporcion de las etiquetas en los diferentes tamaños, pero en pantallas pequeñas el contenido se reduce demasiado y en pantallas grandes este se estira excesivamente.

Para mas informacion consultar Alumni o enviar un correo a cristiandracedo@hotmail.com

Adaptive Design

Modelo que plantea la creacion de paginas individuales para cada tamaño de pantalla en el que se enfoque el diseño, normalmente empezando desde la vista de escritorio. Es facil de mantener y una buena alternativa para clientes con poco presupuesto, pero los contenidos de encontraran por duplicado en el sitio.

Para mas informacion consultar Alumni o enviar un correo a cristiandracedo@hotmail.com

Responsive Design

Modelo que implica la realizacion de un diseño que responda segun cada tamaño de pantalla, implementando un sistema de grillas, multimedia flexible y media queries para generar las reglas del diseño; todo esto utilizando una unica estructura base y, por lo tanto, con el mismo contenido para todas las versiones.
