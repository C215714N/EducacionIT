Nuevos elementos HTML

Contenido Multimedia

En la actualidad los navegadores, en su mayoria, soportan la reproduccion nativa de audio y video. Esto se debe a que cuentan con un reproductor integrado que facilita la incorporacion de este tipo de contenido, pero no todos los formatos se pueden reproducir. En caso del elemento audio soporta .mp3 - .wav - .ogg y el elemento video soporta .mp4 - .webm - .ogv.

Elementos Estadisticos

HTML5 incorpora nuevas etiquetas para la representacion de valores estadisticos. Estan son <progess> que se utiliza para mostrar el avance de una tarea cuando esta aun no fue finalizada y <meter> que permite mostrar y comparar valores cuando una tarea ya fue finalizada, diferenciandolos gracias al atributo optimum, que los identifica mediante un codigo de color.

  • Elemento Progress
    • sin value
    • value:20 ; max:100
    • valor Input (JavaScript)
  • Elemento Meter
    • value* < low < high < optimum
    • low < value* < high < optimum
    • low < high < value* < optimum

API's de Dibujo

Los navegadores desde sus inicios permiten la incorporacion de imagenes mapa de bits y graficos vectoriales, pero actualmente tambien podemos generar desde nuestro codigo imagenes de estos tipos. Mediante la etiqueta SVG podemos utilizar XML para crear graficos compuesto de lineas y curvas; asi como tambien utilizando la etiqueta CANVAS mediante la implementacion de JAVASCRIPT podemos generar graficos raster, compuestos por pixeles.

Los mismos se clasifican segun como realizan su trabajo, en aclarar, oscurecer y tono

Gradientes y Modelos de color

Podemos implementar colores utilizando diferentes tipos de valores, por un lado tenemos los colores html que representan un total de 140 colores como skyblue, luego tenemos los colores que utiliza el modelo rgb expresados en valores decimales como rgb(68, 136, 255) o hexadecimales como #4488ff y por ultimo CSS incorporo el modelo hsl que utiliza el circulo cromatico para definir el tono del color, utiliza la valores de porcetanje para definir la saturacion, asi como tambien la cantidad de luz del mismo.

Transformaciones 3d

Asi como se menciono anteriormente, podemos realizar transformaciones en 3 dimensiones, para ello es necesario definir un origen de perspectiva en el elemento padre de aquellas etiquetas donde querramos aplicar la misma. En comparacion, la forma de trabajar de las transformaciones es similar a position:relative en el caso de las 2 dimensiones y position:absolute en el caso de las transformaciones 3d.

Efectos de Transicion

Para mejorar el aspecto estetico de un elemento cuando realizamos un cambio de estado, por ejemplo :hover o :focus y nuestra etiqueta cambia su atributos, podemos implementar lo que se conoce como transicion, que implica suavizar el cambio de un punto "A" a un punto "B" generando los estadios intermedios utilizando una linea de tiempo. El mismo se puede dar de diferentes maneras, dando la sensacion de totalmente diferentes, utilizando el atributo transition-timing-function
