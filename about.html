Introduccion

Bootstrap es un framework de css desarrollado por el equipo de twitter que se basa en la metodologia BEM (Block Element Modifier) para el uso de clases que se encuentran predefinidas en su hoja de estilos. Dispone de varios elementos como helpers, utilities y components que consisten en clases que definen diferentes atributos, individualmente o de manera general.

Para implementarlo dentro de nuestras paginas, podemos hacerlo de diferentes maneras. Por un lado, utilizando el CDN (Content Distribution Network) que nos brinda el mismo sitio de Bootstrap y consiste en una hoja externa alojada en otro servidor. Por otro lado, podemos descargar la hoja de estilos desde el mismo sitio o utilizando el gestor de paquetes de Node JS mediante el comando npm i bootstrap
