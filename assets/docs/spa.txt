<h3>SPA (Single Page Application)</h3>

<p>
    Sitio web que implementa una sola página con el objetivo de brindar una experiencia más fluida a los usuarios. En un SPA el código HTML, CSS y JavaScript se cargan una sola vez, los recursos necesarios se cargan dinámicamente cuando lo requiera la página, normalmente como respuesta a las acciones del usuario. La interacción con las aplicaciones pueden involucrar comunicaciones dinámicas con el servidor web.
</p>

<ul>
    <li>ReactJS</li>
    <li>VueJS</li>
    <li>Angular</li>
</ul>
