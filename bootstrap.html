Introduccion a Bootstrap

Es un Framework de CSS que ofrece una hoja de estilos y un archivo de scripts para facilitar la aplicacion de un diseño responsive para nuestros sitios web. Utiliza una metodologia conocida como BEM (Block Element Modifier) que consiste en el uso de clases identificadoras para los contenedores principales y elementos de contenido que deban colocarse en su interior, ademas de clases modificadores que deben utilizarse en conjunto con las anteriores para especificar valores alternativos.

Otra caracteristica notable de este Framework es su sistema de grillas responsive de 12 columnas, que consiste en la subdivision de un contenedor en una grilla utilizando flexbox basada en el uso de porcentajes para identificar el valor de cada columna. Ademas de la identificacion de clases para los diferentes breakpoints que puede tener una pagina web.

Utilities y Helpers

Consisten en clases que permiten al desarrollador establecer atributos de manera sencilla, para utilizarlos debemos colocar la letra o abreviatura del atributo a implementar seguido de un valor que pueder numerico o predefinido. Debido a que en las ultimas versiones se enfoca en el diseño a nivel global, para atributos que implique direccion como izquierda o derecha debemo utilizar valores como start y end, segun el tipo de escritura implementada.
