Bootstrap

Responsive Web Design

Introduccion

Framework de CSS que facilita la implementacion de estilos mediante la utilizacion de clases y determinados atributos dentro de la estructura HTML de nuestro sitio web. Para incorporarlo a nuestros proyectos podemos copiar los links del CDN (Content Distribution Network) y utilizarlo online o descargar los recursos desde el sitio web de Bootstrap. Cuando agregamos la hoja de estilos de Bootstrap a nuestro proyecto, se impone por encima de nuestros estilos. Esto se debe a que muchas de las reglas del CSS utilizan el atributo !important

Implementacion

Bootstrap utiliza la Metodologia BEM (Block Element Modifier) para asignar nombres a las diferentes clases. Esta metodologia consiste en identificar el nombre de un elemento contenedor (Bloque) y a cada uno de sus hijos nombre como parte del mismo(Element), asi como tambien utilizar variantes para estilos alternativos (Modifier)

Utilities

son clases generales que permiten la implemetacion de alguna propiedad css, como por ejemplo margin, padding o border. En la mayoria de los casos simplemente debemos colocar la o las letras que componen dicho atributo y luego un valor referencia en boostrap, normalmente expresado en rem.

Ejemplos: (Margenes y Padding)

  • m-0: 0
  • m-1: .25rem
  • m-2: .5rem
  • m-3: 1rem
  • m-4: 1.5rem
  • m-5: 3rem

Helpers

son clases que predefinen ciertas caractericas de algunos elementos, sin llegar a tratarse como componentes. Nomalmente se las asocia con alguna variable definida por el framework, siendo los casos mas comunes los colores. Tambien funcionan para quitar estilos predeterminados y definir alguna caractericas como un estilo particular de bordes.

Ejemplos: (Colores de Fondo)

  • primary
  • secondary
  • light
  • dark
  • danger
  • warning
  • success
  • info

Components

son clases que abarcan bloques contenedores y extienden su nombre a traves de sus elementos hijos, siguiendo la metodologia BEM, facilita el entendimiento de como funcionan dichos elementos y cuales son sus caracteristicas.

Ejemplos: (Lista Agrupada)

  • Block
  • Element
  • Modifier
