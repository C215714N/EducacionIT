Bootstrap

Responsive Web Design

Introduccion

Bootstrap es un framework de css, desarrollado por el equipo de twitter, que se enfoca en el diseño responsive. Nos ofrece una hoja de estilos que facilita la aplicacion de reglas mediante lo que se conoce como utilities, helpers y components. Otra de sus caracteristicas principales es sus sistema de grillas basado en porcentajes.

Funcionamiento Basico

Como se menciona mas arriba, las utilidades son clases que facilita la hoja de estilos de bootstrap para realizar cambios en el diseño de nuestra pagina mediante clases que son nombradas de la siguiente manera: attributo-valor y en la mayoria de los casos, se suele utilizar la letra inicial del attributo.

  • margin: 1rem; padding: .5rem; (.m-3 .p-2)
  • border: 1px solid; padding: 0 3rem; (.border .px-5)
  • border-left: 3px solid; margin-left: 1.5rem (.border-start .border-3 .ms-4)
  • border: 5px solid; padding: .5rem 0; (.border .border-5 .py-2)

las utilidades se aplican sobre los elementos utilizando clases, y en ciertas oportunidades vamos a recurrir a los que se llama clases de ayuda, que permiten modificar alguna caracteristica de un valor establecido. Las podemos diferenciar, porque se repite el nombre del atributo y agrega un valor adicional, como por ejempo: border border-dark

Asi como existen valores predefinidos para ciertos atributos como justify-content, text-align, hay ciertos valores que son definidos en el estilos. Bootstrap aplica unidades que van de 0 a 5 utilizando rem, en la mayoria de los casos, asi como tambien posee nombres de color predeterminados.

  • primary
  • secondary
  • light
  • dark
  • danger
  • warning
  • success
  • info

Sistema de Grillas Responsive

Una de las caracteristicas principales del framework es su sistema de grillas que utiliza flexbox y basa los tamaños de sus elementos en porcentajes. La misma se divide en 12 columnas en las que cada una representa un porcentaje del contenedor en el que se encuentre.

Breakpoints
name width max-width device
xs < 576px 100% smartwatch
sm ≥ 576px & < 768px 540px smartphones
md ≥ 768px & < 992px 720px tablets
lg ≥ 992px & < 1200px 960px laptops
xl ≥ 1200px & < 1400px 1140px Desktop
xxl ≥ 1400px 1320px HD Desktop

Metodologia BEM

El nombre de esta metodologia de aplicacion de clases corresponde a Block Element Modifier donde cada contenedor posee una clase principal que define el nombre de la clase de sus contenido, los elementos corresponden a partes individuales dentro de dicho contenedor y los modificadores corresponden a clases de ayuda que varian alguna de las caracteristicas del elemento afectado.

Tipos de Diseño

Fluid Design

Utiliza porcentajes para definir el tamaño de sus elementos, pero quedan desproporcionados en pantallas grandes o pequeñas, aunque se pueden establecer maximos y minimos.

Adaptive Design

Utiliza varias plantillas segun el tamaño del dispositivo en que se visualice la pagina, facil de mantener e ideal para clientes con bajo presupuesto, pero implica la duplicidad de contenido.

Responsive Design

Utiliza multimedia flexible, sistema de grillas basado en porcentajes y plantea el diseño a partir de la vista mobile y/o utilizando los texto finales, aplicando el concepto One Web for All
