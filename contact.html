Formularios HTML

Son herramientas que permiten la interaccion entre el cliente y el servidor (intercambio de datos). Para poder implementarlos solamente debemos colocar la etiqueta <form> en nuestra pagina y dentro de la misma ubicar los elementos que seran procesados al momento de realizar el envio

多como funcionan?

A la etiqueta mencionada anteriormente debemos colocarle los siguientes atributos, action="..." para definir que hacer una vez realizado el envio y method="..." para definir quien tiene la prioridad el momento de realizarlos.

  • GET metodo menos seguro debido a que la informacion permanece visible en la barra de direcciones al momento de realizar el envio, pero cuya prioridad radica en la respuesta del servidor.
    • Buscador Consulta Query
  • POST metodo seguro que encripta los datos al momento de realizar el envio, ya que la prioridad radica en la proteccion de los datos del usuario y por lo tanto es mas lento.
    • Registro Inscripcion
    • Acceso Inicio Sesion
    • Contacto Consulta Web

Tablas HTML

Son herramientas que permiten organizar la informacion de manera tabular, se organizan en filas, columnas y celdas igual que en cualquier otro ambito. Pero a diferencia de la mayoria de los programas, en HTML debemos crearlas respetando el modelo de cajas, es decir, utilizando una etiqueta contenedora <table> y dentro de la misma los contenedores de fila <tr> y las celdas de contenido. <td>

多Cual es su Estructura?

Estos elementos deben utilizarse para exponer informacion cuya visualizacion no debe alterarse. En HTML5 se reestructuro la forma de hacer tablas, debido a que se incorporo una estructura semantica para las mismas. Esto permite a los motores de busqueda una mayor facilidad para indentificar las partes de las mismas y su contenido, dentro de las cuales encontramos las siguientes etiquetas.

  • THEAD Cabecera de tabla que debe contener los titulos de la misma y en su interior solamente colocaremos celdas de encabezado th (table heading)
  • TBODY Cuerpo de tabla que contiene la informacion relevante para el usuario, suele estar compuesta por varias filas.
  • TFOOT Pie de tabla que contiene la informacion complementaria al cuerpo, como los totales, promedios, max, min, etc.

Contenido Multimedia

Actualmente los navegadores incorporan una API de reproduccion de contenido, podemos acceder a ellas utilizando las etiquetas <audio> y <video>, tambien podemos embeber contenido dentro de nuestra pagina gracias el elemento <iframe>

多Cuando Utilizarlos?

Las primeras etiquetas mencionadas anteriormente, soportan los formatos de audio .mp3, .wav, .ogg y de video .mp4, .webm, .ogv pero debemos definir ciertos atributos para poder utilizarlos, como controls para habilitar el reproductor.

Por otro lado la ultima etiqueta mencionada en el apartado anterior, permite la incorporacion, de manera asincrona, de contenido externo en nuestra pagina. Se utiliza como parte de una categoria web especialmente dise単ada para la compaticion de contenido, conocida como embed
