Formularios HTML

Son herramientas que permiten la interaccion entre cliente / servidor (intercambio de datos), con las cuales el usuario completa o selecciona datos dentro del mismo para que luego sean enviados a modo de peticion/solicitud. Para poder colocar un formulario dentro de la pagina debemos utilizar la etiqueta <form> y dentro de la misma colocar el contenido a ser procesado.

¿Como funcionan?

Dentro de la etiqueta menciona anteriormente debemos colocar al menos 2 atributos, entre ellos estan: action="..." que define lo que debe suceder despues que el usuario presione el boton de envio, normalmente se utiliza un script en lenguaje de servidor (por ejemplo: PHP) para que este ultimo resuelva que hacer con los datos; y method="..." que define como deben manipularse los datos que seran enviados al servidor, en este caso los mismos estan predefinidos y son los siguientes.

  • GET: Metodo menos seguro debido a que la informacion permanece visible en la barra de direcciones, es mas rapido ya que no encripta los datos y se utiliza cuando la prioridad radica en la respuesta del Servidor.
    • Buscador: consulta query
  • POST: Metodo mas seguro ya que antes de realizar el envio encripta los datos que van a ser enviados al servidor, debido a esto es mas lento ya que se debe codificar y decodificar la informacion. En este caso la prioridad radica en la proteccion de los datos del usuario
    • Registro: Inscripcion
    • Acceso: Inicio sesion
    • Contacto: Consulta Web

¿Como estan compuestos?

Para cualquiera de los tipos de formarios mencionados podemos encontrar 2 tipos de elementos, entre ellos estan: las etiquetas (LABEL) que se utilizan para indicar al usuario que datos debe ingresar o seleccionar dentro del formulario y tambien facilitan la accesibilidad, siempre y cuando esten vinculadas a un campo de formulario; por otro lado, los campos (INPUT / TEXTAREA / SELECT) se utilizan para facilitar al usuario la carga de datos, ya sea ingresandolos o seleccionandolos, estos poseen 2 atributos importantes: value="..." y type="...", los mismos pueden o no estar explicitamente.

Tablas HTML

Son elementos que permiten organizar la informacion de manera tabular, deben utizarse siempre y cuando los datos que se mostraran en pantalla requieren de cierta estructura que no debe se modificada por los cambios que pueda llegar a tener la pagina a nivel visual. Para crear una tabla debemos utilizar la etiqueta <table> y en su interior colocar la estructura de la misma.

¿Cual es su estructura?

A la etiqueta mencionada anteriormente, debemos colocarle 2 elementos que son las filas (TR) y las celdas (TD) respetando el modelo de cajas, es decir, utilizando el primero como contenedor del segundo. Ademas las mismas poseen una estructura semantica como se muestra a continuacion.

  • THEAD corresponde a la cabecera de la tabla, donde colocamos los titulos de las columnas utilizando la etiqueta <th> para los encabezados
  • TBODY correponde al cuerpo de la tabla, donde se coloca la informacion principal, distribuida en multiples filas e incluso puede poseer titulos.
  • TFOOT corresponde al pie de la tabla, donde colocamos la informacion complementaria al cuerpo, como totales, promedios o porcentajes, entre otros.
