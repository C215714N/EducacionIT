Formularios HTML

Son herramientas que permiten la interaccion entre el cliente y el servidor (intercambio de datos). Para poder implementarlos solamente debemos colocar la etiqueta <form> en nuestra pagina y dentro de la misma ubicar los elementos que seran procesados al momento de realizar el envio

Â¿como funcionan?

A la etiqueta mencionada anteriormente debemos colocarle los siguientes atributos, action="..." para definir que hacer una vez realizado el envio y method="..." para definir quien tiene la prioridad el momento de realizarlos.

  • GET metodo menos seguro debido a que la informacion permanece visible en la barra de direcciones al momento de realizar el envio, pero cuya prioridad radica en la respuesta del servidor.
    • Buscador Consulta Query
  • POST metodo seguro que encripta los datos al momento de realizar el envio, ya que la prioridad radica en la proteccion de los datos del usuario y por lo tanto es mas lento.
    • Registro Inscripcion
    • Acceso Inicio Sesion
    • Contacto Consulta Web
