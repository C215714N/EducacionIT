Formularios HTML

Son herramientas que permiten una interaccion (intercambio de datos) entre el cliente y servidor, se utilizan para solicitar informacion y enviar consultas a este ultimo. Para poder implementarlos debemos colocar la etiqueta <form> y dentro de la misma los elementos que lo componen, ya que al realizar el envio de los datos, solamente se tomara en cuenta el contenido del mismo.

¿Como funcionan?

A la etiqueta mencionada anteriormente, se le debe especificar al menos 2 atributos, por un lado esta action="..." que define lo que se debe hacer con los datos una vez enviado el formulario, se suele utilizar un script de backend (ej:php) para que el servidor resuelva esto; y por otro lado esta method="..." que define como se van a procesar los datos.

  • GET Metodo menos seguro, ya que los datos siempre estan visibles en la barra de direcciones al momento de hacer la consulta y que prioriza la respuesta del servidor.
    • Buscador - Consulta Query
  • POST Metodo mas seguro que encripta los datos al momento de realizar el envio, dando prioridad de este modo a la informacion provista por el usuario y debido a esto es mas lento.
    • Inscripcion - Registro
    • Acceso - Inicio Sesion
    • Contacto - Consulta Web

¿Como estan compuestos?

Los formularios poseen 2 tipos de elementos, por un lado estan las Etiquetas (LABEL) que funcionan como rotulos que indican al usuario de datos debe seleccionar o escribir; y por el otro estan los Campos (INPUT / TEXTAREA / SELECT) que permiten al usuarios seleccionar/ingresar los datos, estos poseen 2 atributos importantes, value="..." que hace referencia al dato que sera enviado y type="..." que se refiere al tipo de dato.
