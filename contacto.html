Formularios HTML

Son herramientas que se utilizan para intercambio de datos entre Cliente y Servidor, permitiendo al usuario ingresar o seleccionar los mismos antes de realizar el envio. Para implementarlos debemos colocar la etiqueta <form> y dentro de esta los elementos que se contemplaran para el envio, ademas se deben especificar algunos atributos relacionados con la manipulacion de los datos.

¿Como funcionan?

A la etiqueta mencionada anteriormente deberemos especificarle 2 atributos, por un lado esta el action="..." que define que hacer con los datos una vez realizado el envio y se suele utilizar un script en lenguaje de servidor(e.g:php) para que este resuelva que hacer con los mismos. Por otro lado esta el method="..." el cual indica como se llevara a cabo el envio y quien tiene la prioridad al momento de realizarlo.

  • GET: Metodo rapido cuya prioridad radica en la respuesta del servidor, los datos enviados suelen figurar en la barra de direcciones y la comunicacion es en texto plano, por lo tanto no es seguro.
    • Buscadores Consulta Query
  • POST: Metodo seguro cuya prioridad radica en el cliente que realiza la peticion, los datos se encriptan antes de realizar el envio para proteger la informacion, por esta razon es mas lento.
    • Inscripcion Registro
    • Inicio sesion Acceso
    • Contacto Consulta Web

Tablas HTML

Son herramientas que permiten organizar la informacion de manera tabular y deben utilizarse cuando los datos requieren ser presentados de determinada forma. A diferencia de otros ambitos, en html las tablas se crean respetando el modelo de cajas, es decir, que existen etiquetas que funcionan como contenedores y otras que funcionan como contenido.

¿Cual es su estructura?

Para insertar una tabla debemos colocar la etiqueta <table>, para crear las filas utilizamos <tr> (Table Row); tambien utilizamos <td> (Table Division) y <th> (Table Heading) para las celdas de contenido y titulo respectivamente. Ademas contamos con una estructura semantica, que permite definir que parte de la tabla estamos visualizando como THEAD que representa la cabecera, TBODY que representa el cuerpo y TFOOT que representa el pie de tabla, este ultimo es opcional.
