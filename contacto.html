Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente/servidor mediante la interaccion del usuario con el sitio. Para poder implementarlos colocamos la etiqueta <form> y cuando se realice el envio del mismo solamente se contemplara el contenido de la etiqueta y los atributos de la misma.

多Como funcionan?

A la etiqueta previamente mencionada, se le deben colocar al menos 2 atributos, por un lado esta el action="..." que indica al navegador que hacer cuando se realiza el envio, por lo general se utiliza un script en lenguaje de servidor, por ejemplo php. Por otro lado existe el atributo method="..." que define la prioridad de los datos y como se van a procesar los mismos, existen 2 valores posibles que podemos utilizar.

  • GET Metodo rapido que permite el envio de datos a traves de texto plano, permitiendo la visualizacion de los datos ingresados en el formulario desde la URL y la prioridad radica en el SERVIDOR
    • Buscador - Consulta Query
  • POST Metodo seguro que encripta los datos antes de realizar el envio, protegiendolos durante la comunicacion, ya que la prioridad en este caso radica en el CLIENTE
    • Registro - Inscripcion
    • Acceso - Inicio Sesion
    • Contacto - Consulta Web

Tablas HTML

Son herramientas que permiten organizar la informacion de manera tabular y, a diferencia de otros ambitos, se crean respetando el modelo de cajas, es decir, utilizando elementos contenedores y de contenido. Para crear una tabla debemos utilizar la etiqueta <table>, dentro de la misma colocaremos las filas <tr> y dentro de estas pondremos las celdas <td>

Contenido Multimedia

En la actualidad los navegadores incorporan API's de reproduccion de contenido y podemos acceder a estas con las etiquetas <audio y <video>. Estas funcionan de manera similar a la etiqueta <img> debido a que solamente podemos reproducir contenido compatible con estos elementos.

多Contenido Embebido?

Existe una categoria conocida como embed que se utiliza para definir un contenido que no pertenece al sitio o incluso al universo hmtl, pero que fue compartido desde otra direccion. Normalmente este contenido se agrega mediante la etiqueta <iframe> que permite la carga asincronica de contenido.

多Multimedia sin Plugins?

En la actualidad los navegadores incorporan API's de reproduccion de contenido multimedia y podemos acceder a ellas gracias a las etiquetas <audio> y <video> que son compatibles con los formatos .mp3 - .wav- .ogg y .mp4 - .webm - .ogv respectivamente. Tambien podemos utilizar atributos como autoplay muted loop para generar fondos con dichos elementos.
