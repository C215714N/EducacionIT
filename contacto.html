Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente y servidor, para implementarlos debemos utilizar la etiqueta <form> y solamente se tendra en cuenta para el envio, los datos que se encuentren dentro de la misma. Cabe destacar que es necesario especifar algunos atributos para definir como se deben manipular los datos y que se debe hacer con los mismos.

¿Como funcionan?

La etiqueta mencionada anteriormente posee 2 atributos necesarios para la manipulacion de datos, por un lado se encuentra action="..." que indica al navegador cual es la operacion a realizar cuando el usuario presiona el boton enviar, en la actualidad se utilizan scripts en lenguaje de servidor (ej: php) para que este ultimo resuelva que hacer; por otro lado esta method="..." que recibe al menos 2 valores segun la prioridad que deba darse a los datos y su respuesta respectivamente.

  • GET: Metodo que se utiliza para obtener informacion del servidor, la prioridad radica en la respuesta de este ultimo y debido a esto es bastante rapido, pero menos seguro debido a que los datos se envian en texto plano y siempre estan visibles en la barra de direcciones.
    • Buscadores - Consulta Query
  • POST: Metodo que se utiliza para enviar informacion al servidor de manera segura, ya que la prioridad radica en el cliente. Se encriptan los datos para la comunicacion, pero es mas lento debido a esto.
    • Inscripcion - Registro
    • Acceso - Inicio Sesion
    • Consulta Web - Contacto
