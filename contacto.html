Formularios HTML

¿Para que se utilizan?

Son herramientas que permiten el intercambio de datos entre cliente y servidor (interaccion real). Se utilizan para enviar datos a un servidor, asi este luego decide que hacer con dicha informacion; aunque tambien se los utiliza para obtener informacion de los mismos. Para hacer uso de ellos, dentro de la pagina, debemos implementar la etiqueta <FORM> y colocar en su interior los elementos que se procesaran a la hora de realizar el envio.

  • Busqueda - Consulta Query
  • Inscripcion - Registro
  • Acceso - Inicio Sesion
  • Contacto - Consulta Web

¿Como funcionan?

Dentro de la etiqueta mencionada anteriormente debemos colocar por los menos 2 atributos. Por un lado, tenemos action="..." que determina que hacer con el formulario cuando se realice el envio; actualmente es comun utilizar un script en lenguaje de servidor (PHP) para que este ultimo resuelva que hacer con los datos recibidos. Por el otro lado, tenemos el atributo method="..." que recibe 2 valores, GET que lo utilizamos cuando la prioridad esta en la respuesta del servidor, es menos seguro, mas rapido y los datos siempren estan accesibles (barra de direcciones); y POST que lo utilizamos cuando la prioridad esta en los datos enviados por el cliente, es mas seguro (encripta los datos) pero es un poco mas lento.

¿Que elementos los componen?

Dentro de los formularios vamos a encontrar 2 tipos de elementos, las Etiquetas(LABEL) que funcionan a modo de rotulos, indicando al usuario que datos debe colocar o seleccionar dentro del formulario, tambien facilita la accesibilidad; por el otro lado, tambien tenemos los Campos(INPUT/TEXTAREA/SELECT) que son aquellos que permiten al usuario ingresar o seleccionar los datos. Cabe mencionar que, mayoritariamente, los campos de formularios poseen 2 atributos fundamentales, type="..." que indica el tipo de valor a ingresar/seleccionar y value="..." que hace referencia al valor previamente mencionado.
