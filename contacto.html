Formularios HTML

¿Para qué sirven?

Son herramientas que se utilizan para el intercambio de datos entre cliente y servidor , para poder utilizarlos debemos colocar dentro de la pagina la etiqueta contenedora <FORM> y dentro los elementos que se describiran mas adelante. Nos referimos a una interaccion real debido a que otros comportamientos como el :hover son preparados y la respuesta es siempre la misma, en cambios con los formularios, segun el tipo, la respuesta varia en funcion de los datos.

  • Buscador - Consulta Query
  • Inscripcion - Registro
  • Inicio sesion - Acceso
  • Contacto - Consulta Web

¿Como Funcionan?

La etiqueta mencionada anteriormente debe poseer al menos 2 atributos, por un lado, se encuentra action="..." que permite indicar al navegador que hacer una vez completado el formulario. Antiguamente se utilizaba el valor mailto:usuario@servidor pero dependemos que el cliente posea una aplicacion de correos, por esta razon hoy en dia se utiliza un script en lenguaje de servidor (PHP) para que este ultimo resuelva que hacer con los datos enviados a traves del formulario. Por otro lado, se encuentra el atributo method que indica de que manera se van a procesar los datos y como se realizara el envio.

  • GET: Metodo que se utiliza para obtener informacion de un servidor, es menos seguro ya que los datos son accesibles desde la barra de direcciones, pero mas rapido debido a que prioriza la respuesta del servidor.
  • POST: Metodo que se utiliza para enviar datos a un servidor, es mas seguro debido a que encripta los datos, pero es mas lento ya que prioriza los datos del usuarios y la proteccion de los mismos.

¿Cuales son sus componentes?

Los formularios estan compuestos por 2 elementos diferentes, por un lado tenemos las Etiquetas (LABEL) que cumplen la funcion de rotulos para guiar al usuario a la hora de completar/seleccionar los datos y tambien facilicitan la accesibilidad. Por el otro encontramos a los Campo (INPUT, SELECT, TEXTAREA) que representan las areas donde el usuario puede ingresar/seleccionar los datos, estos elementos poseen 2 atributos escenciales que son type="..." para determinar el tipo de dato que se debe ingresar/seleccionar y value="..." que se refiere al dato propiamente dicho.

Tablas HTML

¿Cuando utilizarlas?

Son elementos que permiten organizar la informacion de manera tabular y se utilizan cuando los datos representados en pantalla deben respetar cierta estructura visual. Al igual que en cualquie ambito las mismas se componen de filas y columnas, pero a diferencia de cualquier otra area, en HTML se crean respetando la estructura de cajas utilizando la etiqueta <TABLE> con los elementos que se definiran a continuacion.

¿Como estan compuestas?

Como se menciono anteriormente, las tablas estan compuesta por celdas (<td>), filas (<tr>) y columnas pero al diseñarlas no podemos utilizar todos estos elementos y a diferencia de las otras etiquetas de bloque, las tablas se adaptan a su contenido. Para facilitar la implementacion de por parte de los motores de busqueda la W3c elaboro una estructura semantica para dichos elementos.

  • THEAD: Cabecera de tabla, donde deben colocarse los titulos y en lugar de usar celdas comunes se utilizan celdas de encabezado (<TH>)
  • TBODY: Cuerpo de tabla y unica parte de la estructura semantica que mantiene la estructura clasica utilizando filas (<tr>) y celdas (<td>), aunque tambien en ciertas ocasiones se pueden usar celdas de encabezado (<th>)
  • TFOOT: Pie de tabla o informacion complementaria al cuerpo, se utiliza para colocar un detalle como el total o el promedio de los valores expuesto en el cuerpo de tabla, este elemento es opcional.
