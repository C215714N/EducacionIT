Formularios HTML

Son elementos que permiten una interaccion real (intercambio de datos) entre cliente y servidor , ya que permiten al usuario intervenir en la pagina mediante completado y seleccion de opciones. El servidor tras recibir la solicitud del cliente elabora una respuesta, que puede ser inmediata o no, y la misma dependera del tipo de formulario y los datos que haya ingresado el usuario.

Para poder utilizar dicho elemento es necesario implementar la etiqueta <form> donde deberemos especificar determinados atributos para llevar a cabo su funcionamiento. Cuando el mismo se envia, todos los elementos dentro de la etiqueta seran procesados para la realizacion de la respuesta por parte del servidor.

多Como funcionan?

Estos elementos requieren minimamente de dos atributos, siendo uno action="..." que permite definir que sucedera con los datos ingresados en el formulario tras realizarse el envio de los mismo. Antiguamente, debido a las bajas velocidades de conexion era comun que las personas utilizaran aplicacion de correo como por ejemplo outlook y descargaran los mensajes, por lo tanto se utilizaba el valor action="mailto:usuario@dominio"; pero hoy en dia debido a que la mayoria de los usuario utiliza un servidor webmail, es comun utilizar un script en lenguaje de servidor como php dando como valor action="contacto.php" ya que a diferencia del caso anterior, es el servidor quien resuelve que hacer con los datos y no la computadora cliente.

Por otro lado, el atributo method="..." permite definir la prioridad de los datos y como se va a realizar el envio de los mismo. Este atributo recibe dos valores, uno es method="GET" el cual es un metodo poco seguro, los datos estan accesibles desde la barra de direcciones y admite hasta 8kb, la prioridad recide en la respuesta del servidor. En cambio el valor method="POST" es mas seguro debido a que encripta los datos, los mismo no son visibles desde la barra de direcciones y admite hasta 8mb, siendo los datos del cliente en este caso la prioridad

多Como estan compuestos?

Los formularios poseen dos tipos de elementos, en primaria instancia estan las Etiquetas (label) que se utilizan para indicar al usuario el tipo de dato que debe ingresar o seleccionar de una lista de opciones, tambien se utiliza para el SEO y permite a los lectores de pantalla leer su contenido cuando se pase el mouse por encima. Por otro lado, estan los Campos (input - textarea - select) que son aquellos elementos con los que el usuario interactua y se clasifican segun los tipos de datos que se indiquen; uno de los elementos que mas veces se suele implementar en un formulario es <input> ya que representa un campo monolinea en el que se puede especificar valores o tipos de datos a ingresar, segun corresponda.
