Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente/servidor mediante la interaccion del usuario con el sitio. Para poder implementarlos colocamos la etiqueta <form> y cuando se realice el envio del mismo solamente se contemplara el contenido de la etiqueta y los atributos de la misma.

¿Como funcionan?

A la etiqueta previamente mencionada, se le deben colocar al menos 2 atributos, por un lado esta el action="..." que indica al navegador que hacer cuando se realiza el envio, por lo general se utiliza un script en lenguaje de servidor, por ejemplo php. Por otro lado existe el atributo method="..." que define la prioridad de los datos y como se van a procesar los mismos, existen 2 valores posibles que podemos utilizar.

  • GET Metodo rapido que permite el envio de datos a traves de texto plano, permitiendo la visualizacion de los datos ingresados en el formulario desde la URL y la prioridad radica en el SERVIDOR
    • Buscador - Consulta Query
  • POST Metodo seguro que encripta los datos antes de realizar el envio, protegiendolos durante la comunicacion, ya que la prioridad en este caso radica en el CLIENTE
    • Registro - Inscripcion
    • Acceso - Inicio Sesion
    • Contacto - Consulta Web
