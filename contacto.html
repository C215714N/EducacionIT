Formularios HTML

多Para que se utilizan?

Son herramientas que permiten una interaccion real entre cliente y servidor (intercambio de datos) y se utilizan cuando se necesita obtener, enviar, actualizar o eliminar datos de un servidor. Para implementarlos dentro de una pagina es necesario utilizar la etiqueta <FORM> ademas de algunos atributos y elementos que se mencionan mas adelante.

  • Buscador - Consulta Query
  • Inscripcion - Registro
  • Acceso - Inicio Sesion
  • Contacto - Consulta Web

多Como funcionan?

El elemento anterior debe contar con al menos 2 atributos, por un lado, se encuentra el action="..." que define que sucedera con los datos una vez realizado el envio y en la actualidad se utiliza como valor de este un script en lenguaje de servidor (ej: php). Por el otro, se encuentra el method="..." que indica como se van a manipular los datos y cual es la prioridad en el proceso.

  • GET: Metodo menos seguro debido a que los datos siempre estan accesibles desde la barra de direcciones, pero de respuesta mucho mas rapida ya que prioriza los datos provistos por parte del servidor.
  • POST: Metodo mas seguro debido a que protege los datos que se envian al servidor, pero de respuesta mas lenta y no necesariamente inmediata debido al proceso de encriptacion desencriptacion.

多Cuales son sus elementos?

Los formularios estan compuestos por 2 tipos de elementos, por un lado tenemos las Etiquetas (LABEL) que se utilizan para guiar al usuario al momento de completar/seleccionar los datos en el formulario, tambien facilita la accesibilidad; y por el otro, encontramos los Campos(INPUT / TEXTAREA / SELECT) que permiten al usuario seleccionar/ingresar datos dentro del formulario y por ende la pagina, poseen 2 atributos que son type="..." que indica el tipo de dato a ingresar y value="..." que corresponde al valor ingresado.
