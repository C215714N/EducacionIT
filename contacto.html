Formularios HTML

Son elementos que permiten una interaccion real entre el Cliente y el Servidor , se utilizan para el intercambio de datos entre estos y a diferencia de otros elementos dentro de la pagina el comportamiento y la respuesta de los mismos difiere segun el tipo de formulario y la informacion completada por el usuario . En resumen, en formulario permite realiza una peticion y este en base a la solicitud realizada elabora una respuesta .

多Cuantos tipos existen?

Se pueden clasificar dependiendo del valor correspondiente al atributo METHOD="..." ingresado dentro de la etiqueta FORM , que se utiliza para definir cuales van a ser los datos que van a enviar al Servidor. Por un lado tenemos method="GET", que permite adjuntar en el formulario hasta 8kb de informacion, se lo considera un metodo poco seguro poque NO encripta los datos del usuario y ademas estos estaran visibles en la barra de direcciones, ya que Prioriza los datos enviados desde el servidor como respuesta . Por otro lado se encuentra el method="POST", el cual es utilizado en la mayoria de los formularios, ya que admite encriptacion (cifra los datos del usuario) y por lo tanto brinda mayor seguridad a la hora de utilizar informacion privada o personal en los mismos, permite el envio de hasta 8mb de informacion, por lo que es idel para el envio de archivos.

多Como estan compuestos?

Dentro de la etiqueta de formulario, mencionada anteriormente, encontraremos dos tipos de elementos. Por un lado estan las etiquetas (LABEL) que se utilizan para guiar al usuario a la hora de completar o seleccionar los datos dentro del formulario, asi como tambien son utilizadas por los lectores de pantalla para leer el texto en su interior cuando se selecciona el otro tipo de elemento, conocido tambien como Campo (INPUT - TEXTAREA - SELECT) . A diferencia de las etiquetas, los campos se utilizan para permitir al usuario interactuar con la pagina, ya sea para ingresar datos o seleccionarlos, destacando como fundamental el atributo value="" el cual es el utilizado para el envio de informacion, siendo INPUT el tipo de campo mas utilizado.
