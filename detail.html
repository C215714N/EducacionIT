Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente y servidor, para utilizarlos dentro de una pagina web debemos colocar la etiqueta <form> y dentro de la misma, todo elemento que deba ser contemplado al momento de realizar el envio del mismo. Cualquier etiqueta que no se encuentre dentro del formulario, no sera tomada en cuenta para realizar el envio.

¿Como funcionan?

A la etiqueta mencionada anteriormente debemos especificarle 2 atributos, action="..." que indica lo que debe ocurrir tras enviar el formularios; y method="..." que indica quien tiene la prioridad al realizar el envio y los valores mas utilizados son:

  • GET: Metodo rapido debido a que prioriza la respuesta del servidor, los datos permanecen visibles en la barra de direcciones y se utiliza principalmente para buscadores.
  • POST: Metodo seguro que prioriza la seguridad de los datos del usuario, ya que se cifran durante el envio. Se utiliza para formularios de Registro, Acceso, Contacto, etc...

¿Como estan compuestos?

Dentro de un formulario encontraremos diferentes elementos, aunque algunos son exclusivos de estas herramientas, entres se encuentran las etiquetas (LABEL) que funcionan a modo de rotulos indicando al usuario lo que debe seleccionar o completar; y los campos (INPUT / TEXTAREA / SELECT) que permiten al usuario ingresar o seleccionar los datos solicitados, entre los atributos mas comunes de estas etiquetas se encuentran:

  • name="..." indica el grupo de pertenencia del campo, necesario para capturar los datos e identificar los datos.
  • value="..." corresponde al dato propiamente dicho y debe estar si el usuario solamente selecciona una opcion.
  • type="..." indica como podemos completar el campo corresponde, que valores podemos utilizar.

Contenido Multimedia

Corresponde a todo elemento grafico y audiovisual que podemos encontrar dentro de una pagina web. Antiguamente eran proco frecuentes debido a las limitaciones del ancho de banda y los navegadores de la epoca no soportaban algunos de estos elementos de forma nativa, es por eso que se recurria a la instalacion de plugins o complementos de terceros.

¿Contenido Embebido?

Se conoce con este nombre a una categoria web que se utiliza para indicar que un contenido fue incorporado dentro de una pagina con autorizacion del autor/publicador original. Normalmente se implementa mediante la etiqueta <iframe>(Marco Flotante) que permite la carga de contenido de manera asincrona.

¿Multimedia sin plugins?

Como se hace mencion al principio, actualmente los navegadores no necesitan programas externo para reproducir este material, ya que incorporan una API de reproduccion y podemos acceder a la misma mediante las etiquetas <audio> y <video> que son compatibles con los formatos .mp3, .wav, .ogg y .mp4, .webm, .ogv respectivamente.

Tablas HTML

Son elementos que se utilizan para organizar la informacion de manera tabular, se implementan en estructuras de datos, pero no deben utilizarse para diseñar la pagina, ya que evitan que los elementos dentro de ellas se amolden a las dimensiones de la ventana. Para crear una tabla debemos colocar la etiqueta <table> y dentro de la misma, las filas con la etiqueta <tr> (table row), asi como tambien las celdas de encabezado y contenido con <th>(table heading) y <th> (table division)

¿Poseen estructura semantica?

Como se menciona en el texto anterior, las tablas se crean respetando el modelo de cajas, es decir, que utilizaremos etiquetas contenedoras para las filas y etiquetas de contenido para las celdas, pero a pesar de esto el navegador no entenderia dicha estructura. Es por esta razon que en HTML5 se incorporaron elementos semanticos a las mismas, los cuales son:

  • THEAD: Cabecera de tabla que contiene la fila de encabezados.
  • TBODY: Cuerpo de tabla que principalmente contiene las filas de datos.
  • TFOOT: Pie de tabla que contiene una fila complementaria a los datos.

¿Expansion de celdas?

En caso de ser necesario podemos aumentar la cantidad de filas o columnas que ocupa una celda mediante los atributos rowspan="..." y colspan="..." respectivamente. Cuando hacemos esto debemos tener en cuenta que las celdas afectadas tras haber realizado esto seran desplazadas hacia la derecha.
