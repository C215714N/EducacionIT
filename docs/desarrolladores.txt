FrontEnd(cliente){
	- Desarrollo del lado del Cliente.
	- Aspecto o caracteristica que puede se reproducido por el navegador y que no depende de un servidor para poder llevarse a cabo dicha accion (UI / UX / Funcionalidades Basicas).

	Lenguajes:
		HTML	Estructura
		CSS		Dise√±o
		JS		Funcionalidad
}
BackEnd(servidor){
	- Desarrollo del lado del Usuario
	- Aspecto o caracteristica que para llevarse a cabo depende de la confirmacion de un Servidor ( Administracion / Solicitudes / manipulacion datos)

	Lenguajes:
		SQL		Base Datos
		PHP		Web Dinamicas
		JAVA	Aplicaciones Web
}
FullStack(cliente, servidor){
	- Desarrollo de ambos lados
	- Depuracion, optimizacion de codigo, liderazgo y organizacion de equipos de trabajo (Proyecto Web)

	Lenguajes:
		Todos los anteriores
}
