HTML (HyperText Markup Language){
	Lenguaje de Marcado de HiperTexto (ESTRUCTURA)

	Utiliza etiquetas para definir la funcion que cumple cada elemento dentro de la pagina con relacion a su contexto.
		Ejemplo: <p> soy un parrafo </p>
		El ORDEN de los factores ALTERA el PRODUCTO.
}
CSS(Cascading Styles Sheets){
	Hojas de Estilo en Cascada (DISEÃ‘O GRAFICO)

	Utiliza selectores y reglas para definir la apariencia de los elementos dentro de la pagina.
		Ejemplo: p{ color: blue; }
		El ORDEN de los factores NO ALTERA el PRODUCTO.
}
