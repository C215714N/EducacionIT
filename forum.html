Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente y servidor. Para incorporarlos dentro de la pagina solamente debemos utilizar la etiqueta <form> y dentro de la misma colocar los elementos que seran procesados para el envio de los datos.

多Como funcionan?

A la etiqueta mencionada anteriormente debemos colocarle al menos 2 atributos. action="..." que indica lo que debe suceder cuando se realiza el envio; y method="..." que especifica como se realiza el mismo. Los valores mas comunes de method son los siguientes:

  • GET: la prioridad radica en la respuesta del servidor, los datos se envian en texto plano y la informacion es visible desde la barra de direcciones. Este metodo se utiliza en buscadores.
  • POST: la prioridad es proteger los datos enviados, por eso se cifran antes de realizar el envio, y por lo tanto es mas lento. Este metodo se utiliza en formularios de registro, acceso, contacto, etc...

多Como estan compuestos?

Dentro de un formulario podemos colocar cualquier etiqueta existente, pero hay 2 elementos fundamentales, por un lado estan las Etiquetas (LABEL) que indican que debe completar o seleccionar el usuario y ademas facilitan la accesibilidad; y por el otro estan los campos (INPUT / TEXTAREA / SELECT) que corresponden a los elementos con los que debe interactuar el usuario, estos poseen las siguientes caracteristicas.

  • name="..." corresponde al nombre que el se identificaran los valores enviados, si varios campos corresponden a la misma opcion, pueden compartir dicho valor.
  • type="..." corresponde al tipo de dato que se debe ingresar o seleccionar en el campo de formulario, es necesario utilizarlo en la etiquetas input.
  • value="..." corresponde al dato ingresado por el usuario y es implicito, en caso que este seleccione, debe figurar como atributo de la etiqueta en cuestion.

Tablas HTML

Son Elementos que permiten organizar la informacion de manera tabular y deben ser utilizadas cuando necesitamos estructuras los datos de cierta manera. Para incorporarlas dentro de la pagina simplemente debemos utilizar la etiqueta <table>y dentro de la misma, una estructura de elementos respentando el modelo de cajas.

多Poseen Estructura Semantica?

Dentro de la etiqueta principal, mencionada anteriormente, debemos colocar las filas (table row) y dentro de las estas, las Celdas (table division) utilizando las etiquetas <tr> y <td> respectivamente; pero tambien podemos especificar que parte de la tabla estamos completando, utilizando la siguiente estructura de etiquetas:

  • THEAD: Cabecera de tabla que contiene la fila de encabezados, creados con la etiqueta <th> (table heading)
  • TBODY: Cuerpo de tabla que contiene los datos y generalmente esta compuesto por varias filas.
  • TFOOT: Pie de Tabla que posee informacion complementaria, como totales, porcentajes o promedios y es opcional.

多Combinacion de Celdas?

En ciertas ocasiones, puede que necesitemos crear una tabla cuyas celdas ocupen mas de una fila o una columna, para ello podemos utilizar los atributos colspan="..." y rowpan="..." para indicar que una celda debe ocupar x cantidad de columnas o filas respectivamente. solamente debemmos tener en cuenta que todo contenido posterior sera desplazado hacia la derecha.
