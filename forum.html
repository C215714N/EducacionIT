Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente y servidor. Para incorporarlos dentro de la pagina solamente debemos utilizar la etiqueta <form> y dentro de la misma colocar los elementos que seran procesados para el envio de los datos.

¿Como funcionan?

A la etiqueta mencionada anteriormente debemos colocarle al menos 2 atributos. action="..." que indica lo que debe suceder cuando se realiza el envio; y method="..." que especifica como se realiza el mismo. Los valores mas comunes de method son los siguientes:

  • GET: la prioridad radica en la respuesta del servidor, los datos se envian en texto plano y la informacion es visible desde la barra de direcciones. Este metodo se utiliza en buscadores.
  • POST: la prioridad es proteger los datos enviados, por eso se cifran antes de realizar el envio, y por lo tanto es mas lento. Este metodo se utiliza en formularios de registro, acceso, contacto, etc...
