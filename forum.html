Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente y servidor permitiendo al usuario el ingreso de datos o manipulacion dentro de la pagina. Para poder utilizarlos debemos colocar la etiqueta <form> y dentro de las misma los elementos que formaran parte del mismo al momento de realizar el envio.

多Como funcionan?

A la etiqueta mencionada anteriormente debemos especificarle los atributos action="..." que indica lo que debe suceder cuando realizamos el envio, normalmente se utiliza un script en lenguaje de servidor por ejemplo: PHP para que este resuelva que hacer con los datos; y method="..." que indica la manera en la que realizara el mismo, definiendo quien tiene la prioridad. Entre los metodos mas utilizados encontramos los siguientes:

  • GET: metodo menos seguro, debido a que los datos ingresos permanecen visibles en la barra de direcciones. La prioridad radica en la respuesta del servidor, los datos se envian en texto plano y por lo tanto es mucho mas rapido. Este metodo se utiliza principalmente en Buscadores - Consulta Query
  • POST: metodo mas seguro, ya que los datos se encriptan antes de realizar el envio, los mismos no figuran en ningun otro lado que no sea el formulario y la prioridad radica en la seguridad de los datos del cliente. Por estos motivos es el mas utilizado principalmente para formularios de Registro - Inscripcion, Acceso - Inicio de sesion, Contacto - Consulta Web

多Como se componen?

Los formularios poseen 2 tipos de elementos, por un lado estan las etiquetas (LABEL) que se utilizan a modo de rotulo indicando al usuario que debe ingresar/seleccionar, ademas facilitan la accesibilidad; y por otro lado estan los campos (INPUT / TEXTAREA / SELECT) que corresponden a los elementos donde el usuario ingresa / selecciona los datos. Estos ultimos poseen ciertas caracteristicas que en algunos casos se encuentran definidas de manera implicita como las siguientes:

  • type="..." atributo que indica el tipo de dato admitido por el control, entre ellos se encuentran valores como text, number, email, password, date que permiten el ingreso de datos, pero tambien existen valores como radio o checkbox que limitan al usuario a seleccionar una opcion.
  • value="..." atributo que indica cual es el valor del campo que sera utilizado al momento de enviar el formulario, normalmente corresponde al dato ingresado por el usuario, pero en aquellos casos donde este no puede escribir debe estar definido por el desarrollador para que se reciba algo.
