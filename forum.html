Formularios HTML

Son herramientas que permiten el intercambio de datos entre el cliente y el servidor, mediante la intervencion del usuario. Para poder utilizarlos dentro de la pagina debemos colocar la etiqueta <form> y solamente se considerara para el envio lo que se encuentre dentro de la misma.

  • GET: Metodo que prioriza la respuesta del servidor, La informacion permanece visible en la barra de direcciones, Se utiliza principalmente para formularios de busqueda
  • POST: Metodo seguro que prioriza la seguridad de los datos del usuario, Se realiza el cifrado de los datos antes y despues del envio, Se utiliza para formularios de registro, acceso, contacto, etc...

¿Cuales son sus elementos?

Algunos de los elementos necesarios dentro de un formulario son las Etiquetas (LABEL) que funcionan a modo de rotulos, indicando al usuario que datos debe completar o seleccionar, y los campos (INPUT / TEXTAREA / SELECT) que son los elementos donde el usuario va a seleccionar o completar con sus datos.

  • value="..." Corresponde al dato propiamente dicho, debe existir si el usuario no escribe dentro del formulario.
  • type="..." Indica que criterios tiene el campo para su validacion y completado, es decir, que caracteres permite ingresar.

Tablas HTML

Estructura que permite organizar la informacion de manera tabular y se utiliza cuando debemos representar datos en patalla respetando ciertos criterios. Para colocarla dentro de la pagina debemos utilizar la etiqueta <table> y agregar las filas y columnas, respetando el modelo de cajas.

¿Cual es su estructura?

Dentro de la etiqueta mencionada anteriormente, debemos colocar las filas utilizando la etiqueta <tr> (Table Row) y luego dentro de las mismas colocar las celdas <th> (Table Heading) si son encabezados o <td> (Table division) si son datos.

En la ultima version de HTML se agregaron etiquetas que permiten definir una Estructura semantica para las tablas, de esta manera los motores de busqueda pueden analizar su Contenido de manera mas facil, ademas que facilitan la aplicacion de ciertos estilos. Dichos elementos son los siguientes:

  • THEAD: Cabecera de tabla, contiene las celdas de encabezado.
  • TBODY: Cuerpo de tabla, compuesto por varias filas y contiene los datos.
  • TFOOT: Pie de tabla, contiene datos que complementan al cuerpo.
