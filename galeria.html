Contenido Multimedia

Audio

Sonidos en la Web
(OGG - WAV - MP3)

¿Que es OGG?

Operation Good Guys, formato contenedor libre y abierto, desarrollado por la Fundación Xiph.Org . No está restringido por patentes de software y está diseñado para proporcionar una difusión de flujo eficiente y manipulación de multimedios digitales de alta calidad.

¿Que es WAV?

Waveform Audio File Format es un formato de audio digital con o sin compresión de datos desarrollado por Microsoft e IBM que se utiliza para almacenar flujos digitales de audio en el PC, mono y estéreo a diversas resoluciones y velocidades de muestreo.

¿Que es MP3?

Moving Picture Experts Group Audio Layer 3 es un formato de compresión de audio que usa un algoritmo con pérdida para conseguir un menor tamaño de archivo. Es utilizado para música tanto en Computadoras como en reproductores portátiles.

Video

Reproductores
(OGV - AVI - MP4)

Es el servicio que provee a los usuarios de Internet un sistema para poder almacenar información, imágenes, vídeo, o cualquier contenido accesible vía web. Es una analogía de "hospedaje o alojamiento en hoteles o habitaciones" donde uno ocupa un lugar específico, en este caso la analogía alojamiento web o alojamiento de páginas web, se refiere al lugar que ocupa una página web, sitio web, sistema, correo electrónico, archivos, en Internet o más específicamente en un servidor, que por lo general hospeda varias aplicaciones o páginas web.

¿Que es OGV?

Ogg Video es un archivo de video que utiliza el formato de código abierto Ogg de xiph.org. Este contiene datos de video que pueden usar diferentes codecs.

¿Que es WebM?

Formato multimedia desarrollado por Google orientado para uso con HTML5. Es un proyecto de software libre, bajo una licencia permisiva similar a la licencia BSD.

¿Que es MP4?

Moving Picture Expert Group 4 es un códec creado para web, codifica datos audio vídeo optimizando su calidad de almacenamiento, codificación y distribución en redes.

Lenguaje de Marcas de Hipertexto

Standard

Organizaciones involucradas
(SGML - W3C - WHATWG)

HTML Es el estándar que se ha impuesto en la visualización de páginas web, es el que todos los navegadores actuales han adoptado y basa su filosofía de desarrollo en la diferenciación. Para añadir un elemento externo a la página como imagen, vídeo, script, etc. , no se incrusta directamente en el código de la página, sino que se hace referencia a la ubicación del mismo mediante texto. De este modo, la página web contiene solamente texto mientras que recae en el navegador web, interprete del código, la tarea de unir todos los elementos y visualizar la página final.

¿Que es SGML?

Standard Generalized Markup Language es una norma ISO derivada de GML de IBM, corresponde a Lenguaje de Señalización General Normalizado. Este permite que la estructura del documento pueda ser definida en base a la relación lógica de sus partes.

¿Que es la W3C?

World Wide Web Consortium , es un consorcio internacional fundado en 1994 y dirigido por Tim Berners-Lee, el creador original del URL , genera recomendaciones y estándares que aseguran el crecimiento de la World Wide Web a largo plazo.

¿Que es la WHATWG?

La comunidad de Web Hypertext Application Technology Working Group compusesta de personas y empresas que se interesan en la evolución de HTML y las tecnologías conexas. Fue fundado por integrantes de Apple, la Fundación Mozilla y Opera Software.

Elementos

Tipos de Marcado HTML
(Estructural - Presentacional - Hipertextual)

HTML se escribe en forma de «etiquetas», rodeadas por corchetes angulares <>...</>, puede describir la apariencia de un documento e incluir o hacer referencia a un tipo de programa llamado script, que puede afectar el comportamiento de navegadores web y otros procesadores de HTML. Tambien sirve para referirse al contenido del tipo de MIME text/html o todavía más ampliamente como un término genérico, ya sea en forma descendida de XML como XHTML 1.0 o posterior, o en forma descendida directamente de SGML como HTML 4.0 o anterior.

¿Que es Estructural?

El marcado estructural describe el propósito del texto, no define cómo se verá el elemento, pero la mayoría de los navegadores web han estandarizado el formato de los elementos. La etiqueta <h1>...</h1> define al Texto como un encabezado de Primer Nivel.

¿Que es Presentacional?

El Marcado presentacional describe cual es la apariencia del texto, sin importar su funcion. La mayoria de este fue desechada desde la versión 4.01, en favor de la implementacion de CSS. <style>...<style> se utiliza para la implementacion de estilos en bloque.

¿Que es Hipertextual?

El marcado hipertextual se utiliza para enlazar partes del documento con otros documentos o con otras partes del mismo. Es necesario utilizar la etiqueta ancla <a>...</a> con el atributo href, que establecerá la dirección URL a la que apunta el enlace.

Atributos

Tipos de los Elementos
(block - inline - inline-block)

Cada elemento HTML debe cumplir con las reglas que definen qué tipo de contenido puede contener. Estas reglas se agrupan en modelos de contenido comunes a varios elementos. Cada elemento HTML pertenece a cero, uno o varios modelos de contenido. Un modelo establece las reglas que el contenido del elemento debe cumplir en un documento HTML-conformant.

¿Que es Block?

Son elementos que ocupan todo el espacio de su contenedor, cada vez que se presenta uno se hace un salto de linea de manera predeterminada, siendo estos representados en lineas independientes.

¿Que es inline?

Son elementos cuyo contenido se muestra en una misma linea, de modo que si colocamos dos o más de estos, sin implementar algun salto de linea, estos se veran presentados de manera consecutiva.

¿Que es Inline-Block?

Son aquellos elementos de comportamiento intermedio, fluyen con el texto y demás elementos como si fueran elementos inline y además respetan el ancho, el alto y los márgenes verticales.
