Contenido Multimedia

Este tipo de elementos es muy frecuente en la actualidad, ya que permiten transmitir de manera grafica diferentes mensajes en nuestro sitio. Se utilizan para fondos, presentacion de contenido y por asociacion para simplificar el uso del espacio.

Imagenes

Elementos graficos que se pueden utilizar dentro de una pagina web mediante la etiqueta <img>, la cual posee los atributos src="..." para indicar donde se encuentra el archivo y alt="..." el cual corresponde al texto alternativo que aparece cuando la imagen no se puede visualizar.

Reglas de Nomenclatura

  • Evitar el uso de caracteres especiales como el espacio, ya que se traduce como %20 y no es lo mismo "foto 01.jpg" que "foto%2001.jpg". Se aconseja utilizar guiones (_-) en su lugar.
  • Utilizar nombres descriptivos como "background.jpg" o "perrito_jugando.jpg" para facilitar la implementacion de los archivos. Se aconseja evitar nombres automatizados
  • Nombrar los archivos utilizando solamente minusculas ya que los servidores son caseSensitive y distinguen entre mayusculas y minusculas. De esta manera reducimos el margen de error.
