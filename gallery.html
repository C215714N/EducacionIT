Cuando desarrollamos un sitio web seguramente necesitemos elementos graficos, los mismos son compatibles con cualquier navegador y segun su finalidad, podemos incorporar imagenes de cualquiera de los siguientes formatos.

  • JPG: Joint Photographic Expert Group es un formato de imagen que admite compresion, se utiliza en las fotografias y para galerias de imagenes.
  • PNG: Portable Network Graphics es un formato que admite el uso de un canal alpha (transparencia) y se utiliza para algunos efectos visuales.
  • GIF: Graphics Interchange Format es un formato que admite interpolacion de imagenes y se utiliza comunmente para crear imagenes animadas.
  • WEBP: Web Picture es un formato de imagen desarrollado por Google, cuya finalidad es la comparticion de este tipo de elementos en la web.
  • SVG: Scalable Vector Graphics es un formato de imagen vectorial, que se calcula matematicamente mediante el uso de lineas y curvas, es ideal para elementos iconograficos.
