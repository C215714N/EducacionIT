Contenido Multimedia

En este apartado, explicaremos cuales son los elementos que podemos incorporar dentro de la pagina, a nivel audiovisual. Sus ventajas y desventajas, ademas de los diferentes formatos soportados por los navegadores.

Imagenes

Son elementos que podemos utilizar dentro de nuestra pagina web para exponer, darle una vista mas agradable al usuario o para realizar una muestra visual. Al igual que el resto de los elementos de una pagina web, se representa con una etiqueta, siendo en este caso <img>

La etiqueta de Imágenes cuenta con 2 atributos necesarios, estos son src para indicar la ruta de origen de la imagen y alt para exponer un texto alternativo en caso que la imagen no este accesible.

Integración Imágenes y Texto

En ciertas ocasiones puede que necesitemos utilizar imagenes dentro de un parrafo, similar a como utilizamos las letras capitales. Para ello podemos incorporar estas dentro de los mismos, pero debemos tener en cuenta que hay que aplicar ciertos estilos para que funcione de manera apropiada.

Los estilos los aplicamos mediante el uso de CSS, utilizando un atributo llamado float que permite que el elemento afectado se salga del flujo del documento, para ubicarse en un extremo de la etiqueta y que el resto de los elementos que se encuentren despues se ubique alrededor del mismo.
