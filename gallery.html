Reglas de Nomenclatura

  • Evitar el uso de espacios debido a que este caracter suele ser reemplazado por el conjunto %20 y por lo tanto si nuestro arhivo se llama "foto 01.jpg" el navegador interpretara "foto%2001.jpg" no pudiendo encontrar la misma. Utiliza guiones (-_) como separadores para que el nombre quede como "foto_01.jpg"
  • Utilizar nombres descriptivos para facilitar el trabajo como desarrollador, como por ejemplo: "background.jpg" o "perritos_jugando.jpg". Evitar nombres automatizados o genericos como "2021-07-28-1503100.jpg" o "foto_01.jpg"
  • Implementar solamente minusculas para reducir el margen de error y debido a que los servidores son caseSensitive, ya que no son lo mismo los siguientes nombres "background.jpg", "Background.jpg", "BACKGROUND.jpg".
