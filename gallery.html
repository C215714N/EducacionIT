Contenido Multimedia

Se conoce con este nombre a todo elemento audiovisual que podemos incorporar dentro de una pagina web. Entre ellos destacan las imagenes, que podemos utilizarlas gracias a la etiqueta <img> y dentro de la misma debemos indicar los atributos src="..." y alt="..." que corresponde a la ruta de la imagen y un texto alternativo a modo de descripcion, respectivamente.

Formatos de Imagenes

Cuando utilizamos este de elementos en la pagina debemos tener en cuenta algunos aspectos como las dimensiones de las mismas, asi como tambien el formato a utilizar, ya que existen varios y segun la finalidad podemos elegir entre uno y otro. Entre los formatos soportados por los navegadores se encuentran los siguientes.

  • JPG (Joint Photographics Experts Group) Formato de imagen ligero, que admite compresion y es usualmente utilizado para fotografias.
  • PNG (Portable Network Graphics) Formato de imagen que soporta transparencia, a diferencia de JPG es mas pesado, ya que no admite compresion.
  • GIF (Graphics Interchange Format) Formato de imagen que soporta interpolacion, ideal para iconos animados o imagenes pequeñas.
  • WEBP (Web Picture) Formato similar a jpg, desarrollado por Google, especificamente para la comparticion de contenido a traves de la web.
  • SVG (Scalable Vector Graphics) Formato de imagen que se calcula matematicamente, soporta reescalado y se utiliza para iconos o logotipos.
