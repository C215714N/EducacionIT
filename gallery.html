Contenido Multimedia

Las paginas web soportan todo tipo de elementos, utilizando la etiqueta correspondiente para identificar el mismo. Entre ellos se encuentran las imagenes, audios y videos, etc. Para incorporarlos debemos colocar dentro de la etiqueta pertinente un atributo llamado src que indica donde se encuentra el recurso a utilizar.

Formatos de Imagen

Los archivos de imagen se clasifican en mapa de bits (raster) que consisten en una grilla de pixeles que puede representar campos de profundidad, pero se ve limitada por una cantidad finita de puntos; y vectoriales que corresponden a graficos generados por computadora, compuestos de lineas y curvas que se utilizan para la creacion de elemento iconograficos. Los Formatos mas comunes soportados por los navegadores web son:

  • png (portable network graphics): admite transparencia pero no comprime datos
  • jpg (joint photographics expert group) formato ligero comunmente utilizado en fotografias
  • gif (graphics interchange format) admite compresion e interpolacion de imagenes (animacion)
  • webp (web picture) formato exclusivamente diseñado para comparticion en la web
  • svg (scalable vector graphics) formato vectorial utilizado para iconos

Reglas de nomenclatura

  • evitar caracteres especiales ya que caracteres como el espacio se traducen al conjunto %20
  • utilizar nombres descriptivos para facilitar la identificacion de los elementos
  • utilizar solamente minusculas para reducir el margen de error, ya que los servidores son caseSensitive

Caracteristicas

Las imagenes y demas etiquetas de contenido multimedia, son elementos de linea, o mejor dicho de linea-bloque, es decir que si bien comparten espacio con otros elementos de su tipo, se les puede dar un tamaño. Se diferencian del resto de los elementos de linea, debido a que definen altura, debido a la posibilidad de definir sus dimensiones. Basicamente son como letras gigantes que se ubican en funcion del texto, es decir que para acomodarlos debemos utilizar atributos de texto en el contenedor.

Lamentablemente, debido a lo mencionado anteriormente, con HTML no podemos disponer los elementos graficos a un costado y hacer que un texto que se encuentre por debajo de dicho elemento se acomode a sus dimensiones. Por este motivo podemos utilizar atributos de css como overflow y float que permiten a un elemento salirse del flujo natural del documento y posicionarse en uno de los laterales del contenedor. Debemos tener en cuenta que las dimensiones de altura de las etiquetas estan definidas por su contenido.
