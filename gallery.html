Imagenes en la web

Podemos incluir este tipo de elementos en nuestro proyecto utilizando la etiqueta <img>, en la cual deberemos indicar los atributos src="..." para encontrar el recurso y alt="..." como una breve descripcion del mismo.

Si bien podemos utilizar practicamente cualquier formato de imagen, segun el uso que vayamos a darle a las mismas podemos elegir entre los siguientes formatos:

  • JPG Joint Photographic Expert Group es un formato que admite compresion, por lo tanto es ligero, y se utiliza principalmente para fotografias.
  • WEBP Web Picture es formato de imagen desarrollado por Google con el objetivo de facilitar la comparticion de contenido.
  • PNG Portable Network graphics es un formato de imagen que no admite compresion, pero pueden utilizar un canal alfa para transparencia.
  • GIF Graphics Interchange Format es un tipo de imagen mapa de bits, muy ligero, que soporta hasta 256 colores y permite interpolacion, es decir, imagenes animadas.
  • SVG Scalable Vector Graphics es un formato de imagen vectorial, es decir, que se calcula matematicamente en base a lineas y curvas. Es ideal para elementos inconograficos.
