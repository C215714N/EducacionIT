Contenido Multimedia

Nos referimos con este nombre a todo elemento grafico y audiovisual que podemos incorporar dentro de nuestras paginas web. Es importante tener en cuenta que cuando utilicemos este tipo de elementos, los mismos se encuentren dentro del proyecto o a lo sumo, poseamos la url completa del recurso.

Cuando utilizamos imagenes, podemos implementar cualquiera de los formatos existentes, pero se recomienda utilizar los siguientes:

  • JPG (Joint Photographic Expert Group) es un formato de imagen ligero, debido a que admite compresion y se utiliza principalmente para fotografias.
  • PNG (Portable Network Graphics) es un formato que soporta transparencia, no admite compresion y por lo tanto es mas pesado que jpg; se utiliza en imagenes sin fondo.
  • GIF (Graphics Interchange Format) es un formato de imagen que soporta hasta 256 canales de color y admite interpolacion, ya que se utiliza para imagenes animadas de baja resolucion.
  • WEBP (Web Picture) es un formato desarrollado por google y pensado exclusivamente para la comparticion de contenido en la web como alternativa a JPG, admite compresion y es ligero.
  • SVG (Scalable Vector Graphics) es un formato de imagen vectorial, cuyo contenido se calcula mediante el uso de lineas y curvas, es ideal para iconos o elementos que deban reescalarse.

Imágenes

Para incorporar este tipo de elementos debemos utilizar la etiqueta <img> dentro de la estructura de nuestra pagina web, y en la misma etiqueta debemos especificar los atributos src="..." y alt="..." que indican donde se encuentra la imagen y un texto alternativo que corresponde a una breve descripcion del elemento.
