Imágenes

Elementos graficos que se pueden utilizar dentro de una pagina web, para ello debemos colocar la etiqueta <img> y la misma utiliza los atributos src="..." para indicar donde se encuentra la imagen y alt="..." como texto alternativo en caso que no se pueda acceder al archivo.

Compatibilidad

Si bien existen muchos formatos, los navegadores soportan universalmente los siguientes:

  • JPG (Joint Photographics Experts Group)
  • PNG (Portable Network Graphics)
  • GIF (Graphic Interchange Format)
  • SVG (Scalable Vector Graphics)
