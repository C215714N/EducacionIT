Formularios HTML

Son herramientas que permiten el intercambio de datos en Cliente y Servidor, trabajan mediante peticiones y para incorporarlos dentro de la pagina web debemos utilizar la etiqueta <form>. Cuando realizamos el envio, solamente se procesara el contenido que se encuentre dentro de la etiqueta.

¿Como funcionan?

A la etiqueta mencionada anteriormente debemos colocarle al menos 2 atributos, estos son action="..." que indica lo que debe ocurrir cuando se realice el envio (normalmente una accion determinada por el Servidor); y method="..." que define como deben procesarse los datos para el envio, entre los valores mas comunes se encuentran los siguientes valores:

  • GET: Metodo que prioriza la respuesta del servidor, envia los datos en texto plano y los mismos pueden visualizarse desde la barra de direcciones. Se utiliza normalmente para Buscadores debido a que es bastante rapido.
  • POST: Metodo prioriza la seguridad de los datos del cliente, ya que cifra la informacion antes de ser enviada y los datos son permanecen visibles. Se utiliza en la mayoria de los casos, como formularios de Registro, Acceso, Contacto, etc...

¿Como se componen?

Los formularios se componen de dos elementos principales, Las Etiquetas(LABEL) que funcionan a modo de rotulo para que el usuario sepa que informacion debe completar o seleccionar; y los Campos (INPUT/TEXTAREA/SELECT) que correponden a los controles del mismo, que poseen los siguientes atributos:

  • name: atributo necesario que indica como debe identificarse el elemento cuando se realiza el envio, especialmenete para que su valor pueda ser procesado por el servidor.
  • type: atributo que indica el tipo de valor a ingresar o seleccionar en el campo de formulario, se utiliza en la etiqueta INPUT, ya que existen diferentes tipos.
  • value: corresponde al valor ingresado o seleccionado en el campo, es obligatorio en caso de utilizar un campo de seleccion, ya que de lo contrario este figuraria vacio.

Tablas HTML

Son elementos que permiten organizar la informacion de manera tabular, compuestos por filas y columnas, se utilizan para la representacion de datos. Para incorporarlas dentro de una pagina web debemos colocar la etiqueta <table> y dentro de la misma, respetando el modelo de cajas, las filas <tr> (Table Row) y dentro de estas, las celdas <td> (Table Division) en las que colocaremos los datos.

¿Poseen Carga Semantica?

La estructura expuesta anteriormente correponde a un ejemplo de como se realizaban antiguamente las tablas en html. En la actualidad, se incorporaron elementos que facilitan la identificacion de las partes de la misma, estas son las siguientes etiquetas semanticas.

  • THEAD: cabecera de tabla que corresponde a los titulos de campos de la tabla, dentro de la misma las celdas se crean utilizando la etiqueta <th> (table heading)
  • TBODY: cuerpo de tabla que contiene los datos de la misma, puede incluir celdas de encabezado si se trata de una tabla de referencias cruzadas.
  • TFOOT: pie de tabla que contiene informacion complementaria al cuerpo de la misma, es opcional y posee tanto celdas de encabezado, como celdas de contenido.
