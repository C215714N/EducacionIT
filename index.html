Caracteristicas HTML5

Incorporaciones al estandar

Con el surgimiento de la version 5 se agregaron nuevos elementos para facilitar la accesibilidad y hacer incapie en la semantica de nuestra pagina, entre estas nuevas etiquetas estan los elementos estadisticos, la API's de reproduccion de contenido multimedia y las herramientas de dibujo que podemos utilizar desde HTML.

Elementos Estadisticos

En caso que necesitemos compara valores o analizar el estado de una tarea, podemos utilizar etiquetas que se enfocan en el aspecto semantico de las estadisticas, para que los motores de busqueda puedan interpretar dicha informacion. Estos elementos son Progress para definir el avance en una tarea cuando esta aun no fue finalizada, e.g: la capacitacion; y Meter que se utiliza para establecer una comparacion entre valores cuando la tarea fue finalizada, e.g: un examen.

API's de Dibujo

Preprocesadores

Pseudolenguajes de CSS

Herramienta que facilita la creacion de hojas de estilos, gracias a que agrega caracteristicas que no vienen incluidas en CSS. Generalmente vamos a escribir codigo con una sintaxis bastante parecida al lenguajes original, pero con algunos elementos nuevos y una vez finalizada la creacion de dicho archivo, podemos crear nuestra hoja de estilos compatible con el navegador utilizando un compilador (programa que genera dicho archivo).

LESS (Leaner Style Sheets)

Preprocesador que utiliza JavaScript como lenguaje de compilacion y entre sus caracteristicas, aporta la asincronicidad del lenguaje para la implementacion de funciones o metodos. Para instalarlo debemos ingresar el comando de NodeJS npm i -g less y para generar las hojas de estilos, simplemente ingresamos el comando lessc "input.less" "output.css".

Uno de los inconvenientes de esta herramienta radica en que no posee un metodo de observacion de cambios y por ende, debemos compilar cada vez que modificamos el archivo en formato LESS. Podemos resolver este problema instalando el paquete less-watch-compiler y para compilar solamente ingresamos el comando less-watch-compiler "input/" "output/" "file.less"

SASS (Syntactically Awesome Styles Sheets)

Preprocesador que utiliza Ruby como lenguaje de compilacion y a diferencia de LESS, no posee la capacidad de llamar funciones de manera asincrona, es decir, que los metodos pueden ser utilizados solamente si fueron previamente declarados. Para instalarlo debemos utilizar el comando de NodeJS npm i -g sass.

Si bien carece de asincronicidad, posee algunas otras ventajas, como la implementacion de diferentes tipos de bucles, la facil identificacion de los metodos y por defecto permite observar los archivos a compilar, e incluso agrega los errores de compilacion al codigo para idenficarlos facilmente. Para compilar los archivos debemos ingresar el comando sass --watch "input.scss" "output.scss"
