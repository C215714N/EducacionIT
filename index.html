Elementos de HTML5

Con la aparicion del nuevo estandar se introdujeron nuevas etiquetas, que si bien, son soportadas por la mayoria de los navegadores, aun existen algunos que no las implementan en sus motores. Para este tipo de situaciones el mismo navegador solamente reemplaza dicho elemento por una etiqueta div.

Estadisticas

Cuando debemos exponer en la pagina valores que representas los estados de una tarea podemos utilizar las siguientes etiquetas: <progress> que se utiliza cuando queremos mostrar el avance en una tarea cuando esta aun no fue finalizada ej: el progreso durante la capacitacion; y <meter> que se utiliza cuando queremos comparar los resultados de la tarea cuando ya fue finalizada ej: Resultado de un Examen.

Multimedia

Actualmente la mayoria de los navegadores soporta este tipo de contenido, e incluyen una API de reproduccion para facilitar la implementaicion del mismo. Las etiquetas involucradas son <audio> y <video> que soportan los formatos .mp3, .wav, .ogg y .mp4, .webm, .ogv respectivamente. Un detalle a tener en cuenta es que los navegadores impiden la reproduccion automatica en caso de tener el sonido activado.

Preprocesadores CSS

Tambien conocidos como pseudolenguajes, son herramientas que permiten la creacion de hojas de estilos en un lenguaje similar a css, pero que admite metodos que no vienen incorporados en este ultimo de forma nativa. Se suele utilizar una herramienta de compilacion para generar el archivo de estilos resultante, la mayoria de los preprocesadores admite el uso de variables, anidacion de estilos, funciones, bucles, metodos entre sus caracteristicas.

LESS

Herramienta que utiliza JavaScript como lenguaje inteprete y de compilacion, podemos acceder a la documentacion de esta herramienta desde el sitio lesscss.org y tambien podemos instalarla desde el CLI utilizando el siguiente comando npm i -g less. Para generar nuestras hojas de estilos podemos utilizar el siguiente comando lessc "input.less" "output.css"

Debido a que este compilador debe ser ejecutado cada vez que realizamos un cambio en nuestro proyecto, podemos utilizar una alternativa que posee metodos de observacion, la misma se llama less watch compiler. Para instalarla solamente debemos escribir en el CLI el comando npm i less-watch-compiler y para ejecutar el compilador debemos ingresar el comando less-watch-compiler "input_folder/" "output_folder/" "file.less" pudiendo seleccionar una ubicacion diferente para la compilacion.
