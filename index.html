Etiquetas de Bloque: Elementos que ocupan todo el ancho de su contenedor, se les puede dar tamaño (ancho y alto) y cumplen la funcion de contenedores.
Etiquetas de Linea: Elementos que ocupan una porcion de la linea, no se les puede dar tamaño (automatico) y cumplen la funcion de contenido.

Las rosas son rojas y las violetas azules.

DIV (capa) => elemento de bloque que se utiliza para organizar y contener otras etiquetas, de manera tal que se facilite la aplicacion de estilos.
SPAN (Expandir) => elemento de linea que se utiliza para dar estilos a partes o secciones dentro de un elemento de contenido.
