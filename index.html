FrontEnd

Corresponde al desarrollo del lado del cliente, es decir, todo aquello que puede interpretar el navegador sin la intervencion del servidor. Dentro de este tipo de desarrollo de encuentran la estructura, diseño UI / UX y funcionalidades basicas como la interaccion entre elementos y peticiones al servidor. Entre los lenguajes que componen el frontend se encuentran los siguientes.

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

Es un lenguaje de estructura y sus siglas significan Lenguaje de Marcado de HiperTexto, el mismo utiliza etiquetas para idenficar cada elemento de la pagina y la funcion que este cumple dentro de la misma. Las etiquetas pueden clasificarse de diferentes maneras, por un lado segun su funcion, en semanticos y presentacionales; y por otro lado segun su disposicion, en elementos de bloque y de linea, los cuales poseen las siguientes caracteristicas.

  • De Bloque: Son aquellos que ocupan todo el ancho de su contenedor, generan salto de linea, se les puede dar tamaño y funcionan principalmente como contenedores.
  • De Linea: Son aquellos que ocupan el ancho de su contenido, comparten espacio con otros elementos de su tipo, su tamaño es automatico y funcionan principalmente como contenido.

CSS (Cascading Stylesheets)

Las hojas de estilo en cascada corresponden a un Lenguaje de Diseño Grafico que utiliza selectores y reglas para definir la apariencia de los elementos de la pagina. A diferencia de HTML, en css el orden de los factores no altera el producto, y ademas este lenguaje de puede utilizar de diferentes maneras, como se muestra a continuacion.

  • Estilo en Linea: se utiliza colocando el atributo style="..." dentro de la etiqueta de apertura y dentro de la misma el codigo css. Este estilo es el de mayor importancia, pero el menos practico a la hora de trabajar.
  • Estilo en Bloque: el codigo CSS permanece separado del codigo HTML, puede estar dentro del mismo archivo utilizando la etiqueta style y dentro de la misma el codigo css, o se puede colocar el codigo en un archivo externo.
