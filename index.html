Lenguaje de Programacion

Sistema compuesto de reglas que permiten el desarrollo de alguna aplicacion con una finalidad en particular. Existen diferentes tipos y se catalogan segun su implementacion o forma de generar dichos programas.

Implementacion

  • Escritorio Son aquellos que se instalan sobre algun sistema operativo, como por ejemplo Microsoft Windows y suelen estar compilados.
  • Web Requieren de alguna herramienta que interprete el lenguaje y lo ejecute, en este caso los Navegadores como Google Chrome.
  • Movil Es aquel que se instala en un dispositivo movil, similar a las aplicaciones de escritorio, que utiliza las funcionalidades del dispositivo.
  • Consola Son aquellas aplicaciones que se ejecutan directamente desde la linea de comandos, es decir que, no cuentan con interfaz grafica.

Generacion

  • Interpretado Lenguajes como javascript o python que pueden ser leidos y ejecutados directamente por una herramienta.
  • Compilado Lenguaje que como C++ o JAVA requiere de un programa que lo empaquete para que pueda ser ejecutado por el usuario, desde un archivo que contenga el codigo resultante.
  • Transpilado Son aquellos que se generan a partir de un Framework o Library y son traducidos y compilados a un lenguaje de salida.
