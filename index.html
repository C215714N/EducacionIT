Tipos de Desarrolladores

frontEnd:
Hace referencia al desarrollo del lado el Cliente (navegador) y corresponde a toda caracteristica que puede este reproducir sin necesidad de consultar al servidor (UI / UX / Funcionalidades)

backEnd:
Hace referencia al desarrollo del lado del servidor (hosting) y corresponde a toda consulta que debe ser resulta unica y exclusivamente por este (Administracion de datos / Scripts en lenguaje de servidor)

fullStack:
Hace referencia al desarrollo de aplicaciones web (cliente / servidor), pero principalmente al rol de organizador y administrador de equipos de trabajo, para la depuracion y optimizacion del proyecto.

Lenguajes

  • HTML (hipertext markup language) Lenguaje de marcado de hipertexto, utiliza etiquetas para definir los elementos que componen la pagina y su relacion con el contexto.
  • CSS (cascading styles sheets) Hojas de Estilo en Cascada, utiliza selectores y reglas para definir la apariencia de los elementos de la pagina. El orden de los factores NO altera el producto.
  • JS (javaScript) Lenguaje de Scripting que corre en el navegador, se utiliza para darle funcionalidad a las paginas de un sitio y maneja conceptos como BOM y DOM.

Tipos de elementos

En HTML podemos encontrar 2 grupos de elementos, por un lado se organizan segun su funcionalidad y por el otro se organizan por su tipo. Para el primer caso estan los presentacionales y semanticos, y para el segundo los de bloque y los de linea. Son presentacionales aquellos elementos que solo cumplen funciones esteticas, en la actualidad solamente se utilizan de este grupo las etiquetas DIV y SPAN. EN cuanto a elementos Semanticos, nos referimos a la mayoria de las etiquetas HTML que poseen un significado dentro de la pagina.

  • DE BLOQUE: Son aquellas que ocupan todo el espacio de su contenedor, generan salto de linea y no comparten espacio con otros elementos, tambien se les puede dar tamaño (ancho y alto) y generalmente funcionan como contenedores
  • DE LINEA: Son aquellas que ocupan una parte de su contenedor, comparten espacio con otros elementos de su mismo tipo, no se les puede dar tamaño (automatico) y generalmente funcionan como contenido

Caracteristicas de las Etiquetas

Todas las etiquetas poseen atributos, caracteristicas que podemos modificar y que poseen valores predeterminados, dentro de las mas importantes o principales encontramos las siguientes:

  • margin: distancia entre las etiquetas
  • border: linea que rodea la etiqueta
  • padding: relleno que existe entre el borde y el contenido
  • content: elementos (texto / etiquetas) que se encuentran dentro
