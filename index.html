¿Que es HTML?

Hyper Text Markup Language es el acronimo que corresponde a Lenguaje de Marcado de Hipertexto, es utilizado para el desarrollo de paginas web permitiendo definir la Estructura de nuestra pagina web.

¿Que es CSS?

Cascading Styles Sheets corresponde a las Hojas de Estilo en Cascada, utilizadas para definir el estilo visual de nuestra pagina web mediante el uso de selectores y reglas definidas por el desarrollador
