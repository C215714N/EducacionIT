Desarrollo FrontEnd

Desarrollo del lado del cliente (Browser) y corresponde a toda caracteristica que puede ser interpretada por el navegador sin intervencion de un servidor, como por ejemplo, la estructura, el diseño y algunas funcionalidades.
Dentro de los lenguajes que componen el FronEnd se encuentran:

  • HTML
  • CSS
  • JS

HTML

Es una lenguaje de marcado de Hipertexto que utiliza etiquetas para identificar cada uno de los elementos que forman de la pagina. Las mismas poseen atributos y valores, que pueden estar predefinidos o los podemos establecer nosotros.

Es fundamental entender que, al ser la parte que corresponde a la estructura de la pagina, el orden de los factores altera el producto. Tambien podemos clasificar a las etiquetas, segun su tipo, de la siguiente manera:

  • De Bloque: Son aquellas que ocupan todo en ancho de su contenedor, generan salto de linea y se les puede dar tamaño. Generalmente funcionan como contenedores
  • De Linea: Son aquellas que ocupan el ancho de su contenido, comparten espacio con otros elementos de su tipo, no generan salto de linea y no se les puede dar tamaño. Generalmete se utilizan como contenido

CSS

El lenguaje de Hojas de estilo en cascada, se utiliza para definir la apariencia de los elementos de una pagina; utiliza selectores y reglas para establecer la misma, y a diferencia del caso anterior, el orden de los factores NO altera el producto.

Entre las caracteristicas mas comunes que podemos modificar de las etiquetas, se encuentran las siguientes:

  • margin: distancia que separa a las etiquetas y que no suma tamaño
  • border: linea que rodea a las etiquetas y que por defecto es cero
  • padding: relleno que existen entre el borde y el conetnido de una etiqueta
  • content: elementos que se encuentran dentro una etiqueta, a los que se les puede aplicar atributos como width y height
