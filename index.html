Tipos de etiquetas

Etiquetas de Bloque: Ocupan todo el ancho de su contenedor, generan un salto de linea y no comparten espacio con otras etiquetas. Se les puede dar tamaño (ancho y alto) y funcionan como contenedores principalmente.
Etiquetas de Linea: Ocupan una porcion de su contenedor, no generan salto de linea y comparten espacio con otras etiquetas. No se les puede dar tamaño (automatico) y funcionan como contenido principalmente.

Las rosas son rojas y las violetas azules

Elementos presentacionales

Existen otro tipo de etiquetas, aparte de las semanticas y estas son la presentacionales, cuya funcion principal es facilitar la aplicacion del diseño, No tienen carga semantica y no representan otra cosa, mas alla de un contenedor.

  • div: etiqueta de bloque que se utiliza para agrupar elementos dentro de la pagina, y de esta manera poder aplicarles el mismo estilo.
  • span: etiqueta de linea que se utiliza para facilitar la aplicacion del diseño a una parte del contenido, siempre y cuando este no requiera del uso de la semantica.
