Elementos Estadisticos

Con el nuevo estandar se incorporaron 2 formas de trabajar con valores estadisticos, gracias a las etiquetas <progress> que define el valor de una tarea cuando esta aun no fue finalizada, ej: el estado de la cursada; y <meter> que se utiliza para la comparacion de valores cuando una tarea fue finalizada, ej: el resultado de un examen.

  • PROGRESS
    • sin value definido
    • value = 35; max = 100
    • value = javaScript
  • METER
    • value < low < high < optimum
    • low < value < high < optimum
    • low < high < value < optimum

Contenido Multimedia

Los navegadores actuales poseen api's reproduccion de este tipo de contenido y podemos acceder a estas gracias a las etiquetas <audio> y <video> que son compatibles con los formatos .mp3 - .wav - .ogg y .mp4 - .webm - .ogv respectivamente. Ademas podemos reproducir de manera automatica el contenido gracias a atributos como autoplay muted loop

Preprocesadores CSS

LESS

Preprocesador que utiliza javascript como herramienta de compilacion para crear hojas de estilo. Su principal ventaja es la asincronicidad del lenguaje que permite al usuario llamar a los estilos sin importar el orden de aparicion en las lineas de codigo, para compilarlo debemos utilizar el comando lessc input.less output.css

SASS

Preprocesador que utiliza ruby como herramienta de compilacion, y ademas brinda la opcion de observar los archivos que utilizamos mediante el comando sass --watch input.scss output.css. Esta biblioteca actualmente es utilizada por el equipo de desarrollo de Bootstrap y cuenta con metodos parecidos a los de cualquier lenguaje de programacion.
