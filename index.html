Tipos de Estructuras

Etiquetas de Bloque: Son aquellas que ocupan todo el espacio de su contenedor, generan un salto de linea, se les puede definir tamaño (ancho y alto) y funcionan como contenedores.
Etiquetas de Linea: Son aquellas que ocupan una porcion del contenedor no generan salto de linea y comparten espacio con otros elementos, no se les puede definir tamaño (automatico) y funcionan como contenido.

Elementos presentacionales

Son elementos que no poseen carga semantica (funcion dentro del contexto) y se representan con 2 etiquetas DIV (capa) que se utiliza para agrupar varios elementos y facilitar la aplicacion del estilo; y la otra es SPAN (expandir) que es de linea y se utiliza para dar estilo a un elemento dentro de una etiqueta.
