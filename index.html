Desarrollo FrontEnd

Corresponde a toda caracteristica que el navegador puede interpretar o llevar a cabo sin intervencion del servidor, como por ejemplo lo relacionado con el diseño UX / UI, la estructura de la pagina y la logica e interaccion entre elementos.
Entre los lenguajes que conforman el FrontEnd se encuentran los siguientes.

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

El lenguaje de marcado de hipertexto corresponde a un lenguaje de Estructura, que utiliza etiquetas para definir la funcion que cumple cada elemento dentro de la pagina. Las etiquetas tambien poseen atributos, es decir, caracteristicas que podemos modificar y debido a que el navegador interpreta el codigo en cascada, el orden de los factores altera el producto.

Las etiquetas HTML se pueden clasificar de diferentes maneras, entre ellas la mas comun es por sus dimensiones y los nombres que se les a las mismas son los siguientes

  • De Bloque: Son aquellas que ocupan todo el ancho de su contenedor, se les puede dar tamaño y generan salto de linea. Funcionan principalmente como contenedores.
  • De Linea: Son aquellas que comparten espacio con otras de su tipo, no se les puede dar tamaño, ya que es automatico, y funcionan principalmente como contenido.

CSS (Cascading StyleSheets)

Las hojas de estilo en cascada corresponden a un lenguaje de Diseño grafico, que utiliza selectores y reglas para definir la apariencia de los elemtos que componen la pagina. En este caso el orden de los factores no altera el producto, pero debemos tener en cuenta la forma en la que aplicamos los estilos, ya que este puede implementarse de diferentes maneras:

  • CSS Inline: Es la forma mas basica y consiste en utilizar el atributos style="..." para indicar la apariencia del elemento a modificar.
  • CSS InBlock: Que consiste en separar el codigo CSS del HTML, para una mejor lectura del mismo; y tambien se puede utilizar dentro del mismo archivo o por separado.
