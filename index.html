Caracteristicas de HTML5

Nuevas Incorporacion al estandar

Con el surgimiento de la version 5 se agregaron varias caracteristicas que facilitan el uso de contenido multimedia, herramientas de dibujo y uso de elementos estadisticos dentro de la pagina. La finalidad de esto es mejorar la semantica y reducir la dependencia de elementos externos como plugins o complementos por parte de nuestro sitio.

Elementos Estadisticos

Cuando necesitamos analizar una o varias situacion y requerimos que estan sean procesadas por las herramientas de accesibilidad y los motores de busqueda, podemos utilizar etiquetas como progress para indicar el avance mientras que se esta realizando una tarea, e.g.: una capacitacion; y meter para comparar los resultados de una tarea que ha sido finalizada, e.g: un examen.

API's de Dibujo

Los navegadores actuales incluyen herramientas que nos permiten realizar dibujos dentro de la pagina, estas son SVG para la realizacion de graficos vectoriales, es decir, imagenes creadas a partir de lineas y curvas matematicamente, utilizando el lenguaje XML;y Canvas para la creacion de imagenes rasters (mapa de bits), utilizando el lenguaje JS para la creacion de los mismos.

Preprocesadores

Pseudolenguajes CSS

Son herramientas que facilitan la creacion de hojas de estilos css mediante una herramienta de compilacion, las mismas estan desarrolladas con algun lenguaje de programacion y se caracterizan por agregar a la sintaxis de css funciones que no se encuentran disponibles en el lenguaje de hojas de estilo. Una vez realizada la compilacion, dicha herramienta creara la sintaxis css en un archivo que pueda ser leido por el navegador.

LESS (Leaner Styles Sheets)

Preprocesador que utiliza Javascript como lenguaje principal para la compilacion de la hoja de estilos, podemos instalarlo mediante NodeJS utilizando el comando npm i -g less. Cabe destacar que dicha herramienta posee algunas ventajas del lenguaje como la asincronicidad y la capacidad para llamar funciones o metodos; y para compilar nuestros archivos debemos utilizar el comando lessc "input.less" "output.css"

Uno de los inconvenientes que presenta el paquete original de LESS radica en la falta de un metodo de observacion, es decir, que debemos compilar cada vez que realizamos cambios. Una solucion a esto es el paquete less-watch-compiler que podemos instalar con NodeJS y para compilar debemos utilizar el comando less-watch-compiler "input/" "output/" "file.less"

SASS (Syntactically Awesome Styles Sheets)

Preprocesador que utiliza Ruby como lenguaje principal para realizar la compilacion, una de las principales diferencias con respecto a LESS es que no permite el llamado de metodos si estos no fueron declarados previamente, es decir, que el orden de los elementos es fundamental. Tambien se caracteriza por diferencia cada tipo de elemento mediante su sintaxis. SASS posee dos tipos de sintaxis, sass que se maneja mediante tabulaciones y scss que es mas similar a css.

Para instalarlo debemos ejecutar desde la consola el comando npm i -g sass y para realizar la compilacion de los archivos utilizamos el comando sass --watch "input.scss" "output.css" permitiendonos realizar un seguimiento al mismo. En caso de presentarse algun problema en la compilacion, este permanecera visible dentro del sitio que utilice la hoja de estilos compilada con errores.
