Tipos de Desarrolladores

Front-End:

Desarrollo del lado del cliente
(HTML - CSS - Javascript)

Es la parte de una web que conecta e interactúa con los usuarios que la visitan. Es la parte visible, la que muestra el diseño, los contenidos y la que permite a los visitantes navegar por las diferentes páginas mientras lo deseen. Es una de las dos mitades en las que se divide la estructura de cualquier página web. Es lo que debe mostrar la identidad corporativa de una compañía dentro de Internet, la parte que puede atraer o alejar al consumidor cuando este accede a la página. De ahí que sea tan importante y requiera de tanto esfuerzo.

¿Que es HTML?

Hyper Text Markup Language corresponde a Lenguaje de Marcado de Hipertexto y se utiliza para el desarrollo de la Estructura de las paginas web mediante el uso de Etiquetas.

¿Que es CSS?

Cascading Styles Sheets significa Hojas de Estilo en Cascada, son las que permiten definir el Estilo visual de un sitio web mediante el uso selectores y reglas previamente definidas.

¿Que es JavaScript?

Js como se suele abreviar, es un lenguaje de Programación Orientada a Objetos y se utiliza para darle funcionalidad a los elementos dentro de nuestra pagina web.

Back-End:

Desarrollo del lado del servidor
(SQL - PHP - JAVA)

Es la parte trasera de cualquier página web. Se trata de todo el conjunto del desarrollo que se encarga de que una página funcione y de que lo haga como lo hace, pero que al mismo tiempo es totalmente invisible para el usuario, que solo ve lo visual y gráfico. Consiste en un servidor, una aplicación y una base de datos. Se toman los datos, se procesa la información y se envía al usuario. Los desarrolladores suelen trabajar juntos para que todo funcione correctamente. Un desarrollador Back end debe tener amplios conocimientos de lenguajes: frameworks y los tipos de base de datos. No siendo necesario conocer todos los lenguajes pero sí entender y saber trabajar con algunos de ellos.

¿Que es SQL?

Structured Query Language es utilizado para obtener información de una Base de Datos con comandos que permiten seleccionar, insertar, actualizar los datos, etc.

¿Que es PHP?

Hypertext PreProcessor es un lenguaje utilizado para el desarrollo de paginas web dinámicas, el código se ejecuta en el servidor y luego se incrusta dentro del código HTML.

¿Que es JAVA?

Lenguaje de programación independiente de las plataformas y orientado a objetos,que muchas aplicaciones y sitios web utilizan y no funcionarán a menos que lo tengan instalado.

Full-Stack:

Proceso de Desarrollo
(VCS - Librerias - Frameworks)

Un desarrollador Full Stack es el encargado de manejar los aspectos relacionados con la creación y el mantenimiento de una aplicación web. Para ello es fundamental que tenga conocimientos en desarrollo Front-End y Back-End además de manejar diferentes sistemas operativos y lenguajes de programación. Es una pieza fundamental en los departamentos de desarrollo de cualquier empresa. Conoce cómo se diseña la aplicación web basándose en principios del diseño UX/UI y además, sabe programarla.

¿Que es un VCS?

Un Sistema de Control de Versiones consiste en la gestión de cambios en los elementos de un producto o configuración del mismo, de forma manual o mediante una herramienta. Una versión, revisión o edición, es el estado en el que se encuentra en un momento dado de su desarrollo o modificación.

¿Que es una Libreria?

Conjunto de implementaciones codificadas, que ofrecen una interfaz definida para una funcion determinada. Una biblioteca no suele ser utilizada de forma autónoma; su finalidad es la de ser utilizada por otros programas, independientes, de forma simultánea; aunque pueden requerir de otras para funcionar

¿Que es un Framework?

Es una estructura conceptual y tecnológica de asistencia definida, generalmente, con artefactos o módulos concretos de software. Puede incluir soporte para ciertos programas, bibliotecas, y un lenguaje interpretado, para ayudar a desarrollar y, de esta manera, unir los diferentes componentes de un proyecto.

Requerimientos de un Sitio Web

Dominio

Sistema de Nombres
(TLD - ccTLD - gTLD)

Es un identificador único de Internet, cada equipo informatico conectado a Internet se identifica con una dirección IP y para no tener que memorizar las direcciones de cada sitio existente se establece el protocolo DNS. Los servidores DNS son los encargados resolver estas direcciones y asociarlas con los nombres de Dominio registrados, de esta manera podemos acceder a nic.ar ingresando el nombre en el navegador sin necesidad de ingresar la direccion IP: 200.108.145.10 en la barra de direcciones.

¿Que es TLD?

Dominio de Nivel Superior es la categoría más alta resuelta por los DNS oficiales de Internet, administrados por la Internet Corporation for Assigned Names and Numbers (ICANN)

¿Que es gTLD?

Un Dominio de Nivel Superior Generico posee tres letras o mas, como .com. Pueden usarse sin restricción, aunque se mantiene una serie de ellos para usarse de manera restringida.

¿Que es ccTLD?

El Dominio de Nivel Superior Geografico es usado por un país o un territorio dependiente. Implementa los códigos de país de 2 letras asignados por la ISO 3166-1, como .ar.

Hosting

Almacenamiento Web
(SSL - SMTP - FTP)

Es el servicio que provee a los usuarios de Internet un sistema para poder almacenar información, imágenes, vídeo, o cualquier contenido accesible vía web. Es una analogía de "hospedaje o alojamiento en hoteles o habitaciones" donde uno ocupa un lugar específico, en este caso la analogía alojamiento web o alojamiento de páginas web, se refiere al lugar que ocupa una página web, sitio web, sistema, correo electrónico, archivos, en Internet o más específicamente en un servidor, que por lo general hospeda varias aplicaciones o páginas web.

¿Que es SSL?

La tecnología Capa de Socket Segura utiliza algoritmos de cifrado, codifica los datos que se transmiten e impide que agentes externos los lean al enviarlos a través de la conexión.

¿Que es SMTP?

Un servidor SMTP permite el envio de correo corporativo con el Protocolo de Transferencia Simple de Correo. Este suele poseer el nombre de la marca o empresa en la direccion.

¿Que es FTP?

Un cliente FTP es un programa que emplea el Protocolo de Transferencia de Archivos para conectarse a un servidor y transferir archivos, ya sea para descargarlos o para subirlos.
