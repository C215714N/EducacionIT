Elementos HTML5

Etiquetas para Estadisticas

Existen dos elementos que permiten llevar a cabo el control de tareas segun sus estados, estos son <progress> que se utiliza para definir el estado de una tarea cuando esta aun no fue finalizada, e.g: el avance durante la capacitacion; y <meter> que se implementa cuando necesitamos comparar los valores de una tarea cuando ya fue finaliza, e.g el resultado de un examen.

  • PROGRESS
    • sin value definido
    • value:20; max:100
    • value: JavaScript
  • METER
    • value < low < high < optimum
    • low < value < high < optimum
    • low < high< value < optimum

Preprocesadores

LESS

Preprocesador que utiliza javascript como herramienta de compilacion para crear hojas de estilo. Su principal ventaja es la asincronicidad del lenguaje que permite al usuario llamar a los estilos sin importar el orden de aparicion en las lineas de codigo, para compilarlo debemos utilizar el comando lessc input.less output.css

Caracteristicas CSS3

Transformaciones

Al igual que los programa de diseño, css permite modificar los puntos de ajuste de una etiqueta, es decir, nodos que permiten la modificacion de las etiquetas. Entre los cambios que podemos realizar se encuentran el escalado, desplazamiento, rotacion y sesgado y a diferencia de otros atributos de css, estos cambios modifican el contenido de dicha etiqueta.

scale(x,y)
translate(x,y)
rotate(degrees)
skew(x,y)
perpective

Animaciones

Al igual que programas como Adobe Animate (antiguamente flash), css nos permite crear nuestras propias peliculas utilizando las etiquetas. Esto es posible gracias a los @keyframes, que consisten en estilos que definen el aspecto (inicial o final) de nuestra animacion, aunque tambien podemos llegar a definir estados intermedios.
