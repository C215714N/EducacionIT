Desarrollo FrontEnd

Se conoce con este nombre al desarrollo del lado del cliente (navegador) y se corresponde con aquellas caracteristicas que este puede reproducir, sin necesidad de la intervencion del servidor, como por ejemplo la estructura, el diseño de ui / ux y algunas funcionalidades relacionadas a esto.
Dentro de los lenguaje que el navegador puede interpretar de manera independiente se encuentran los siguientes:

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

Lenguaje de marcado de Hipertexto (Estructura) que utiliza etiquetas para definir la jerarquia de la pagina y la funcion que cumple cada uno de los elementos de la misma y debemos tener en cuenta que el orden de los factores altera el producto. Las etiquetas se pueden clasificar segun sus caracteristicas, que pueden ser:

  • De Bloque: Son aquellas que ocupan todo el ancho de su contenedor, generan salto de linea y se les puede definir tamaño. Funcionan principalmente como contenedores
  • De Linea: Son aquellas que ocupan lo que mida su contenido, comparten espacio con otros elementos de su tipo y su tamaño es automatico. Funcionan principalmente como contenido

CSS (Cascading StyleSheets)

Las hojas de estilo en cascada Corresponden a un lenguaje de diseño grafico, que utiliza selectores y reglas para definir el aspecto de los elementos de la pagina web. En este caso el orden de los factores no altera el producto y algunas de los atributos mas comunes que se pueden modificar con este lenguaje son:

  • margin: distancia que existe entre las etiquetas y siempre aplica el de mayor tamaño.
  • border: linea que rodea a las etiquetas y por defecto su valor es cero.
  • padding: relleno o espacio que existe entre el borde de una etiqueta y su contenido.
  • content: Elementos que se encuentran dentro de una etiqueta, que poseen ancho y alto.
