Introduccion a Bootstrap

Es un Framework de CSS desarrollado por el equipo de twitter, que facilita la implementacion de un diseño web responsive. Utiliza la Metodologia BEM (Block Element Modifier) para las clases definidas en su hoja de estilos y las mismas se organizan segun su funcion en el sitio web. Los tipos de clases que podemos encontran se dividen en 3 categorias:

  • Helpers: clases que modifican alguna caracteristica de otro elemento previamente definido. Su objetivo es ofrecer valores alternativos.
  • Utilities: clases que afectan algun atributo en particular y suelen tener la regla !important al final del valor.
  • Components: clases que definen y se aplican a una estructura de elementos, aqui podemos ver reflejada la metodologia BEM (Block Element Modifier)

Caracteristicas Basicas

Como se comenta anteriormente, este framework utiliza la metodologia BEM que consiste en identificar los contenedores principales (Bloques) que definen a los componentes y a partir de ahi crear clases para cada Contenido (Elemento) que este relacionado con este. A la vez, tambien se pueden alterar los diseños predeterminados, definiendo una clase que indique el elemento afectado y su valor alternativo (Modificador)

Paleta de Colores

Podemos implementar un total de 8 colores diferentes, aunque tambien podemos crear nuestras propias clases utilizando las variables definidas en el selector :root de la hoja de estilos de Bootstrap. Al ser helpers, se aplican al atributo correspondiente, por ejemplo, si quiero cambiar el color del texto a rojo, debo agregar la clase text-danger al elemento que debemos modificar.

  • bg-secondary
  • bg-light
  • bg-dark
  • bg-success
  • bg-warning
  • bg-danger
  • bg-info
