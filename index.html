Tipos de Desarrolladores

Front-End:

Desarrollo del lado del cliente
(HTML - CSS - Javascript)

Es la parte de una web que conecta e interactúa con los usuarios que la visitan. Es la parte visible, la que muestra el diseño, los contenidos y la que permite a los visitantes navegar por las diferentes páginas mientras lo deseen. Es una de las dos mitades en las que se divide la estructura de cualquier página web. Es lo que debe mostrar la identidad corporativa de una compañía dentro de Internet, la parte que puede atraer o alejar al consumidor cuando este accede a la página. De ahí que sea tan importante y requiera de tanto esfuerzo.

¿Que es HTML?

Hyper Text Markup Language corresponde a Lenguaje de Marcado de Hipertexto y se utiliza para el desarrollo de la Estructura de las paginas web mediante el uso de Etiquetas.

¿Que es CSS?

Cascading Styles Sheets significa Hojas de Estilo en Cascada, son las que permiten definir el Estilo visual de un sitio web mediante el uso selectores y reglas previamente definidas.

¿Que es JavaScript?

Js como se suele abreviar, es un lenguaje de Programación Orientada a Objetos y se utiliza para darle funcionalidad a los elementos dentro de nuestra pagina web.

Back-End:

Desarrollo del lado del servidor
(SQL - PHP - JAVA)

Es la parte trasera de cualquier página web. Se trata de todo el conjunto del desarrollo que se encarga de que una página funcione y de que lo haga como lo hace, pero que al mismo tiempo es totalmente invisible para el usuario, que solo ve lo visual y gráfico. Consiste en un servidor, una aplicación y una base de datos. Se toman los datos, se procesa la información y se envía al usuario. Los desarrolladores suelen trabajar juntos para que todo funcione correctamente. Un desarrollador Back end debe tener amplios conocimientos de lenguajes: frameworks y los tipos de base de datos. No siendo necesario conocer todos los lenguajes pero sí entender y saber trabajar con algunos de ellos.

¿Que es SQL?

Structured Query Language es utilizado para obtener información de una Base de Datos con comandos que permiten seleccionar, insertar, actualizar los datos, etc.

¿Que es PHP?

Hypertext PreProcessor es un lenguaje utilizado para el desarrollo de paginas web dinámicas, el código se ejecuta en el servidor y luego se incrusta dentro del código HTML.

¿Que es JAVA?

Lenguaje de programación independiente de las plataformas y orientado a objetos , que muchas aplicaciones y sitios web utilizan y no funcionarán a menos que lo tengan instalado.

Full-Stack:

Proceso de Desarrollo
(VCS - Librerias - Frameworks)

Un desarrollador Full Stack es el encargado de manejar los aspectos relacionados con la creación y el mantenimiento de una aplicación web. Para ello es fundamental que tenga conocimientos en desarrollo Front-End y Back-End además de manejar diferentes sistemas operativos y lenguajes de programación. Es una pieza fundamental en los departamentos de desarrollo de cualquier empresa. Conoce cómo se diseña la aplicación web basándose en principios del diseño UX/UI y además, sabe programarla.

¿Que es un VCS?

Un Sistema de Control de Versiones consiste en la gestión de cambios en los elementos de un producto o configuración del mismo, de forma manual o mediante una herramienta. Una versión, revisión o edición, es el estado en el que se encuentra en un momento dado de su desarrollo o modificación.

¿Que es una Biblioteca?

Conjunto de implementaciones codificadas, que ofrecen una interfaz definida para una funcion determinada. Una biblioteca no suele ser utilizada de forma autónoma; su finalidad es la de ser utilizada por otros programas, independientes, de forma simultánea; aunque pueden requerir de otras para funcionar

¿Que es un Framework?

Es una estructura conceptual y tecnológica de asistencia definida, generalmente, con artefactos o módulos concretos de software. Puede incluir soporte para ciertos programas, bibliotecas, y un lenguaje interpretado, para ayudar a desarrollar y, de esta manera, unir los diferentes componentes de un proyecto.

Requerimientos de un Sitio Web

Dominio

Sistema de Nombres
(TLD - ccTLD - gTLD)

Es un identificador único de Internet, cada equipo informatico conectado a Internet se identifica con una dirección IP y para no tener que memorizar las direcciones de cada sitio existente se establece el protocolo DNS. Los servidores DNS son los encargados resolver estas direcciones y asociarlas con los nombres de Dominio registrados, de esta manera podemos acceder a nic.ar ingresando el nombre en el navegador sin necesidad de ingresar la direccion IP: 200.108.145.10 en la barra de direcciones.

¿Que es TLD?

Dominio de Nivel Superior es la categoría más alta resuelta por los DNS oficiales de Internet, administrados por la Internet Corporation for Assigned Names and Numbers (ICANN)

¿Que es gTLD?

Un Dominio de Nivel Superior Generico posee tres letras o mas, como .com . Pueden usarse sin restricción, aunque se mantiene una serie de ellos para usarse de manera restringida.

¿Que es ccTLD?

El Dominio de Nivel Superior Geografico es usado por un país o un territorio dependiente. Implementa los códigos de país de 2 letras asignados por la ISO 3166-1 , como .ar.

Hosting

Almacenamiento Web
(SSL - SMTP - FTP)

Es el servicio que provee a los usuarios de Internet un sistema para poder almacenar información, imágenes, vídeo, o cualquier contenido accesible vía web. Es una analogía de "hospedaje o alojamiento en hoteles o habitaciones" donde uno ocupa un lugar específico, en este caso la analogía alojamiento web o alojamiento de páginas web, se refiere al lugar que ocupa una página web, sitio web, sistema, correo electrónico, archivos, en Internet o más específicamente en un servidor, que por lo general hospeda varias aplicaciones o páginas web.

¿Que es SSL?

La tecnología Capa de Socket Segura utiliza algoritmos de cifrado, codifica los datos que se transmiten e impide que agentes externos los lean al enviarlos a través de la conexión.

¿Que es SMTP?

Un servidor SMTP permite el envio de correo corporativo con el Protocolo de Transferencia Simple de Correo . Este suele poseer el nombre de la marca o empresa en la direccion.

¿Que es FTP?

Un cliente FTP es un programa que emplea el Protocolo de Transferencia de Archivos para conectarse a un servidor y transferir archivos, ya sea para descargarlos o para subirlos.

Lenguaje de Marcas de Hipertexto

Standard

Organizaciones involucradas
(SGML - W3C - WHATWG)

HTML Es el estándar que se ha impuesto en la visualización de páginas web, es el que todos los navegadores actuales han adoptado y basa su filosofía de desarrollo en la diferenciación. Para añadir un elemento externo a la página como imagen, vídeo, script, etc. , no se incrusta directamente en el código de la página, sino que se hace referencia a la ubicación del mismo mediante texto. De este modo, la página web contiene solamente texto mientras que recae en el navegador web, interprete del código, la tarea de unir todos los elementos y visualizar la página final.

¿Que es SGML?

Standard Generalized Markup Language es una norma ISO derivada de GML de IBM, corresponde a Lenguaje de Señalización General Normalizado. Este permite que la estructura del documento pueda ser definida en base a la relación lógica de sus partes.

¿Que es la W3C?

World Wide Web Consortium , es un consorcio internacional fundado en 1994 y dirigido por Tim Berners-Lee, el creador original del URL , genera recomendaciones y estándares que aseguran el crecimiento de la World Wide Web a largo plazo.

¿Que es la WHATWG?

La comunidad de Web Hypertext Application Technology Working Group compusesta de personas y empresas que se interesan en la evolución de HTML y las tecnologías conexas. Fue fundado por integrantes de Apple, la Fundación Mozilla y Opera Software.

Elementos

Tipos de Marcado HTML
(Estructural - Presentacional - Hipertextual)

HTML se escribe en forma de «etiquetas», rodeadas por corchetes angulares <>...</>, puede describir la apariencia de un documento e incluir o hacer referencia a un tipo de programa llamado script, que puede afectar el comportamiento de navegadores web y otros procesadores de HTML. Tambien sirve para referirse al contenido del tipo de MIME text/html o todavía más ampliamente como un término genérico, ya sea en forma descendida de XML como XHTML 1.0 o posterior, o en forma descendida directamente de SGML como HTML 4.0 o anterior.

¿Que es Estructural?

El marcado estructural describe el propósito del texto, no define cómo se verá el elemento, pero la mayoría de los navegadores web han estandarizado el formato de los elementos. La etiqueta <h1>...</h1> define al Texto como un encabezado de Primer Nivel.

¿Que es Presentacional?

El Marcado presentacional describe cual es la apariencia del texto, sin importar su funcion. La mayoria de este fue desechada desde la versión 4.01, en favor de la implementacion de CSS. <style>...<style> se utiliza para la implementacion de estilos en bloque.

¿Que es Hipertextual?

El marcado hipertextual se utiliza para enlazar partes del documento con otros documentos o con otras partes del mismo. Es necesario utilizar la etiqueta ancla <a>...</a> con el atributo href, que establecerá la dirección URL a la que apunta el enlace.

Atributos

Tipos de los Elementos
(block - inline - inline-block)

Cada elemento HTML debe cumplir con las reglas que definen qué tipo de contenido puede contener. Estas reglas se agrupan en modelos de contenido comunes a varios elementos. Cada elemento HTML pertenece a cero, uno o varios modelos de contenido. Un modelo establece las reglas que el contenido del elemento debe cumplir en un documento HTML-conformant.

¿Que es Block?

Son elementos que ocupan todo el espacio de su contenedor, cada vez que se presenta uno se hace un salto de linea de manera predeterminada, siendo estos representados en lineas independientes.

¿Que es inline?

Son elementos cuyo contenido se muestra en una misma linea, de modo que si colocamos dos o más de estos, sin implementar algun salto de linea, estos se veran presentados de manera consecutiva.

¿Que es Inline-Block?

Son aquellos elementos de comportamiento intermedio, fluyen con el texto y demás elementos como si fueran elementos inline y además respetan el ancho, el alto y los márgenes verticales.

Hojas de Estilo en Cascada

Estilos CSS

Tipos de Implementación
(Inline - Internal - External)

Las hojas de estilo en cascada (CSS) se utilizan para formatear el diseño de una página web. Con CSS, se puede controlar el color, la fuente, el tamaño del texto, el espacio entre los elementos, cómo se colocan y distribuyen los elementos, qué imágenes de fondo o colores de fondo se utilizarán, diferentes pantallas para diferentes dispositivos y tamaños de pantalla, y muchas otras cosas más.

¿Que es Inline CSS?

Se utiliza CSS en línea para aplicar estilo a un único elemento HTML mediante el atributo <body style="..."> de un elemento HTML. Este puede contener cualquier propiedad CSS

¿Que es Internal CSS?

Se usa un CSS interno para definir un estilo para una sola página HTML. Se define en la seccion <head> de una página HTML, dentro de un elemento <style>...</style> .

¿Que es External CSS?

Las Hojas de Estilo Externa se usan para definir el estilo de multiples páginas HTML. En la seccion <head> de cada página se agrega <link rel="stylesheet" type="text/css" href="...">

Selectores

Escala Jerarquica
(Tag - Class - Id)

Una hoja de estilos consiste en una serie de reglas. Cada regla, o conjunto de reglas consisten en uno o más selectores, y un bloque de declaración. CSS tiene una sintaxis simple y usa un conjunto de palabras clave en inglés para especificar los nombres de varias propiedades de estilo. Un selector simple es aquel que está formado solo por una única cadena textual, sin ningún combinador. Son selectores simples el selector universal [*], el selector por tipo de elemento del DOM, el selector de ID y de clase, el de atributo, el de pseudoclase y el de pseudoelemento.

¿Que es Tag selector?

Un selector de etiqueta permite indicar a cual de los elementos se les debe aplicar un estilo determinado mediante el nombre del mismo. p{...} selecciona a todos los parrafos.

¿Que es Class selector?

Para seleccionar elementos con una clase específica, debe escribirse un carácter de punto [.], Seguido del nombre de la clase. p.css{...} selecciona los parrafos con clase css.

¿Que es Id selector?

Para seleccionar un elemento con Id debe escribirse un carácter hash [#], seguido del valor utilizado como identificacion unica. p#html{...} selecciona al parrafo con Id html.
