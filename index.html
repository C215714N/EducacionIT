Elementos de una pagina Web

Estructura Semantica

Modelo implementado en HTML que permite definir la jerarquia y funcionalidad de cada parte de nuestra pagina, surge como una solucion al uso excesivo de DIVs, debido a que no existian elementos apropiados para el facilitar el posicionamiento.

  • HEADER cabecera de pagina, contiene los titulos principales y siempre debe estar al comienzo.
  • NAV navegacion de pagina, contiene los links mas importantes y puede estar dentro de la cabecera
  • MAIN contenido principal de toda la pagina, destacado para el SEO
  • SECTION seccion de la pagina o contenido contextualizado, no funciona fuera de la pagina
  • ARTICLE articulo de la pagina o contenido independiente, posee su propia estructura y funciona fuera del sitio.
  • ASIDE lateral o contenido tangencial que se relaciona indirectamente con sus hermanos, suele utilizarse para publicidad
  • FOOTER pie de pagina, debe ir al final y se utiliza para brindar informacion o un medio de contacto, copyright, un mapa del sitio y/o enlaces externos.

Atributos Principales

En una pagina web encontramos tres tipos de elementos, Etiquetas de Bloque, que se caracterizan por ocupar todo el ancho de su contenedor, generar salto de linea y cumplen la funcion de contenedores; Etiquetas de linea, son aquellas que comparten espacio con otros elementos, su tamaño es automatico y funcionan como contenido. Por ultimo, se encuentran las Etiquetas de linea-bloque que toman caracteristicas de las anteriores ya que se les puede dar tamaño, comparten espacio con otros elementos, pero se las trata como contenido(texto).

  • margin: distancia que existe entre etiquetas, no suma tamaño y siempre aplica el mayor.
  • border: linea que rodea la etiqueta, suma tamaño y por defecto su valor es 0.
  • padding: relleno de una etiqueta, podria considerarse sangria y al igual que el borde suma tamaño
  • content: elemento que se encuentran dentro de la etiqueta, se puede definir su tamaño.
					header{
						width: 350px;
						border: 5px solid #333;
						padding: 40px;
					}

Box-sizing

  • CONTENT-BOX: final-width = width + padding + border (440px)
    content-width = width (350px)
  • BORDER-BOX: final-width = width (350)
    content-width = width - padding - border (260px)
