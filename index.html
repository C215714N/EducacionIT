Nuevas Caracteristicas

Incorporaciones al estandar

Con la aparicion de html5 se agregaron nuevas etiquetas al lenguaje para facilitar a los motores de busqueda la identificacion de cierto tipo de contenido, que anteriormente se implementaban de forma visual. Entre estos se encuentran API's de reproduccion de contenido multimedia, API's que permiten dibujar sobre el navegador, e incluso elementos que facilitan el analisis estadistico.

Elementos estadisticos

Para la representacion de valores numericos en diferentes circunstancias existen las etiquetas <progress> que se utiliza para representar el avance en una tarea cuando esta aun no fue finalizada, por ejemplo: la realizacion de una capacitacion; y <meter> que se utiliza para la comparacion de valores cuando una actividad fue finalizada, por ejemplo: el resultado de un examen.

  • Progress
    • value = undefined
    • value = 12.5; max = 100
    • value = javascript
  • Meter
    • value < low < high < optimum
    • low < value < high < optimum
    • low < high < value < optimum
