Desarrollo FrontEnd

Toda caracteristica que puede reproducir el navegador (cliente) sin necesidad de solicitar la intervencion de un servidor, generalmente se relaciona con el Diseño UI / UX / Funcionalidades Basicas. Dentro de los lenguajes que interpreta el navegador, se encuentran:

  • html
  • css
  • js

HTML (HyperText Markup Language)

Es Lenguaje de Marcado de Hipertexto se utiliza para definir la estructura de las paginas web, implementa etiquetas que sirven para indicar la jerarquia de los elementos dentro de las mismas. A esto se conoce como carga semantica y sirve para que los motores de busqueda identifiquen la funcion que cumple cada elemento de la pagina.

Las etiquetas tambien poseen atributos y los mismos se colocan dentro de la apertura; si bien la mayoria de las caracteristicas las manipularemos con css, los elementos de la pagina se pueden clasificar de la siguiente manera:

  • De Bloque: Ocupan todo el ancho de su contenedor, generan salto de linea y se les puede dar tamaño, generalmente funcionan como contenedores.
  • De Linea: Ocupan el ancho de su contenido y es automatico, comparten espacio con otros elementos de su tipo, generalmente funcionan como contenido

CSS (Cascading StyleSheets)

Las hojas de estilo en cascada se utilizan para definir el aspecto de los elementos de la pagina web, utilizan selectores y reglas para indicar a cuales les corresponden dichas modificaciones.
