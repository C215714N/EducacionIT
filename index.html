Tipos de Diseño

Maquetacion de un sitio

Desde que existen las paginas web y con la aparicion de CSS en el año 1996 surgieron diferentes formas de estilizarlas, utilizando medidas absolutas, relativas e incluso flexibles y de todos los metodos implementados podemos destacar principalmente 3, que no necesariamente se aplican de manera individual:

  • Fluid Web Design Estilo que utiliza porcentajes para mantener las proporciones de los elementos de la pagina con respecto a su diseño. Debido a que solo utilizan porcentajes, estos diseños no suelen ser escalables en pantallas pequeñas y grandes. Una solucion a este problema surge con el Liquid Web Design que consiste en la implementacion de maximos y minimos, pero en pantallas grandes se generan espacios no contemplados
  • Adaptive Web Design Estilo que consiste en crear una pagina para cada dispositivo objetivo, generalmente smartphone, tablet, desktop; es bastante practico cuando contemplamos dispositivo de bajos recursos, como poca capacidad de procesamiento o ancho de banda limitado, pero implica tener contenido duplicado. Es relativamente facil de mantener ya que maneja archivos por separado y es una solucion practica para clientes con bajo prespuesto, ademas suelen implementarse diferentes estilos segun el tamaño de la pantalla, empezando generalmente desde la vista de escritorio y reduciendo los contenido para las otras vistas.
  • Responsive Web Design Estilo sugerido originalmente por Ethan Marcotte en una publicacion del sitio A list apart donde aplica un concepto conocido como One Web for All donde habla del uso de media-queries, sistemas de grillas con porcentajes y multimedia flexible entre otros conceptos. Se aconseja implementar una caracteristica conocida como Content-first, evolucion del Mobile-first, que consiste en planificar el diseño desde la vista movil utilizando los contenidos finales para tener una clara nocion de como se vera la pagina en el navegador, para luego ir cambiando el diseño segun las dimesiones de la pantalla donde se reproduzca.

Unidades de medida

usos y aplicaciones

  • Absolutas Son aquellas que representan siempre el mismo valor independientemente del tamaño de la pantalla del dispositivo donde se visualice la pagina web. Se utilizan principalmente para establecer los limites de tamaño de los elementos de la pagina web, tanto maximos como minimos respecto al ancho de los mismos.
    • distancia: mm, cm, in(2.54cm)
    • tipografia:pc(1/6in), pt(1/72in 0.35mm)
    • hibrida:px (1/96in) su representacion depende de la resolucion de la pantalla.
  • Relativas Son aquellas que dependen de otro elemento, generalmente el contenedor, para establecer su medida; por lo tanto un valor no siempre representa lo mismo. Se aplican principalmente a la tipografia, atributos relacionados con esta como margin, padding y border, e incluso para definir el tamaño de las cajas de contenido.
    • tamaño: %(tamaño), fr(fraccion)
    • tipografia: em(ancho "M"), ex(alto de "x"), ch(ancho de "0"), %(letra)
    • hibrida:rem (root em) su unica referencia es la etiqueta Raiz(html)
  • Flexibles Son aquellas que toman como referente las dimensiones del viewport (ventana del navegador) y a diferencia del resto, no dependen de ningun otro contenedor. Se implementan principalmente para la generacion de tipografia responsive, establecer tamaños en vertical debido a que por defecto ningun elemento posee una altura definida.
    • dimension: vw(% ancho viewport), vh(%alto viewport)
    • proporcion: vmin(% menor medida), vmax(% medida mayor)

Bootstrap

Framework CSS

Conjunto de herramientas desarrollada por el equipo de twitter implementando sass y es6 (en las ultimas versiones), cuya finalidad es facilitar la creacion de sitios web responsive, accediendo a los estilos y scripts provistos por los desarrolladores del mismo.

Podemos implementarlo de manera externoa en nuestro proyecto a traves de la vinculacion del CDN (content distribution network) o descargar los archivos fuente, ya sea la hoja de estilos compilada o los archivos de preprocesador.

Estilos de utilidad o ayuda

Una de las caracteristicas de Bootstrap radica en sus helpers o utilities que son basicamente clases CSS que se utilizan dentro de la estructura para definir la apariencia basica de los elementos. La estructura de dichas clases suele ser la siguiente atributo-valor(unidad) representando el atributo con una letra o la palabra completa dependiendo de que tan general sea este.

A partir de la version 4.5, como el enfoque de bootstrap cambio para compatibilizar los sitios con los diferentes tipos de escritura (orientacion de texto) las clases cuyo valor corresponde a una orientacion como izquierda y derecha ya no reciben esos nombres, sino que pasan a llamarse start y end

    • margin-right: 1.5rem; margin-left: 1.5rem; .mx-4
