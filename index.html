Tipos de etiquetas

  • Bloque: ocupan todo el espacio del contenedor, generan un salto de linea, se les puede dar tamaño (ancho | alto) y funcionan como contenedores
  • Linea: ocupan una porcion de la linea, comparten espacio con otras etiquetas, no se les puede dar tamaño (automatico) y funcionan como contenido
  • Linea-Bloque: ocupan una porcion del contenedor, comparten espacio con otros elementos, se les puede dar tamaño (ancho | alto) y funcionan como contenido (huerfanas)

  • Background-image: permite aplicar una imagen de fondo utilizando el protocolo URL para indicar la ruta de la misma. Los formatos soportados son .jpg - .png - .svg - .gif
  • Background-size: permite definir el tamaño del fondo con respecto a la etiqueta contenedora, se puede indicar mediante medidas: horizontal y vertical o utilizando palabras clave: cover para abarcar toda la etiqueta y contain para que entre dentro de la misma.

  • Background-repeat: define como se repetira la imagen de fondo, en caso que sea necesario, el valor predeterminado es repeat pero podemos seleccionar entre repeat-x, repeat-y o no-repeat.

Float es un atributo que permite posicionar a un elemento por fuera del flujo del documento, es decir que podemos orientarlo hacia la derecha o izquierda para que el resto del contenido que se encuentre despues de este, se posicione del lado contrario y fluya a su alrededor. Cuando utilizamos este atributo debemos tener en cuenta que las etiquetas contenedoras se pueden desbordar, y para resolver esto podemos utilizar dos atributos diferentes.
Clear: se utiliza para limpiar el espacio utilizado por float y recibe los valores left, right o both. Es necesario que el float a limpiar sea correcto, ya que de lo contrario no se aplica. La etiqueta donde se utiliza dicho atributo es la siguiente al float, donde querramos generar un salto de linea en limpio.
Overflow: se utiliza para modificar el comportamiento de una etiqueta contenedora con respecto al desbordamiento de su contenido. El valor por defecto es visible y para impedir esto debemos utilizar hidden que contendra obligatoriamente a la etiqueta flotante que se encuentre dentro del contenedor donde se aplique.
