Front-End

Hace referencia al desarrollo del lado del cliente (navegador), es decir, toda caracteristica que pueda ser reproducida por este, sin solicitar la intervencion del servidor. Entre dichas caracteristicas se encuentran el diseño UI / UX / funcionalidades, y los lenguajes que corresponden a este tipo de desarrollo son:

  • html
  • css
  • js

HTML (HyperText Markup Language)

El Lenguaje de Marcado de HiperTexto es un lenguaje de Estructura que utiliza etiquetas para definir la funcion que cumple cada Elemento dentro de la pagina, asi como tambien su jerarquia. Cabe destacar que el Orden de los factores altera el producto.

CSS (Cascading StyleSheets)

Las hojas de Estilo en cascada corresponden a un lenguaje de diseño grafico, que permite cambiar el aspecto de los elementos que se representan en la pagina web. Utiliza selectores y reglas para aplicar estilos y en este caso el Orden de los factores NO altera el producto.
