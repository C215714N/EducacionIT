FrontEnd

Desarrollo del lado del Cliente, es decir, toda caracteristica que puede reproducir el navegador sin intervencion del servidor. Entre dichas caracteristicas se encuentran la estructura, el aspecto visual y algunas funcionalidades y entre los lenguajes que podemos encontrar tenemos:

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

Fue desarrollado por Tim Berners-lee, fundador de la W3C.

Lenguaje de Marcado de Hipertexto que utiliza etiquetas para identificar a cada uno de los elementos de una pagina web, las mismas poseen atributos y valores que pueden estar o no definidos.
Se clasifican segun el tipo de elemento y el orden de los factores altera el producto.

  • De Bloque: Son aquellas que ocupan todo el ancho de su contenedor, Se le puede definir tamaño y generan salto de linea. Generalmente funcionan como contenedores.
  • De Linea: Son aquellas que miden el ancho de su contenido, su tamaño es automatico y comparten espacio con otros elementos de su tipo. Generalmente funcionan como contenido.
