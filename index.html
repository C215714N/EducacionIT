Tipos de etiquetas

  • Etiquetas de Bloque:
    Aquella que ocupa todo el espacio de su contenedor, genera un salto de linea y no comparte espacio con otros elementos, se les puede dar tamaño (ancho y alto), funcionan principalmente como contenedores.
  • Etiquetas de Linea:
    Aquella que ocupa una porcion de su contenedor, comparte espacio con otros elementos del mismo tipo, no se les puede dar tamaño (automatico), funcionan principalmente como elementos de contenido.

Atributos de las Etiquetas

Son caracteristicas de las etiquetas que podemos manipular asignandoles un valor, dentro de algunos ya establecidos dentro del lenguaje. Entre todos los atributos que podemos encontrar, podemos destarcar 4 principales.

  • margin: distancia o espacio que existe entre una etiqueta y otra, siempre aplica el mas grande y no suma tamaño.
  • border: linea que rodea a las etiquetas, por defecto es cero y suma tamaño.
  • padding: relleno o espacio que existe entre el borde y el contenido de una etiqueta, tambien suma tamaño.
  • content: elementos (texto o etiquetas) que se encuentran dentro de las etiquetas, se les puede asignar tamaño (ancho y alto).
