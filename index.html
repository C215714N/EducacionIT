Desarrollo FrontEnd

Toda caracteristica de la pagina que puede ser reproducida por el navegador, sin necesidad de la intervencion del servidor. Dentro de dichas caracteristicas se encuentran el Diseño UI/UX/Funcionalidades o logica de algunos elementos de la pagina. Dentro de los lenguajes que puede interpretar el navegador, se encuentran:

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

Es un lenguaje de estructura y sus siglas significan Lenguaje de marcado de hipertexto, que se utiliza para identificar la jerarquia y funcion que cumplen los diferentes elementos dentro de la pagina. Utiliza etiquetas, valores y atributos para que el navegador represetan dichos elementos dentro de la ventana, en este caso, el orden de los factores altera el producto.

Los atributos de las etiquetas son caracteristicas que pueden ser modificadas y permiten definir algun aspecto o configuracion de las mismas. Dentro de las clasificaciones de elementos de la pagina, podemos encontrar los siguientes valores:

  • De Bloque: Ocupan todo el ancho de su contenedor, se les puede dar tamaño y generan salto de linea, generalmente funcionan como contenedores.
  • De Linea: Ocupan el ancho de su contenido, su tamaño es automatico y comparten espacio con otros de su tipo, generalmente funcionan como contenido.

CSS (Cascading StyleSheets)

Lenguaje de diseño grafico, cuyas siglas significan Hojas de estilo en Cascada y se utiliza para definir el aspecto de los elementos dentro de la pagina. Utiliza selectores y reglas para identificar a que elementos debe afectar y cuales son las modificaciones correspondientes, en este caso el orden de los factores NO altera el producto.

Dentro de las caracteristicas que se pueden modificar con CSS se encuentran los siguientes atributos:

  • margin: distancia que existe entre las etiquetas
  • border: linea que rodea a las etiquetas, por defecto es cero
  • padding: relleno que existen entre el borde y el contenido
  • content: elementos que se encuentran dentro de la etiqueta que poseen ancho y alto
