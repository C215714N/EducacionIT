Tipos de etiquetas

  • Etiquetas de Bloque:
    Ocupan todo el espacio de su contenedor, generan salto de linea, Se les puede definir tamaño y funcionan como contenedores.
  • Etiquetas de Linea:
    Ocupan una porcion de su contenedor, comparten espacio con otros elementos de su tipo, no se les puede definir tamaño (auto) y funcionan como contenido.

Caracteristicas principales

Las etiquetas poseen diferentes atributos, estos representan ciertos aspectos de las mismas que podemos modificar. Para hacer cambios en los mismos debemos colocarlo dentro de la etiqueta de apertura seguido de un igual y su respectivo valor, todas las etiquetas poseen los siguientes atributos:

  • Margin: Es la distancia que separa una etiqueta de otra, no suma tamaño y siempre aplica el mas grande.
  • Border: Es la linea que rodea a cada etiqueta, por defecto es cero y suma tamaño para la etiqueta.
  • Padding: Relleno o espacio que existe entre el borde y el contenido de la etiqueta, al igual que el borde suma tamaño.
  • Content: Elementos que se encuentran dentro de las etiquetas, estos se suelen adaptar a su contenedor, pero les podemos definir un tamaño limite (width y height)
