Desarrollo FrontEnd

Desarrollo del lado del cliente y se refiere a toda caracteristica que pueda reproducir el navegador sin necesidad de enviar una peticion al servidor, entre ellas estan el UI/UX/Funcionalidades y los lenguajes involucrados son:

  • html
  • css
  • js

HTML (HyperText Markup Language)

Lenguaje de marcado de hipertexto es un lenguaje de estructura que utiliza etiquetas para definir la funcion de cada elementro dentro de la pagina. El orden de los factores altera el producto.

Clasificacion

  • Etiquetas de Bloque: Son aquellas que ocupan todo el ancho de su contenedor, se les puede definir tamaño y generan un salto de linea. Funcionan principalmente como contenedores.
  • Etiquetas de linea: Son aquellas que ocupan el ancho de su contenido, no se les puede definir tamaño y comparten linea con otros elementos de su tipo. Funcionan principalmente como contenido

CSS (Cascading StyleSheets)

Las hojas de estilo en cascada funcionan como un lenguaje de Diseño grafico, que se implementa para definir el aspecto de los elementos de la pagina utilizando reglas y selectores. El orden de los factores NO altera el producto.

  • margin distancia que existe entre las etiquetas, siempre aplica la de mayor tamaño.
  • border linea que rodea a las etiquetas, por defecto su valor es cero y suma tamaño.
  • padding relleno que existe entre el borde y el contenido de la etiqueta, tambien suma tamaño.
  • content son los elementos que se encuentran dentro de la etiqueta, a los que podemos darle tamaño.
