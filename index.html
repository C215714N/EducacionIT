Elementos HTML5

Con el nuevo estandar se incorporaron varias etiquetas que hacen incapie en la semantica y el analisis de contenido, asi como tambien, aquellas que facilitan la implementacion de contenido multimedia sin necesidad de instalar plugins o herramientas de terceros, debido a las modificaciones que presentan los navegadores actuales.

Estadisticas

Entre las nuevas caracteristicas de html5, se incorporan los elementos estadisticos gracias a las etiquetas <progress> que se debe utilizar cuando necesitamos indicar el valor de una tarea cuando esta aun no fue finalizada o se desconoce su resultado final (e.g: avance de la capacitacion); y <meter> que se debe utilizar para la comparacion de valores cuando una tarea fue finalizada y por lo tanto, estos son definitivos (e.g: resultado de un examen)

  • PROGRESS
    • sin value definido
    • value = 20; max = 100
    • value = JavaScript
  • METER
    • value < low < high < optimum
    • low < value < high < optimum
    • low < high < value < optimum

Multimedia

Actualmente se incorporaron elementos capaces de reproducir contenido multimedia gracias a las etiquetas <audio> y <video> que son compatibles con los formatos .mp3 - .wav - .ogg y .mp4 - .webm - .ogv respectivamente. Esto es posible debido a que los navegadores actuales cuentan con una API de reproduccion, e incluso podemos habilitarlos con los atributos controls o autoplay muted loop si queremos que los videos se reproduzcan al cargar la pagina.
