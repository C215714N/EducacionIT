Preprocesadores

Es un programa que permite generar CSS a partir de una sintaxis única, estas herramientas disponen de utilidades que no están en el lenguaje CSS y, por tanto, no las interpretan los navegadores directamente. Utilizando un preprocesador conseguimos economizar tiempo, ya que escribimos menos código. Además, su mantenimiento es fácil y resulta más comprensible, debido básicamente a su optimizada organización. Por el contrario, utilizar un preprocesador implica aprender un nuevo lenguaje y escribir código poco común, que debe ser compilado en CSS y que afecta al flujo de desarrollo. Este código, que no entiende el navegador, deberemos traducirlo antes de llevar un sitio web a producción.

Transformaciones

Efectos que podemos aplicar de manera combinada sobre las etiquetas, funcionan de manera similar a los atributos de position, width, height, entre otros; pero a diferencia de estos los valores se colocan sin comas dentro del atributo transform

Transformaciones 2D

Son las mas simples de aplicar, solamente debemos colar los valores en el elemento a modificar y aplican los cambios tomando en cuenta los ejes X e Y y un atributo llamado transform-origin

translate(x,y)
scale(uX,uY)
rotate(deg)
skewX(deg)
skewY(deg)

Transformaciones 3D

A diferencia de los casos anteriores, en este tipo de transformaciones se utiliza de referencia al elemento padre, que debe poseer los atributos perspective y perspective-origin que indican la distancia y el angulo.

translateZ(d)
rotateX(deg)
rotateY(deg)
perspective

Animaciones

Consisten en la generacion de fotogramas clave a partir de ciertas reglas que indican como debe verse un elemento de principio a fin. A diferencia de las transiciones, no necesitan un gatillador, se pueden ejecutar indefinidamente y pueden tener multiples variantes. Cuando los implementamos debemos utilizar la regla @keyframes para definir el nombre de nuestra animacion y y luego sus dentro de la misma, sus fotogramas.

Animation-Timing-Function

Podemos utilizar este atributo para indicar como debe reproducirse la animacion, significando la palabra ease lento para la parte que corresponde, tambien podemos utilizar linear para indicar que es constante o cubic-bezier(x1,y1,x2,y2) donde x indica el tiempo e y indica la progresion.

Animation-Direction

Este atributo permite controlar la direccion de la animacion, pudiendo variar entre los valores de from{} y to{}, para utilizar valores como alternate y alternate-reverse, es necesario que la misma se ejecute por los menos 2 veces utilizando el atributo animation-iteration-count
