Es un lenguaje de programacion multiparadigma que es Interpretado por el navegador. Admite todas las caracteristicas de cualquier lenguaje de programacion, como el uso de variables, condicionales, bucles, funciones, etc.

Javascript utiliza 2 objetos principales, con los que vamos a estar trabajando, nos referimos al BOM (Browser Object Model) y al DOM (Document Object Model). Estas son dos estructuras jerarquicas en forma de arbol que identifican los elementos que componen a cada uno de esos Objetos. Una de las ventajas del DOM es poder capturar y manipular elementos que se encuentren dentro del documento HTML, asi como tambien, podemos crearlos directamente desde Javascript.
