HTML (HyperText Markup Language)

Lenguaje de estructura cuyas siglas se traducen como Lenguaje de marcado de Hipertexto, el mismo utiliza etiquetas y atributos para definir la funcion que cumple cada elemento dentro de la pagina (jerarquia). En este caso el orden de los factores altera el producto.

En HTML encontraremos lo que se conoce como elementos semanticos, es decir etiquetas que definen su contenido y su jerarquie dentro de la estructura de la pagina. Esto es importante para que los motores de busqueda puedan clasificar el contenido de las mismas.

CSS (Cascading StyleSheets)

Lenguaje de Diseño grafico cuyas siglas corresponden a Hojas de Estilo en Cascada, se utiliza para definir el aspecto de los elementos de la pagina mediante el uso de selectores y reglas. En este contexto el orden de los factores NO altera el producto.

Con CSS modificamos la mayoria de los atributos de los elementos de la pagina, se puede implementar de diferentes maneras como inline, inblock (interno o externo), pero en cualquiera de los casos algunos de las caracteristicas que encontramos en la mayoria de las etiquetas son:

  • Margin: Distancia que existe entre las etiquetas, no suma tamaño y solo aplica el del mayor tamaño.
  • Border: Linea que rodea a las etiquetas y cuyo valor por defecto es cero, este suma tamaño.
  • Padding: Relleno que existe entre el borde y el contenido, suele ser cero y tambien suma tamaño.
  • Content: Elementos que se encuentran dentro de las etiquetas, a los que se les puede dar tamaño como ancho y alto.
