Caracteristicas html5

Elementos Estadisticos

HTML5 incorpora elementos que permiten definir el valor de una tarea; ya sea durante su realizacion, por ejemplo: el tramo de un recorrido, como es el caso del elemento <progress> o si esta ya fue finalizada y debe compararse con otras, por ejemplo: una votacion, como es el caso del elementos <meter>.

  • PROGRESS
    • min="0" sin value max="1"
    • value = "30" max="100"
    • value="js"
  • METER
    • value * < low < high
    • low < value * < high
    • low < high < value *

Herramientas de Dibujo

Existen 2 tipos de graficos que se pueden representar dentro de nuestro HTML, por un lado estan las imagenes vectoriales representadas con la etiqueta SVG(scalable vector graphics), que funcionan generando lineas y curvas calculadas matematicamente utilizando el lenguaje XML. Por el otro estan las imagenes raster (mapa de bits) que se pueden generar con la etiqueta CANVAS que utiliza el lenguaje JS para representar los elementos dentro de la pagina.

Contenido multimedia

En la actualidad podemos utilizar este tipo de contenido sin necesidad de instalar algun plugin o complemento en el navegador como solia pasar en el caso de flashPlayer. Esto se puede realizar gracias a las etiquetas <audio> y <video> que son compatibles con los formatos de sonido .mp3 - .wav - .ogg y de pelicula .mp4 - .webm - ogv y gracia al atributo controls no es necesario utilizar otro lenguaje como JS para manipularlos.
