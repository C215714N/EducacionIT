Nuevos elementos HTML

Contenido Multimedia

En las ultimas versiones de HTML se incorporo la posibilidad de reproducir de manera nativa contenido multimedia, sin necesidad de plugins o componentes de terceros. Los formatos compatibles son, en el caso del elemento audio .mp3 - . wav - .ogg y en el caso del elemento video .mp4 - .webm - .ogv. Tambien podemos ofrecer varias alternativas en caso que uno de los formatos no sea soportado por el reproductor web.

Elementos Estadisticos

HTML incorpora nuevos elementos a la hora de implementar valores estadisticos, por un lado tenemos el elemento progress que se utiliza cuando necesitamos indicar el avance de una tarea cuando esta aun no fue finalizada; por otro lado tenemos el elemento meter que se utiliza para comparar valores una vez haya sido finalizada la tarea.

  • Elemento Progress
    • sin value
    • value = 20 ; max = 100
    • value = js
  • Elemento Meter
    • value = 30 < low < high < optimum
    • low < value = 60 < high < optimum
    • low < high < value = 90 < optimum

Herramientas de Dibujo

Actualmente se incorporaron a los navegadores 2 elementos que permiten la implementaciond de graficos dentro de nuestra pagina web. El elemento SVG (scalable vector graphics) permite la creacion de graficos vectoriales mediante el uso del lenguaje xml y por otro lado, el Elemento Canvas permite la creacion de graficos raster (Mapa de bits) utilizando JS para los mismos.

Nuevas caracteristicas CSS

Modelos de Color

CSS permite la representacion de diferentes modelos de color, contemplando diferentes alternativas. Por un lado encontramos los colores HTML que poseen nombres definidos y un total de 140 variantes; luego los colores que utilizan la sintesis adhitiva empleando los canales de color rgb, los mismos se pueden utilizar con dos variantes decimal y hexadecimal donde solamente cambia la forma de representacion. Por ultimo, se incorporo el modelo que utiliza el circulo cromatico alterando los valores de tono, saturacion y brillo.

html color: midnightblue;
rgb decimal: rgb(25,33,104);
rgb decimal alpha: rgb(25,33,104,.5);
rgb hexadecimal: #192168;
cromatico: hsl(240,60%,25%);
cromatico alpha: hsl(240,60%,25%,.5);

Gradientes de Fondo

Otra de las caracteristicas incorporadas es la implementacion de gradientes, los mismos se consideran como imagenes de fondo y no como colores. La forma de implementarlos es indicando tipo(direccion, color-a, color-b) pudiendo aplicar cualquier cantidad de valores de color e incluso las paradas de los mismos.

linear-gradient()
linear-gradient(to right)
linear-gradient(-45deg)
radial-gradient()
radial-gradient(circle)
radial-gradient(circle at top right)

Modos de Fusion

Al igual que otros programas de diseño como Adobe Photoshop, CSS permite la utilizacion de modos de fusion que consisten en la alteracion de los pixeles entre una capa inferior(base) y una capa superior(fusion), utilizando las imagenes de fondo de una etiqueta para aplicar efectos que en otras circunstancias requeririan de la edicion de imagenes.
