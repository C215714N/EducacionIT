FrontEnd

Corresponde al desarrollo del lado del cliente, es decir, todo aquello que puede interpretar el navegador sin la intervencion del servidor. Dentro de este tipo de desarrollo de encuentran la estructura, diseño UI / UX y funcionalidades basicas como la interaccion entre elementos y peticiones al servidor. Entre los lenguajes que componen el frontend se encuentran los siguientes.

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

Es un lenguaje de estructura y sus siglas significan Lenguaje de Marcado de HiperTexto, el mismo utiliza etiquetas para idenficar cada elemento de la pagina y la funcion que este cumple dentro de la misma.
