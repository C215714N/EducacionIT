Desarrollo FrontEnd

Desarrollo del lado del cliente (Browser), es decir, toda caracteristica o funcionalidad que puede interpretar el navegador sin intervencion del servidor.

Algunos de los lenguajes que son interpretados por el navegador y forman parte del frontend son los siguientes:

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Langugage)

El lenguaje de marcado de hipertexto es un lenguaje de estructura, es decir, que se utiliza para definir la funcion que cumple cada elemento dentro de la pagina.
Este lenguaje utiliza etiquetas para identificar cada elemento y dentro de las mismas podemos encontrar atributos, para especificar alguna caracteristica de las mismas.

CSS (Cascading StyleSheets)

Las hojas de estilo en cascada se utilizan como lenguaje de diseño grafico, es decir, indicando mediante reglas como debe lucir cada elemento o parte de nuestro pagina.

  • en linea: el codigo esta escrito dentro de las etiquetas
  • en bloque: el codigo esta escrito fuera del codigo de la pagina (puede ser interno o externo)
