Desarrollo FrontEnd

Corresponde a toda caracteristica que puede interpretar el navegador (cliente) sin necesidad de solicitar la intervencion del servidor. Entre ellas se encuentran el diseño UX/UI/Funcionalidades Basicas, cabe destacar que los lenguajes involucrados en este tipo de desarrollo, son los siguientes:

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

El lenguaje de marcado de hipertexto , permite definir la estructura de la pagina web. Utiliza etiquetas y atributos para establecer la jerarquia de los elementos, ademas de alguna que otra caracteristica y en este caso el orden de los factores altera el producto.

CSS (Cascading StyleSheets)

Las hojas de estilo en cascada facilita la aplicacion del diseño dentro de la pagina web. Utilizan selectores y reglas para establecer la apariencia de los elementos de la misma, en este caso el orden de los factores NO altera el producto.
