Introduccion a la maquetacion

Elementos semanticos

  • HEADER Es el contenido introductorio que contiene los titulos principales, a veces suele tener una galeria a modo de presentacion e incluso la navegacion
    • NAV Es la navegacion principal y donde se encuentran solamente los links mas importantes de nuestro sitio.
  • MAIN Es el contenido principal donde colocamos la informacion relevante para el usuario, la mayoria de los elementos se encontraran dentro.
    • SECTION Elemento que forma parte del contenido, depende del contexto y solamente funciona dentro de la pagina, ya que esta ligado al contenido de la mismo.
    • ARTICLE Elemento independiente al contexto, funciona tanto dentro como fuera de la pagina y debe ser autoconclusivo.
    • ASIDE Elemento lateral o contenido tangencial que guarda una relacion indirecta con sus elementos hermanos.
  • FOOTER Es el contenido de descenlace, podemos encontrar informacion del autor, datos y/o formularios de contacto, mapas del sitio e incluso links a las redes sociales.

Jerarquia de Selectores CSS

  • Tag Selector: es aquel que apunta a un tipo de etiqueta, si bien se pueden combinar es el menos especifico y por lo tanto al ser muy general con respecto a los demas, es el que posee menor prioridad ya que no permite distinguir entre las etiquetas que seleccionamos.
    • header{color:darkred}
    • header h1{color:tomato}
  • Class Selector: es aquel que apunta a las clases aplicadas sobre los elementos y se diferencia por la implementacion de un punto (.) antes del nombre de la clase. Especificamente permite seleccionar un grupo de etiquetas, incluso pudiento aclarar que tipo de etiqueta es a la que apuntamos.
    • .header{color:darkgreen}
    • .header h1{color:limegreen}
  • Id Selector: es aquel que apunta al Id de un elemento en particular, se diferencia por la implementacion de un numeral (#). Es el selector con mayor prioridad porque se considera muy especifico con respecto a los demas, la razon de esto es que no deberian existir mas de una etiqueta con el mismo Id.
    • #header{color:darkblue}
    • #header h1{color:skyblue}

Atributos Generales

  • Box-Sizing:
    • content-box: final-width = width + padding + border | content-width = width;
    • border-box: final-width = width | content-width = width - padding - border;
  • Display:
    • BLOCK: ocupa todo su contenedor, genera salto de linea (no comparte espacio), se les puede definir tamaño (contenedores);
    • INLINE: ocupa en ancho de su contenido, comparte espacio con otras etiquetas, su tamaño es automatico - ancho (contenido);
    • INLINE-BLOCK: ocupa el espacio de su contenido, comparte espacios, se les puede definir tamaño - ancho/alto (contenido);

Flexbox

Es una herrienta que permite indicar a un contenedor, la forma en la que se van a distribuir y organizar sus elementos. A diferencia de los valores vistos anteriormente, que corresponden al atributo display y modifican la forma en la que se representa el elemento seleccionado, FLEXBOX actua sobre la etiqueta padre y afecta a las etiquetas que sean hijos directos

  • Flex-Start: Es valor predeterminado y permite distribuir los elementos al principio del flex(izquierda)
  • Flex-End: Permite distribuir los elementos en orden hacia el final del flex(derecha)
  • Center: Alinea todos los elementos hacia el centro del flex(horizontal)
  • Space-Around: Calcula la distancia que debe tener cada elemento y la aplica al flex las distancias entre los elementos que se encuentran en medio es mayor debido a que estas se suman.
  • Space-Between: Separa de extremo a extremo los elementos dentro del flex y la distancia resultante es equitativa entre cada etiqueta, el primer y ultimo elemento estan en el limite de la caja contenido.
  • Space-Evenly: Calcula exactamente la misma distancia de separacion para cada etiqueta, quedante distribuido de manera precisa, pero no es compatible con todos los navegadores.

  • Flex-Start: A diferencia de justify-content, no hay valor predeterminado y este permite organizar los elementos al principio del flex(arriba)
  • Flex-End: Organiza los elementos dentro del flex hacia el final del mismo, es necesario que el contenedor posea un alto definido.
  • Center: Organiza los elementos hacia el centro de la fila(verticalmente)
  • Baseline: Acomoda las etiquetas independientemente de su tamaño, de manera tal que el texto quede alineado para mejor legibilidad
  • Align-Self(*): Atributo que permite a la etiqueta seleccionada dentro del flex cambiar su alineacion con respecto a los demas elementos.
