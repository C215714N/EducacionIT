Introduccion a la Maquetacion

Elementos Semanticos

HTML5 incorporo en el estandar nuevas etiquetas que facilitan a los motores de busqueda y navegadores la interpretacion del contenido de las paginas web. Las mismas forman parte de lo que se conoce como Estructura Semantica cuya finalidad el mejorar el indexado de los sitio para que los elementos, mencionados anteriormente, puedan realizar mejores sugerencias y la vez evaluar con diferentes criterios el posicionamiento a la hora de mostrar resultados de busqueda. Cada una de las etiquetas que se se mencionan a continuacion definen la finalidad que cumple su contenido con relacion a su contexto.

  • HEADER Corresponde a la cabecera o contenido introductorio, debe contener los encabezados, puede contener la navegacion principal e incluso una galeria a modo de presentacion.
  • NAV Corresponde a la navegacion principal, debe contener los links mas importantes del sitio, independientemente de la cantidad de paginas.
  • MAIN Corresponde al contenido principal o de mayor relevancia para el usuario; y segun el tipo de sitio, puede estar dividido en mas partes.
  • SECTION Corresponde al contenido contextualizado, es decir, que funciona solamente dentro de la pagina debido a que depende de otros elementos para su correcta comprension.
  • ARTICLE Corresponde al contenido independiente, ya que funciona tanto dentro como fuera del sitio debido a que posee su propia estructura y no depende de otro elemento dentro de la pagina.
  • ASIDE Corresponde al contenido tangencial, es decir, que se relaciona indirectamente con sus elementos hermanos y normalmente se utiliza para publicidad o sugerencias.
  • FOOTER Corresponde al pie de pagina o contenido complementario, se utiliza para mostrar la informacion de contacto, copyright, redes sociales e incluso un mapa del sitio.

Caracteristicas de las Etiquetas

Todas los elementos que implementamos en una pagina web con HTML poseen propiedas con valores predeterminados, pero que pueden ser modificados. Si bien, cada uno de los elementos pertenece a una categoria como de Bloque, de Linea, Presentacional o funcional, todas las etiquetas poseen al menos un atributo; y dentro de los principales que posee la mayoria se encuentran:

  • MARGIN Distancia que existe entre las etiquetas y siempre aplica la de mayor tamaño.
  • BORDER Linea que rodea a una etiqueta, su valor predeterminado es cero y suma tamaño.
  • PADDING Relleno o distancia entre el borde y el contenido, que tambien suma tamaño.
  • CONTENT Elementos que se encuentran dentro de una etiqueta a los que se les puede dar tamaño.

Dimensionamiento (Box-Sizing)

  • content-box
    • final-width = width + padding + border
    • content-width = width
  • border-box
    • final-width = width
    • content-width = width - padding - border

Jerarquia de Selectores

En CSS3, si bien, no es importante el orden de las reglas, debido a que estas no afectan el producto final, si es relevante el orden de aplicacion de las mismas para determinar los estilos que se van a priorizar. En igualdad de condiciones el navegador determinara que la regla que a aplicar siempre sera la ultima declarada dentro de las hojas de estilos, pero antes que eso priorizara el tipo de elemento a que estamos aplicando los estilos, segun los atributos class y Id correspondiendo a la siguiente lista de selectores.

  • Tag Selector
    • header{...}
    • header h1{...}
  • Class Selector
    • .header{...}
    • .header h1{...}
  • Id Selector
    • #header{...}
    • #header h1{...}

Planificacion del Diseño

Aspectos escenciales

Uno de los aspectos mas importantes a la hora de maquetar es el paso previo a la misma, la planificacion. Es fundamental que tengamos definida la estructura de nuestro sitio, la relacion entre las paginas y los esquemas de color y posicionamiento de los elementos dentro de la misma, ademas de las proporciones.

Unidades de medida

  • Absolutas son aquellas que siempre valen lo mismo independienmente del medio en el que se visualicen, ya que poseen un tamaño definido.
    • distancia: cm, mm, in, pc
    • tipograficas: pt
    • hibrida: px(depende de la resolucion)
  • Relativas son aquellas que dependen del elemento contenedor o referente para establecer su tamaño y por lo tanto dependen del contexto.
    • contenedor: %(porcentaje)
    • tipograficas: em, ex, ch, %
    • hibrida: rem (ancho de M en raiz HTML)

Modelos de Color

  • color HTML: darkslateblue
  • color DEC: rgba(68,68,136, .5)
  • color HEX: #44448888
  • color HSL: hsla(240, 40%, 40%, .5)

Flexbox

El modelo de cajas flexible es estandar que se incorporo en CSS3 y corresponde a un modelo unidireccional que permite la organizacion del contenido mediante la implementacion de un eje primario y un eje secundario en una etiqueta contenedora. Debe implementarse en diseños simples, donde el contenido deba disponerse en una fila o columna de manera organica.
