Tipos de elementos

Etiquetas de Bloque: Son aquellas que ocupan todo el ancho de su contenedor, se les puede dar tamaño (ancho y alto), generan un salto de linea y cumplen la funcion de contenedores.
Etiquetas de linea: Son aquellas que ocupan una porcion de su contenedor, no se les puede dar tamaño (automatico), comparten espacio con otros elementos y cumplen la funcion de contenido.

Elementos Presentacionales

Div (Capa): Es una etiqueta de bloque sin carga semantica que se utiliza para agrupar elementos y facilitar la aplicacion del diseño
Span (Expandir) Es una etiqueta de linea que no posee carga semantica y se utiliza para Aplicar un estilo particular a una parte del contenido de una etiqueta

Caracteristicas de las etiquetas

  • Margin
  • Border
  • Padding
