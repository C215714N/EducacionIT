Caracteristicas HTML5

Elementos estadisticos

El nuevo estandar introduce la posibilidad de utilizar elementos estadisticos, con las etiquetas <progress> para definir el estado de una tarea cuando esta aun no fue finalizada ej: el avance durante la capacitación y <meter> para indicar los valores finales de una tarea finalizada ej: el resultado de un examen.

  • PROGRESS
    • sin value
    • value = 20; max = 100
    • value = javascript
  • METER
    • value < low < high < optimum
    • low < value < high < optimum
    • low < high < value < optimum

Contenido Multimedia

En la actualidad podemos incorporar este tipo de contenido sin necesidad de instalar ningun plugin o herramienta de tercero, gracias a las etiquetas <audio> y <video> que soportan los formatos .mp3 - .wav - .ogg y .mp4 - .webm - .ogv respectivamente. Ademas podemos habilitar un reproductor incorporado al navegador utilizando el atributo controls e incluso utilizarlo indefinidamente con los atributos autoplay muted loop en caso de los videos.
