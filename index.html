Introduccion a Flexbox

Distrubucion de elementos (Justify-Content)

  • Flex-Start: Distribuye todos los elementos al princio del flex, por defecto el lado izquierdo.
  • Flex-End: Distribuye todos los elementos al final del flex, por defecto el lado derecho.
  • Center: Distribuye todos los elementos hacia el centro del flex, por defecto de manera horizontal.
  • Space-Between: Distribuye los elementos entre el principio y el final del flex, estando el primer y ultimo item pegados a los bordes y separados con la misma distancia.
  • Space-Around: Distribuye los elementos de manera equitativa entre el princio y el final del flex, sumando las distancia entre los items, es decir, el primer y el ultimo tendran margenes dispares.
  • Space-Evenly: Similar a space-around, distribuye los elementos entre el princio y el final del flex sin sumar las distancias entre etiquetas, dando margenes iguales entre los elementos (no compatible con algunos navegadores).

Organizacion de elementos (Align-Items)

  • Flex-Start: Distribuye todos los elementos al princio del flex, por defecto la parte superior.
  • Flex-End: Distribuye todos los elementos al final del flex, por defecto la parte inferior.
  • Center: Distribuye todos los elementos hacia el centro del flex, por defecto de manera vertical.
  • Baseline: Distribuye los elementos en base al tamaño de la tipografia al principio del flex indistintamente del tamaño de las etiquetas.
  • Align-Self: Distribuye la etiqueta especificada dentro del flex de manera independiente al resto de los elementos.
