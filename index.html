Caracteristicas HTML5

Incorporaciones al estandar

Con el surgimiento de la version 5 se agregaron nuevos elementos para facilitar la accesibilidad y hacer incapie en la semantica de nuestra pagina, entre estas nuevas etiquetas estan los elementos estadisticos, la API's de reproduccion de contenido multimedia y las herramientas de dibujo que podemos utilizar desde HTML.

Elementos Estadisticos

En caso que necesitemos compara valores o analizar el estado de una tarea, podemos utilizar etiquetas que se enfocan en el aspecto semantico de las estadisticas, para que los motores de busqueda puedan interpretar dicha informacion. Estos elementos son Progress para definir el avance en una tarea cuando esta aun no fue finalizada, e.g: la capacitacion; y Meter que se utiliza para establecer una comparacion entre valores cuando la tarea fue finalizada, e.g: un examen.

  • Progress
    • value = undefined
    • value = 20; max = 100
    • value = javaScript
  • Meter
    • value < low < high < optimum
    • low < value < high < optimum
    • low < high < value < optimum
