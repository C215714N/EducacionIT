Desarrollo FrontEnd

Corresponde a toda caracteristica que puede interpretar el navegador (cliente) sin necesidad de la intervencion del servidor. Entre dichas caracteristicas se encuentran el Diseño UX / UI / Logica de la pagina y los lenguajes involucrados en este tipo de desarrollo son:

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

Lenguaje de estructura cuyas siglas significan Lenguaje de marcado de Hipertexto, este utiliza etiquetas y atributos para definir la jerarquia de los elementos de la pagina. En este caso el orden de los factores altera el producto.

Existen distintos tipos de etiquetas segun su representacion dentro de la pagina:
las etiquetas de bloque que ocupan todo el ancho de sus contenedores, generan alto de linea y principalmente funcionan como contenedores.
las etiquetas de linea que ocupan el ancho de su contenido, comparten espacio con otras etiquetas de su tipo y principalmente funcionan como contenido.
