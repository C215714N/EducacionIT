Lenguajes FrontEnd

html

Lenguaje de marcado de hipertexto que se utiliza para definir la estructura de las paginas web, utiliza etiquetas para establecer la funcion y jerarquia de los elementos de la misma. En los ultimos se hace mucho enfasis en la semantica, que permite a los motores de busqueda indizar los contenidos del sitio con mayor facilidad.

css

Las Hojas de estilos en cascada se utilizan para definir la apariencia de los elementos de nuestra pagina web, utilizando selectores y reglas. Este lenguaje permite realizar modificaciones en la pagina a nivel visual y es ideal para proporcionar una grata experiencia de usuario, ademas de una interfaz que responda segun el tamaño de la pantalla donde se visualice.

Javascript

Lenguaje de programacion orientada a objetos que se ejecuta directamente en el navegador, ya que no necesita ser compilado. Posee caracteristicas que permiten la manipulacion de la pagina mediante eventos o acciones de usuario, gracias a los modelos DOM y BOM que corresponden a Document Object Model y Browser Object Model.
