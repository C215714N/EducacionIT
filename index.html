Lenguajes FrontEnd

html

Lenguaje de marcado de hipertexto que se utiliza para definir la estructura de las paginas web.

css

Las Hojas de estilos en cascada se utilizan para definir la apariencia de los elementos de nuestra pagina web.

Javascript

Lenguaje de programacion orientada a objetos que se ejecuta directamente en el navegador, ya que no necesita ser compilado. Posee caracteristicas que permiten la manipulacion de la pagina mediante eventos o acciones de usuario, gracias a los modelos DOM y BOM que corresponden a Document Object Model y Browser Object Model.
