FrontEnd

Se conoce con este nombre al desarrollo del lado del cliente (Navegador/Browser), que consiste en toda caracterista que este puede interpretar sin requerir la intervencion del servidor. Entre estas caracteristicas se encuentran la creacion de la estructura, diseño UI / UX, y algunas funcionalidades basicas(logica) que puede tener la pagina.
Entre los lenguajes que engloba el FrontEnd se encuentran los siguientes:

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

Lenguaje de marcado de hipertexto, es decir, un lenguaje de estructura, que utiliza etiquetas para definir la jerarquia o funcion que cumple cada elemento dentro de la pagina. Debemos tener en cuenta que el orden de los factores altera el producto, y que las etiquetas de una pagina se pueden clasificar de la siguiente manera:

  • DE BLOQUE: son aquellas que ocupan todo el ancho de su contenedor, se les puede definir tamaño y generan salto de linea, principalmente funcionan como contenedores
  • DE LINEA: son aquellas que ocupan el ancho de su contenido, su tamaño es automatico y comparten espacio con otros elementos de su tipo, principalmente funcionan como contenidos

CSS (Cascading StyleSheets)

Las hojas de estilo en cascada se utilizan para definir el aspecto de los elementos dentro de la pagina web. Existen diferentes formas de aplicar este lenguaje, mediante la implementacion de Selectores y reglas; y el orden de los factores, en este caso, no altera el producto.

Dentro de las caracteristicas o atributos mas comunes que podemos modificar, se encuentran los siguientes:

  • margin: distancia que separa a las etiquetas, siempre aplica la mayor.
  • border: linea que rodea a las etiquetas y su valor predeterminado es cero.
  • padding: relleno que existe entre el contenido de una etiqueta y el borde.
  • content: Elementos que se encuentran dentro de la etiqueta y a los que les podemos dar ancho y alto
