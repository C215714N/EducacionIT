Clase 1: Nuevos elementos HTML5

Con la incorporacion de la ultima version de HTML se modificaron e incluyeron diferentes elementos que antiguamente debian representarse de maneras alternativas, como por ejemplo div para generar barras, utilizar js para determinada funcionalidad, o mediante herramientas de terceros. Entre dichos elementos encuentramos contenido multimedia para la reproduccion de audio y video, etiquetas para reprentacion de elementos estadisticos y API´s de dibujo para facilitar el diseño con otros lenguajes.

Tema 1: Multimedia

Con el surgimiento del nuevo standard, se incorporaron nuevas formas para agregar contenido multimedia dentro de nuestras paginas web sin necesidad del uso de plugins o complementos de terceros. Estas etiquetas son AUDIO y VIDEO, que admiten los formatos .mp3/.wav/.ogg y .mp4/.webm/.ogv respecticamente.

Aprovechando las caracteristicas del navegador para poder reproducir el contenido multimedia de manera automatica al cargar la pagina mediante <video autoplay muted></video> podemos utilizar el mismo como un fondo animado, para ello es necesario que posicionemos los elementos mediante el atributo {position:absolute;} con un {width:100%;}

Tema 2: Elementos estadisticos

Otro de los elementos introducidos en HTML5 son Progress y Meter, y a diferencia de los elementos mencionados anteriormente debemos llamar a la herramienta ::-webkit-element para poder modificar su aspector visual. El primero se utiliza para definir el avance de una tarea, sin haber esta terminado y el ultimo se utiliza para comparar valores estadisticos una vez haya sido terminada su evaluacion.

  • PROGRESS

    • Sin valor definido
    • Con valor 20/100

  • METER

    • value < low < high < optimum
    • low < value < high < optimum
    • low < high < value < optimum

Es importante destacar que la funcionalidad principal de estas etiquetas es semantica y no decorativa, como los elementos de dibujo que se detallan mas adelante.

Tema 3: API´s de dibujo

Ademas de poder utilizar imagenes en nuestro proyecto, HTML5 nos permite utilizar elementos muy utiles para dibujar mediante codigo. Por un lado tenemos la etiqueta <SVG> que permite la creacion de Graficos vectoriales mediante XML y por el otro, la etiqueta <CANVAS> que permite la creacion de Imagenes Mapa de Bits mediante la JS

Una de las ventajas principales de los Graficos vectoriales es que, al ser calculados matematicamente, no se pixelan cuando se escala la pagina y por lo tanto se mantienen las proporciones. Pero a diferencia de los elementos multimedia y los elementos estadisticos no estan pensados para ser manipulados mediante scripting. Por el contrario, los lienzos manipulados con Canvas, si fueron pensado para el desarrollo y mediante javascript se puede brindar funcionalidad a los dibujos realizados con dicha herramienta; pero una de las desventajas principales de este tipo de graficos es que son finitos y por ende se distorcionan al aumentar el tamaño de los elementos.

Clase 2: Estilos visuales

Tema 1: Colores

La representacion de los colores son la caracteristica mas comun de HTML. Los colores HTML poseen 140 valores definidos y son respresentados por su nombre en ingles; luego estan los colores RGB que se escriben de manera decimal "rgb(18,36,136)" o hexadecimal "#224488", siendo esta ultima la mas convencional, con un total de 16.777.216 colores (256 por canal). Y por ultimo el modelo HSL que utiliza el circulo cromatico para su implementancion, ademas de porcentajes de luz y saturacion.

color: grey;
rgb(128,128,128)
hex:#808080;
hsl(0,0%,50%)

Tema 2: Degradados

Una de las caracteristicas incorporadas en las ultimas versiones de CSS son los degradados, los cuales son considerados como imagenes de fondo y a diferencia de cualquiera de los valores mencionados anteriormente no se pueden aplicar utilizando atributos como color o background-color. Dentro de los gradientes que se pueden aplicar estan el lineal que utiliza definiciones de orientacion, color inicial y color final y por otro estan los degradados radiales, donde se define el tipo (elipse o circulo), posicion, color inicial y color final.

linear-gradient( to right, slateblue, darkslateblue );
linear-gradient( 45deg, skyblue, darkblue );
radial-gradient( circle at 75% 75%, gold, red );
radial-gradient( ellipse, orange, tomato )
