Estadisticas

Podemos evaluar los resultados de una tarea segun su estado, para ello existen los elementos <progress> que se utiliza cuan la misma no fue finalizada ej: el avance durante la capacitacion y <meter> cuando esta ya fue concluida ej: el resultado de un meter

  • PROGRESS
    • sin value
    • value = "40"; max = "100"
    • value = javaScript
  • METER
    • value < low < high < optimum
    • low < value < high < optimum
    • low < high < value < optimum
