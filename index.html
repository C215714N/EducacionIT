Desarrollo Front-End

Corresponde al desarrollo del lado del Cliente (Navegador) y toda caracteristica que este puede ejecutar sin necesidad de solicitar datos o una resolucion al servidor web. Entre los lenguajes que componen este tipo de desarrollo se encuentran:

  • html
  • css
  • js

HTML (HyperText Markup Language)

Lenguaje de marcado de hipertexto que utiliza etiquetas para identificar cado uno de los elementos que componen la pagina web y de esta manera establecer la jerarquia dentro de la misma. Las etiquetas corresponden a diferentes tipos que analizaremos a continuacion y el orden de los factores altera el producto.

  • DE BLOQUE:
    Son aquellos que ocupan todo el ancho de su contenedor, se les puede dar tamaño y generan salto de linea, generalmente funcionan como contenedores.
  • DE LINEA:
    Son aquellos elementos que ocupan en ancho de su contenido, su tamaño es automatico y comparten linea con otros de su tipo, generalmente funcionan como contenido.

CSS (Cascadings StyleSheets)

Las hojas de estilo en cascada se utilizan para definir la apariencia de los elementos de la pagina web, utilizan selectores y reglas para definir el aspecto de los mismos. Debido a que afecta la forma en la que se ven los elementos, podemos decir que el orden de los factores NO altera el producto.

Las etiquetas poseen diferentes caracteristicas, entre ellas podemos encontrar 4 atributos en comun, que corresponden a los siguientes:

  • margin: distancia que separa a las etiquetas, no suma tamaño
  • border: linea que rodea a las etiquetas, por defecto es cero, suma tamaño
  • padding: relleno que existe entre el borde y el contenido, tambien suma tamaño
  • content: elementos que se encuentran dentro de las etiquetas, a los que podemos definirles ancho y alto.
