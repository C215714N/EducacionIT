ETIQUETAS DE BLOQUE: Elementos que funcionan como contenedores, ocupan todo el ancho de su contenedor, generan salto de linea y se les puede dar tamaño (ancho y alto).
ETIQUETAS DE LINEA: Elementos que funcionan como contenido, ocupan una porcion de su contenedor, comparten linea con otros elementos no se les puede dar tamaño (automatico).

Las rosas son rojas y las violetas azules.

DIV: etiqueta de bloque que se utiliza para agrupar elementos, no tiene jerarquia (peso semantico) y facilita la aplicacion del diseño.
SPAN: etiqueta de linea que funciona para aplicar estilos visuales a partes de un elemento de contenido
  • margin: distancia que separa a cada etiqueta (aplica el valor mas alto)
  • border: linea que rodea a las etiquetas (por defecto es 0 y suma tamaño)
  • padding: relleno de cada etiqueta (por defecto suele ser 0 y suma tamaño)
  • content: elementos que se encuentran dentro de la etiqueta (se les puede dar tamaño si es de BLOQUE )
