Estructura Semantica

En HTML se conoce con ese nombre al conjunto de elementos de bloque que definen la funcion que cumple cada parte dentro de la pagina, es decir, que los mismos poseen una jerarquia que facilita a los motores de busqueda su comprension. Las etiquetas que componen dicha estructura son las siguientes:

  • header contenido introductorio, contiene encabezados y alguna galeria.
  • nav navegacion principal, contiene los links mas importantes del sitio.
  • main contenido principal, la informacion mas relevante para el usuario.
  • section contenido contextualizado, solamente funciona dentro de la pagina.
  • article contenido independiente, funciona tanto dentro como fuera de la pagina.
  • aside contenido tangencial, se relacion indirectamente con sus hermanos y se utiliza normalmente para sugerencias o publicidad.
  • footer contenido complementario, contiene informacion como derechos de autor, datos de contacto, redes sociales e incluso un mapa del sitio.

Jerarquia de Selectores

En CSS cuando aplicamos estilos, debemos tener en cuenta el peso de cada selector que utilizamos, ya que en caso de solapamiento de reglas el navegador determinada cual debe aplicar por encima del resto en base al elemento afectado. Esto solamente aplica cuando las reglas corresponden a la misma etiqueta.

  • tag selector de menor jerarquia que se puede sobrescibir facilmente, afecta a todas las etiquetas con el nombre seleccionado
  • .class selector de jerarquia intermedia que afecta a todas las etiquetas cuyo atributo class sea el especificado, pesa por encima de tag
  • #id selector de jerarquia superior que afecta a la etiqueta cuyo id corresponde con el del selector, aplica por encima del resto
