Desarrollo FrontEnd

Se conoce con este nombre al desarrollo del lado del cliente, es decir, toda caracteristica que puede interpretar el navegador sin que se requiera la intervencion del servidor. Entre estas caracteristicas podemos encontrar el Diseño UX / UI, asi como tambien algunas funcionalidades basicas, que estan representadas por los siguientes lenguajes.

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

El Lenguaje de marcado de HiperTexto se utiliza para definir la Estructura de una pagina web, mediante el uso de etiquetas y atributos, donde el orden de los factores altera el producto.
Fue desarrollado por Tim Berners-Lee, fundador de la W3C (World Wide Web Consortium) que es la encargada de definir el estandar de este y varios lenguajes que se manejan en la web. Cuando utilizamos html encontramos diferentes tipos de elementos, entre ellos podemos apreciar los siguientes.

  • De Bloque: Son aquellas que ocupan todo el ancho de su contenedor, generan salto de linea y se les puede dar tamaño. Funcionan principalmente como Contenedores.
  • De Linea: Son aquellas que comparten espacio con otras etiquetas de su tipo, su tamaño es automatico y Funcionan principalmente como Contenido.

CSS (Cascading StyleSheets)

Las hojas de estilo en cascada corresponden a un lenguaje de Diseño Grafico en el que se utilizan selectores y reglas para definir la apariencia de los elementos que componen una pagina, en este caso el orden de los factores no altera el producto. Existen diferentes formas de implementar css en un sitio, entre las cuales se encuentran las siguientes.

  • En Linea: se utiliza el atributo style="..." y colocamos dentro del mismo nuestro codigo. Este tipo de implementacion es individual para cada etiqueta pero tiene mayor prioridad.
  • En Bloque: el codigo se separa de la estructura, el mismo puede estar dentro de la misma pagina o en un archivo aparte; y permite afectar a varios elementos mediante el uso de selectores.
