Desarrollo FrontEnd

Es un termino que hace referencia a la escritura de codigo del lado del cliente. Corresponde a toda caracteristica que puede llevar a cabo el navegador sin intervencion del servidor, como Estructura de la pagina, diseño de UI / UX y algunas funcionalidades basicas. Dentro de los lenguajes que componen el FrontEnd se encuentran los siguientes.

  • HTML
  • CSS
  • JS

HTML (HyperText Markup Language)

Es un Lenguaje de Marcado que utiliza etiquetas para indicar la funcion que cumple cada elemento, y estas tambien poseen atributos que definen las caracteristicas de las mismas.
Las Etiquetas HTML se pueden clasificas segun sus caracteristicas en las siguientes categorias

  • De Bloque: son aquellas que ocupan todo el ancho de su contenedor, generan salto de linea y se les puede dar tamaño. Funcionan Principalmente como contenedores
  • De Linea: son aquellas que ocupan el ancho de su contenido, comparten espacio con otros elementos de su tipo, y Funcionan principalmente como contenido
