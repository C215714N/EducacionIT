Front-End

Hace referencia al desarrollo del lado del cliente (navegador), es decir, toda caracteristica que pueda ser reproducida por este, sin solicitar la intervencion del servidor. Entre dichas caracteristicas se encuentran el diseño UI / UX / funcionalidades, y los lenguajes que corresponden a este tipo de desarrollo son:

  • html
  • css
  • js

HTML (HyperText Markup Language)

El Lenguaje de Marcado de HiperTexto es un lenguaje de Estructura que utiliza etiquetas para definir la funcion que cumple cada Elemento dentro de la pagina, asi como tambien su jerarquia. Cabe destacar que el Orden de los factores altera el producto.

Clasificación

Las etiquetas HTML poseen diferentes caracteristicas y en base a estas se clasifican de la siguiente manera:

  • De Bloque: Son aquellas que ocupan todo el ancho de su contenedor, generan salto de linea y se les puede definir tamaño. Funcionan principalmente como contenedores.
  • De Linea: Son aquellas que ocupan el ancho de su contenido, comparten espacio con otros elementos de su tipo y su tamaño es automatico. Funcionan principalmente como contenido.

CSS (Cascading StyleSheets)

Las hojas de Estilo en cascada corresponden a un lenguaje de diseño grafico, que permite cambiar el aspecto de los elementos que se representan en la pagina web. Utiliza selectores y reglas para aplicar estilos y en este caso el Orden de los factores NO altera el producto.

Atributos

Las etiquetas poseen caracteristicas propias y algunas de estas pueden heredarse, como por ejemplo el color. Dentro de los atributos mas comunes se encuentran:

  • margin: distancia que separa a las etiquetas, siempre aplica el de mayor tamaño.
  • border: linea que rodea a las etiquetas, por defecto su valor es cero.
  • padding: relleno que existe entre el borde y el contenido de las etiquetas.
  • content: elementos que se encuentran dentro de las etiquetas, a los que podemos darle tamaño (ancho y alto)
