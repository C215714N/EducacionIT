Comandos Cisco - VLANS
Modo Comando Descripcion
Sw(config)# Vlan (vlan-ID) Permite crear una VLAN (rango normal 1 - 1005 | rango extendido 1006 - 4094).
Sw(config-vlan)# Name (nombre) Asigna un nombre a una VLAN.
Sw(config-vlan)# State (Estado | Suspended) Suspende una VLAN, todo puerto en esta VLAN descartara trafico.
sw# Show vlan id (vlan-ID) Permite ver informacion de una VLAN en particular.
Sw# Show vlan | brief Permite ver la base de datos de VLANs y la asignacion de puertos.
