Nuevos elementos HTML

Elementos Estadisticos

Con la aparicion del nuevo estandar, se incorporaron herramientas para la evaluacion estadistica, entre los elementos encontramos las etiquetas <progress> que se utiliza para identificar el estado de una tarea cuando esta todavia no fue culminada y <meter> que se implementa para comparar valores estadisticos una vez finalizada determinada tarea. Para la manipulacion del aspecto visual de dichos elementos debemos recurrir a las herramientas de :-webkit debido a que cada navegador aplica su propio estilo sobre dichos elementos.

Tema 2: Contenido Multimedia

Ademas de los elementos mencionados anteriormente, HTML5 incorpora de forma nativa la reproduccion de contenido multimedia mediante las etiquetas <audio> & <video> que permite al navegador reproducir los formatos .mp3 - .wav - .ogg para los sonidos y .mp4 - .webm - .ogv para los videos. A diferencia del apartado anterior, si se quiere personalizar el estilo de los reproductores, se debe hacer con HTML y CSS ya que no existe herramientas para editar el reproductor integrado al navegador. En caso de que el navegador no sea compatible con dichos elementos, debemos colocar un mensaje a modo de alternativa a nuestro material.

Tema 3: API's de Dibujos

Antiguamente, si necesitabamos modificar un elemento a nivel visual, como un dibujo o realizar alguno, teniamos que utilizar herramientas como Gimp, Inkscape dependiendo del tipo de imagen. Hoy en dia se puede dibujar desde el navegador utilzando XML para la creacion de Graficos vectoriales tambien conocidos como SVG; asi como tambien podemos utilizar JavaScript para realizar dibujos Mapa de Bits mediante Canvas.
