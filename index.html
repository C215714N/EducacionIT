HTML (HyperText Markup Language)

El Lenguaje de marcado de Hipertexto se utiliza para definir la estructura de una pagina web mediante etiquetas de identifican a cada uno de los elementos elementos de la pagina, asi como tambien, la funcion que cumplen. En este caso el orden de los factores altera el producto.
Este y muchos otros lenguajes que se utilizan en la web son definidos por la W3C (World Wide Web Consortium), asi como tambien el WHATWG (Web Hypertext Application Technology Working Group) se encarga de definir las buenas practicas

  • Elementos de Bloque son aquellos que ocupan todo el ancho de su contenedor, se les puede dar tamaño y generan salto de linea, Funcionan principalmente como contenedores
  • Elementos de Linea son aquellos que ocupan el ancho de su contenido, su tamaño es automatico y comparten linea con otros elementos de su tipo, Funcionan principalmente como contenido

CSS (Cascading StyleSheets)

Las hojas de estilo en cascada corresponden a un lenguaje de diseño grafico, que se utiliza para definir el aspecto de los elementos de la pagina. Utiliza selectores y reglas para afectar la estructura, en este caso, el orden de los factores no altera el producto.

  • inline utilizamos el atributo style para definir las reglas css, tiene mayor prioridad pero es mas dificil de mantener.
  • inblock puede ser interno o externo y utiliza selectores que indican al navegador a que etiquetas les corresponde cada regla.
