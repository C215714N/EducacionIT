Nuevos Elementos HTML

Contenido Multimedia

A diferencia de otras versiones, actualmente HTML5 permite reproducir contenido multimedia de forma nativa sin necesidad de plugins de terceros como Flash. Las etiquetas que intervienen son <AUDIO> y <VIDEO> que admiten los formatos .mp3 - .wav - .ogg y .mp4 - .webm - .ogv respectivamente; y en caso que el navegador no cuente con una API integrada, podemos generar algun mensaje de compatibilidad.

Elementos Estadisticos

Dependiendo de los valores que querramos representar visualmente, HTML nos ofrece 2 elementos. por un lado la etiqueta <PROGRESS> que se utiliza para mostrar el avance de una tarea cuando esta aun no fue finalizada, y por el otro, el elemento <METER> desarrollado para comparar los valores finales de una actividad cuando esta fue finalizada.

  • Progress
    • value="undefined"
    • value="20" max="100"
    • value="javascript"
  • Meter
    • value* < low < high < optimum
    • low < value* < high < optimum
    • low < high < value* < optimum

API's de Dibujo

Una de las ultimas herramientas que incorpora HTML, son las de dibujo, dividiendose en 2 grupos. Por un lado se encuentra <SVG> que permite la creacion de graficos vectoriales (conjuntos de lineas y curvas calculados matematicamente) mediante la implementacion de XML; y por el otro <CANVAS> permite la realizacion de graficos mapa de bits (grilla que combina canales de color para la representacion de elementos con pixeles) mediante la implementacion de JS.

Aspectos Visuales CSS

Modelos de Color

Existen varias formas de representar los colores en la web, desde las mas sensillas como HTML Colors donde se asocia un nombre a un valor como darkslateblue; el Modelo RGB que utiliza la sintesis aditiva mezclando los canales de luz rojo, verde y azul, del cual se pueden implementar 2 variantes, decimal (75, 60, 140) o hexadecimal #4B3C8C. Por ultimo, se incorpora el Modelo HSL que hace uso del circulo cromatico para la seleccion del tono y luego mediante la manipulacion de saturacion y luz obtenemos el valor deseado como (250, 40%, 40%)

darkslateblue
rgb(75, 60, 140)
#4B3C8C
hsl(250, 40%, 40%)

Transformaciones 2d

CSS permite la aplicacion de alteraciones en el aspecto de las etiquetas, mediante lo que se conoce como transformaciones. Las mismas utilizan como referencia a la etiqueta donde se aplican y permiten, entre otras cosas, rotar, escalar o redimensionar las mismas junto a su contenido.

translate(dX,dY)
scale (uX, uX)
rotate (deg)
skew(degX, degY)

Animaciones CSS

Fotogramas en funcion del tiempo

Las animaciones en css funcionan de manera similar a como lo hacen programas como Adobe Flash/Animate, mediante puntos de cambio (fotogramas clave) que indican al navegador como debe lucir nuestro elemento para luego rellenar los espacios con transiciones. Las animaciones basicas cuentas con dos estados from{...} y to{...} y tambien se les puede asignar un timing-function al igual que las transiciones.

ease
ease-in
ease-out
ease-in-out
linear
cubic-bezier(x1,y1,x2,y2)

Direccion de la pelicula

Al igual que un video, la animacion puede ser reproducida en dos sentidos, normal o reversa, en el caso que hayamos especificado un animation-timing-function mayor a uno tambien podemos elegir entre 2 valores mas que son alternate y alternate-reverse. Estos valores indican cual fotograma sera el inicial y cual el final, en el caso de los ultimos se alterna del from{...} al to{...} y luego al from{...} nuevamente y viceversa.

reverse
alternate
alternate-reverse
