Elementos HTML5

Incorporaciones al estandar

Con el surgimiento de HTML5 en 2014 se añadieron nuevos elementos, entre ellos, etiquetas que facilitan a los motores de busqueda la identificacion de cierto tipo de contenidos, haciendo incapie en la semantica, como los que se mencionan mas adelante.

Etiquetas estadisticas

Son elementos que se utilizan para definir el estado de una tarea, estan son <progress> que se debe implementar cuando la misma aun no fue finalizada y por lo tanto sus valores pueden cambiar ej: el avance durante la capacitacion; y <meter> que a diferencia de la anterior, se utiliza cuando la tarea fue finalizada y necesitamos realizar una comparacion, ej: resultado de un examen.

    • value = javaScript
  • Meter

Preprocesadores CSS

Pseudolenguajes

Es un programa que permite generar CSS a partir de una sintaxis única, estas herramientas disponen de utilidades que no están en el lenguaje CSS y, por tanto, no las interpretan los navegadores directamente.

Utilizando un preprocesador conseguimos economizar tiempo, ya que escribimos menos código. Además, su mantenimiento es fácil y resulta más comprensible, debido básicamente a su optimizada organización.

Por el contrario, utilizar un preprocesador implica aprender un nuevo lenguaje y escribir código poco común, que debe ser compilado en CSS y que afecta al flujo de desarrollo. Este código, que no entiende el navegador, deberemos traducirlo antes de llevar un sitio web a producción.

LESS

Preprocesador que utiliza javascript como herramienta de compilacion para crear hojas de estilo. Su principal ventaja es la asincronicidad del lenguaje que permite al usuario llamar a los estilos sin importar el orden de aparicion en las lineas de codigo, para compilarlo debemos utilizar el comando lessc input.less output.css

SASS

Preprocesador que utiliza ruby como herramienta de compilacion, y ademas brinda la opcion de observar los archivos que utilizamos mediante el comando sass --watch input.scss output.css. Esta biblioteca actualmente es utilizada por el equipo de desarrollo de Bootstrap y cuenta con metodos parecidos a los de cualquier lenguaje de programacion.

Diseñando con estilos

Transformaciones

Al igual que los programa de diseño, css permite modificar los puntos de ajuste de una etiqueta, es decir, nodos que permiten la modificacion de las etiquetas. Entre los cambios que podemos realizar se encuentran el escalado, desplazamiento, rotacion y sesgado y a diferencia de otros atributos de css, estos cambios modifican el contenido de dicha etiqueta.

scale(x,y)
translate(x,y)
rotate(degrees)
skew(x,y)
perpective(rotateX)
perpective(rotateY)
perpective(TranslateZ)
