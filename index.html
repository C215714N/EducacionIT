Estudios

HTML

Lenguaje de marcado de hipertexto utilizado para definir la jerarquia y la funcion que cumple cada elemento dentro de la pagina web mediante el uso etiquetas.

w3c

CSS

Las hojas de estilo en cascada corresponden a un lenguaje de diseño grafico, utilizado para definir el aspecto y disposicion de los elementos de la pagina web.

whatwg

JS

Lenguaje de programacion interpretado por el navegador y multiparadigma, utilizado para brindar dinamismo y logica a las paginas web. Se caracteriza por ser asincrono.

mozilla

SQL

Lenguaje de consulta estructurada utilizado por los motores de base de datos y que permite de manera sencilla la definicion y solicitud de estructuras de datos en el servidor.

oracle

PHP

El preprocesador de Hipertexto es un lenguaje de programacion de tipado dinamico y orientado a objetos, desarrollado para realizar paginas web dinamicas.

php.org

Python

Lenguaje de programacion multiparadigma y de un solo hilo, que abarca muchas areas como la robotica y se caracteriza por manejar el codigo segun los niveles de sangrado.

py

GIT

Sistema de control de versiones utilizado para registrar los cambios realizados durante el desarrollo de un proyecto, funciona mediante la implementacion de un grafo dirigido aciclico.

git scm
