Maquetacion Web

Introduccion a la Maquetacion

Cuando desarrollamos un sitio web debemos tener en cuenta varios aspectos del diseño, es por esto que la planificacion es importante, ademas de saber traducir las propuetas de diseño grafico a codigo que sera interpretado por el navegador.

Por otro lado debemos aprovechar las herramientas de los motores de busqueda, para optimizar el posicionamiento y brindar una grata experiencia de usuario. Para ello tendremos que considerar varios aspectos de los lenguajes, que se detallan mas adelante.

Estructura semantica HTML

Se denomima asi a un conjunto de etiquetas que surgieron a partir de la version 5 de html y que permiten tanto a los desarrolladores como a los motores de busqueda identificar mas facilmente los elementos que componen una pagina web. Entre la etiquetas agregadas se encuentran las siguientes:

  • header Cabecera o contenido introductorio, debe contener los encabezados y suele poseer alguna galeria a modo de introduccion.
  • nav Corresponde a los links mas importantes, se aconseja evitar el uso de redes sociales y tratar que sea de facil acceso.
  • main Contenido principal que suele englobar la informacion relevante para el usuario, aunque se utilizar tambien para resaltar la seccion mas importante de la pagina.
  • article Contenido Independiente que funciona tanto dentro como fuera de la pagina debido a que posee una Estructura definida.
  • section Contenido Contextualizado que solamente funciona dentro de la pagina y se relacion con otro elemento.
  • aside Contenido tangencial que se relaciona indirectamente con sus elementos hermanos, se utiliza para publicidad o sugerencias de contenido.
  • footer Contenido complementario de la pagina, se utiliza para mostrar informacion de contacto, redes sociales, mapa del sitio y derechos de autor.

Jerarquia de selectores CSS

Cada una de las reglas aplicadas con CSS posee una prioridad segun el tipo de selector utilizado, es fundamental que las mismas esten organizadas para facilitar la comprension del diseño y no superponer o repetir reglas de forma innecesaria. Los selectores segun su importancia se dividen de la siguiente manera:

  • Tag Selector
    • header{...}
    • headerh1{...}
  • Class Selector
    • .header{...}
    • .headerh1{...}
  • Id Selector
    • #header{...}
    • #headerh1{...}

Planificacion del Diseño

Uno de los aspectos a tener en cuenta cuando maquetamos es la traduccion de un diseño a un formato adaptable, para ello debemos seleccionar las medidas que mejor representen la proporcion de los elementos.

Unidades de medida

CSS soporta cualquier medida existente, pero esto no quiere decir que debamos utilizar todas, ya que cada una se aplica dependiendo de la situacion. Esta se pueden clasificar de la siguiente manera:

  • Absolutas Son aquellas que siempre valen lo mismo independientemente del medio donde se reproduzcan. Se utilizan principalmente para definir limites y la que se usa en mayor medida es el px (puntos en pantalla) como por ejemplo max-width: 720px;
  • Relativas Son aquellas que dependen del tamaño de su contenedor o su tipografia. Se utilizan para definir el tamaño de las cajas como por ejemplo width: 100%; o modificar la tipografia como padding: 1.5rem; y como se ve las mas utilizadas son porcentaje y rem (Ancho de "M" en la raiz)
  • Flexibles Identificadas como unidades relativas, estas toman de referencia las dimensiones del viewport y se implementan para tipografia flexible y definir alturas. Las medidas relacionadas con la ventana son vmin, vmax, vh y vw

Paleta de Colores

Corresponde a seleccion de Colores y es una de las caracteristicas principales a la hora de desarrollar un sitio web Normalmente el equipo de diseño grafico se encarga de la seleccion y aplicacion de la misma y nosotros como desarrolladores solamente debemos traducirlo a formato web, pero en ocasiones, especialmente si trabajamos de forma independiente deberemos seleccionar e implementar los colores. Entre las diferentes opciones se encuentran:

  • Color HTML: darkslateblue
  • Color Decimal: rgb(68, 68, 136)
  • Color Hexadecimal: #224488
  • hsl(250, 45%, 45%)

Implementacion del Diseño

Cuando organizamos los contenido dentro de la pagina web, podemos alterar de manera visual la disposicion de los elementos de la pagina, para ello existen diferentes herramientas/atributos como float, flexbox y grid, cada uno aplicando segun el objetivo a cumplir.

Flexbox

Modelo de cajas flexible unidimensional, que permite disponer el contenido sobre un eje principal, que por defecto es horizontal y jusficifarlo dentro del mismo. Tambien posee un eje secundario que permite alinear el contenido.
