Tema 1: Contenido Multimedia

Con la llegada de HTML5 se incorporaron 2 nuevas etiquetas, estas son AUDIO y VIDEO , permitiendo a los navegadores la reproduccion de contenido multimedia de forma nativa sin necesidad de utilizar plugins de terceros como FLASH. Los formatos soportados por los navegadores son .mp3, .wav, .ogg en el caso del elemento AUDIO y .mp4, .avi, .ogv en el caso del elemento VIDEO

Autor: Cristian Racedo

Tema 2: Elementos Estadisticos

Al igual que los elementos anteriores, se incorporaron en HTML5 dos nuevas etiquetas, que permiten llevar un control a nivel estadistico. Por un lado tenemos PROGRESS que se utiliza para indicar una instancia (avance) de una tarea que aun no ha finalizado. Por otro lado se encuentra METER que sirve para evaludar los estados (Comparar) de una tarea que ya fue finalizada, y por consecuente sus valores no pueden ser modificados

  • Progress
    • Sin Value
    • Con Value
  • Meter
    • VALUE* <= LOW <= HIGH <= OPTIMUM
    • LOW <= VALUE* <= HIGH <= OPTIMUM
    • LOW <= HIGH <= VALUE* <= OPTIMUM

Tema 3: API's de DIBUJO

Unas de las ultimas incorporaciones de los navegadores, son las API de dibujo, a las que podemos acceder utilizando las etiquetas SVG para la crecion de graficos vectoriales y CANVAS para la creacion de mapa de bits. Debemos tener en cuenta que al trabajar en un editor de codigo con estos elementos necesitaremos implementar codigo XML (svg) y JS (canvas) .
