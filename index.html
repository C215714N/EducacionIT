  • Background-image: permite aplicar una imagen de fondo utilizando el protocolo URL para indicar la ruta de la misma. Los formatos soportados son .jpg - .png - .svg - .gif
  • Background-size: permite definir el tama単o del fondo con respecto a la etiqueta contenedora, se puede indicar mediante medidas: horizontal y vertical o utilizando palabras clave: cover para abarcar toda la etiqueta y contain para que entre dentro de la misma.
  • Background-position: permite indicar la posicion inicial de la primer imagen de fondo (en caso que se repita), al igual que el atributo anterior se indica mediante medidas: para ejeX y ejeY o utilizando palabras clave: left y right para el eje X, top y bottom para el el eje y, o center para ambos.
  • Background-attachment: define como sera el comportamiento del fondo con respecto a la pagina, este recibe los valores normal y fixed cuyo caracteristica el posicionarse con respecto a la ventana.
  • Background-repeat: define como se repetira la imagen de fondo, en caso que sea necesario, el valor predeterminado es repeat pero podemos seleccionar entre repeat-x, repeat-y o no-repeat.
