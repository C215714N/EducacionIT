Desarrollo Front-End

Front-end corresponde al desarrollo del lado del cliente (navegador) y engloba la estructura, el diseño grafico (UI/UX) y la logica o funcionalidad basica y los lenguajes involucrados son los siguientes:

  • html
  • css
  • js

HTML (HyperText Markup Language)

El lenguaje de marcado de hipertexto corresponde a un lenguaje de estructura, es decir, que es la base fundamental de un sitio web. Utiliza etiquetas para definir el rol que cumple cada elemento dentro de la pagina y su relacion con el contexto.

Clasificacion de las etiquetas

  • Segun su tipo:
    De Bloque: Son aquellas que ocupan todo el espacio de su contenedor, generan salto de linea y no comparten espacio con otros elementos, se les puede dar tamaño y funcionan principalmente como contenedores.
    De Linea: Son aquellas que ocupan el espacio de su contenido, no generan salto de linea y comparten espacio con otros elementos de su mismo tipo, no se les puede dar tamaño (automatico) y funcionan principalmente como contenido.
  • Segun su funcion:
    Con Carga Semantica: Son aquellas que representan algun aspecto dentro de la pagina y corresponde a la mayoria de los elementos que podemos utilizar, permiten asignar una jerarquia.
    Presentacionales: Son aquellas cuya finalidad es apoyar o facilitar la aplicacion del diseño, actualmente son pocas etiquetas las que pertenecen a este tipo, pero podemos destacar SPAN (expandir) y DIV(capa) que sirven para aplicar estilos dentro de un texto o a un caracter, y agrupar elementos respectivamente.

CSS (Cascading Styles Sheets)

Las hojas de estilo en cascada corresponden a un lenguaje de diseño grafico, que se utiliza para definir la apariencia de los elementos dentro de la pagina. Utiliza principalmente selectores y reglas para poder realizar dichos cambios.

  • margin: distancia que separa a las etiquetas
  • border: linea que rodea a la etiqueta y por defecto es cero
  • padding: distancia entre el border y el contenido y, al igual que el borde, suma tamaño
  • content: elementos que se encuentran dentro de la etiqueta, a los que podemos darle tamaño
