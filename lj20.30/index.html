Lenguajes FrontEnd

HTML

HyperText Markup Language o como indica su traduccion, lenguaje de marcado de hypertexto, utiliza etiquetas y atributos para definir la estructura de la pagina. Es interpretado por el navegador y sus etiquetas se clasifican como elementos de bloque y elementos de linea, es necesario ya que representa la base de un sitio web.

CSS

Cascading StyleSheets tambien conocido como Hojas de Estilo en Cascada corresponde a un lenguaje de Dise√±o Grafico que utiliza selectores y reglas para definir el aspecto de los elementos de la pagina y a diferencia del lenguaje anterior, las reglas se aplican en orden, respetando la jerarquia de los selectores.

JS

Nombrado JavaScript con el fin de ganar adeptos gracias a la popularidad de JAVA, pero a diferencia de este, es un lenguaje de programacion interpretado por el navegador. Nuestro codigo se lee y se ejecuta desde el FrontEnd, aunque en la actualidad tambien se puede utilizar desde el lado del BackEnd gracias a NodeJS

Bootstrap

Framework de CSS desarrollado por el equipo de twitter que facilita la creacion de paginas web, debido a que nos brinda una hoja de estilos de mas de 10 mil lineas de codigo y un script de JS de al menos 7 mil lineas, que esta pensado para que quien utilice dicha herramienta use la menor cantidad de codigo posible.

GIT

Sistema de control de versiones desarrollado por Linus Torvalds con el objetivo de registrar los cambios realizados en el kernel de Linux. Nos permite manejar un historial de cambios y deshacer modificaciones de un proyecto, asi como tambien reaplicar cambios previos de ser necesario.

Preprocesadores

Son herramientas que se utilizan para agilizar la creacion de hojas de estilos, mediante el uso de algun lenguaje de programacion y caracteristicas propias del mismo como el uso de condicionales y bucles. Las mismas normalmente son compiladas, para luego crear el css correspondiente.

SASS

Syntactically Awesome Style Sheets Es un Preprocesador que utiliza Ruby como lenguaje compilador y, al igual que este, el codigo de nuestro archivo sass o scss debe estar correctamente organizado, ya que no se pueden utilizar elementos que no hayan sido declarados previamente.

Podemos acceder a la documentacion de este preprocesador, muy popular, desde el sitio oficial de SASS, donde figuran sus caracteristicas, ejemplos, registro de cambios, etc. Cabe mencionar que entre sus caracteristicas principales se encuentran las siguientes:

  • Variables
  • Nesting
  • Mixins
  • Condicionales
  • Bucles
