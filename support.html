Formularios HTML

Son herramientas que permiten el intercambio de datos entre cliente y servidor, para utilizarlos dentro de la pagina solamente debemos colocar la etiqueta <form> y dentro de la misma los elementos que seran procesados durante el envio del mismo.

多Como funcionan?

A la etiqueta mencionada anteriormente debemos especificarle por lo menos 2 atributos, que son action="..." el cual indica que debe suceder al momento de enviar el Formulario, utilizandose normalmente un script en lenguaje de servidor; y method="..." que especifica la manera en la que se va a realizar el envio del mismo y cuyos valores son predefinidos.

  • GET: metodo que prioriza la respuesta del servidor, los datos permanecen visibles en la barra de direcciones y se envian en texto plano. Se utiliza principalmente en buscadores.
  • POST: metodo cuya prioridad radica en la proteccion de los datos del usuario, ya que estos se cifran antes de realizar el envio. Se utiliza en formularios de Registro, Acceso, Contacto, etc...

多Como estan compuestos?

Dentro de un formulario podemos colocar cualquier etiqueta, pero existen 2 elementos necesarios, estos son las Etiquetas (LABEL) que se utilizan para guiar al usuario al momento de completar el mismo; y los Campos (INPUT / TEXTAREA / SELECT) que permiten al usuario seleccionar o escribir datos, estos poseen diferentes atributos.

  • name="..." especifica el nombre con el que se va a identificar el dato enviado a traves del formulario.
  • type="..." indica el tipo de dato admitido por el campo de formulario, en el caso de los input es practicamente obligatorio.
  • value="..." correponde al valor ingresado en campo, y en caso que este sea seleccionado, debe figurar de manera explicita.

Tablas HTML

Son elementos que permiten organizar la informacion de manera tabular y deben utilizarse cuando los datos a representar corresponden a cierta estructura. Para utilizarlas dentro de la pagina simplemente debemos colocar la etiqueta <table> y dentro de la misma los elementos que la componen, respetando el modelo de cajas.

多Poseen Estructura Semantica?

Como se menciona anteriormente, el contenido de la tabla esta dividido en partes,es decir que, creamos las filas utilizando la etiqueta <tr> (Table Row) y dentro de estas colocamos las celdas con la etiqueta <td> (Table Division). Pero debemos tener en cuenta que hoy dia podemos identificar cada parte de la tabla con los siguientes elementos.

  • THEAD Cabecera de tabla que contiene las celdas de encabezado <th>
  • TBODY Cuerpo de tabla que posee las filas de datos y puede contener encabezados
  • TFOOT Pie de tabla que posee informacion complementaria como totales, promedios, etc...
