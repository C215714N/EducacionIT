Cisco CCNA - Switching

Las redes modernas continúan evolucionando para adaptarse a la manera cambiante en que las organizaciones realizan sus actividades diarias. Ahora los usuarios esperan tener acceso instantáneo a los recursos de una compañía, en cualquier momento y en cualquier lugar. Estos recursos incluyen no solo datos tradicionales, sino también de video y de voz. También hay una necesidad creciente de tecnologías de colaboración que permitan el intercambio de recursos en tiempo real entre varias personas en sitios remotos como si estuvieran en la misma ubicación física.

Virtual Local Area Network

Comandos Cisco - VLAN
Modo Comando Descripcion
sw(config)# vlan [vlan-id] crea una VLAN (rango normal 1 - 1005 | rango extendido 1006 - 4094).
sw(config-vlan)# name [vlan] asigna un nombre a una VLAN.
sw(config-vlan)# state [active | suspended] cambia el estado, todo puerto suspendido en esta VLAN descartara trafico.
sw# show vlan [vlan-id] muestra informacion de una VLAN en particular.
sw(config)# interface [vlan-id] ingresa al modo de configuracion de SVI
sw(config-if)# ip address [ip] [subnet] asigna un direccion y mascara a la interfaz para el acceso remoto
sw(config-if)# no shutdown habilita la SVI para administracion

Virtual Trunking Protocol

Comandos Cisco - VTP
Modo Comando Descripcion
sw(config)# interface [if n°] accede al submodo de configuracion de interfaz (fa0/1 | gi0/1)
sw(config-if)# switchport mode [access | trunk] define el modo de enlace para la interfaz
sw(config-if)# switchport access [vlan-id] asocia una vlan a la interfaz en modo acceso
sw(config-if)# switchport trunk allowed [range] restringe las vlan permitidas en el enlace troncal
sw(config-if)# switchport port-security habilita la seguridad de puertos en la interfaz
sw(config-if)# switchport port-security mac-address [a.a.a.a | sticky] agrega una mac address a la interfaz de forma estatica o dinamica
sw(config-if)# switchport port-security violation [protect | restrict | shutdown] define la operacion a realizar cuando se hayan violado las restricciones de seguridad
